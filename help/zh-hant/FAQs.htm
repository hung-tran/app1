常見問題集

Multi-Factor Authentication™ 的運作方式為何？

Multi-Factor Authentication 可透過撥打確認電話、傳送簡訊、登入期間通知 Multi-Factor Authentication 行動裝置應用程式或提示輸入 Multi-Factor Authentication 行動裝置應用程式或權仗的驗證碼。

 

步驟 1：

請輸入慣用的使用者名稱和密碼。

步驟 2：

您會立即收到通話、簡訊、行動裝置應用程式通知或者輸入驗證碼的提示。若使用通話方法，請回答並按下 #、輸入您的 PIN 或者說出您的聲紋片語。 若使用簡訊方法，請使用簡訊中的單次密碼 (OTP) 回覆，或以 OTP + 您的 PIN 進行回覆。 若使用行動裝置應用程式方法，請按下 [驗證] 或者輸入您的 PIN 並按下安裝於裝置上 Multi-Factor Authentication 行動裝置應用程式中的 [驗證]。 若使用 OATH 權仗方法，請輸入 Multi-Factor Authentication 行動裝置應用程式或權杖中顯示的驗證碼。

這樣就行了！

 

這個簡單的程序能透過兩種不同的管道提供兩種不同的驗證要素 (電腦和電話服務)。它適用於任何通話模式中的一般電話或行動電話、簡訊模式中任何具有簡訊功能的電話，或支援 Multi-Factor Authentication 行動裝置應用程式的裝置。

 

當我遺失電話時，該怎麼辦？

選取 [變更電話號碼] 選項以輸入新的電話號碼。一旦建立完整的身分後，您也可以致電給支援人員來設定其他號碼。

 

如果我身處沒有行動電話收訊的區域內，該怎麼辦？

您可以選取 [變更電話號碼] 選項將帳戶改成指向其他電話號碼。有些公司也會提供讓您在沒有通話或簡訊的情況下登入一次的 [單次許可] 選項。如果可用，選取 [單次許可] 選項即可存取此功能。或者使用任何電話撥打給支援人員尋求協助。 OATH 權杖方法並不需要行動電話收訊。

 

如果我在未嘗試登入時收到通話、簡訊或來自 Multi-Factor Authentication 的行動裝置應用程式通知，該怎麼辦？

只有當其他人嘗試登入您的帳戶，且他們已經知道您的密碼時，才會發生這種情況。請切記，只有在使用者名稱和密碼通過驗證後才會撥打電話、發送簡訊或者傳送行動裝置應用程式通知。因此當這種情況發生時，表示 Multi-Factor Authentication 剛阻止了其他人違法存取您的帳戶！若要暫時封鎖嘗試驗證您的帳戶，使用通話方法者請遵循電話功能表的提示，使用簡訊方法者請以公司指派的詐騙碼回覆簡訊，或者在行動裝置應用程式中按下 [拒絕並回報詐騙]。 如果貴公司已啟用 [詐騙警示]，除了會封鎖您的帳戶之外，還會立即傳送警示到貴公司的 IT 保障安全的團隊。若要解除帳戶的封鎖，請聯絡您的支援人員。
