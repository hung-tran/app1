新增使用者

[新增使用者] 頁面可讓您建立新的 Multi-Factor Authentication 帳戶。

 

注意： 許多的使用者設定 (例如通話、簡訊或行動裝置應用程式的方法) 都會根據 Multi-Factor Authentication Server [公司設定] 區段中指定的使用者預設值進行預設。

 

若要新增使用者：

  1. 輸入使用者詳細資料。

  2. 按一下 [新增] 按鈕。

 

功能

描述

使用者名稱

使用者名稱通常是使用適當電子郵件格式的使用者電子郵件。

 

範例： chuck@abc.com

 

名字

輸入使用者的名字。

 

姓氏

輸入使用者的姓氏。

 

電子郵件地址

輸入使用者的電子郵件地址。 電子郵件將會傳送至此電子郵件地址。 若未指定，但電子郵件的格式正確，電子郵件就會傳送至該使用者名稱。

 

使用者群組

輸入使用者的使用者群組。

 

國碼 (地區碼)

在下拉式清單中選取適當的國碼 (地區碼)。預設會設定為美國和加拿大 +1。

 

注意：某些國際電話將需要收取費用。 如果您的公司並未提供該選項，則可能無法撥打至某些國碼 (地區碼)。

 

電話

輸入 Multi-Factor Authentication 在驗證期間將撥打或傳送簡訊的電話號碼。這可能是行動電話或桌上型電話的電話號碼，或其他通話方式的號碼。 此電話必須具備傳送簡訊的功能。 電話號碼格式必須符合在 [國碼 (地區碼)] 欄位中選取的格式。

 

範例：若您使用美國和加拿大電話號碼，則必須輸入 10 碼電話號碼 (例如，555-123-4567)。

 

分機號碼

輸入電話接通之後要撥打的分機號碼。 可使用數字、逗號、* 及 #。 逗號可用來暫停一秒，以瀏覽電話系統功能表。 此為選用性欄位。

 

備用國碼 (地區碼)

為備用電話號碼選取適當的國碼 (地區碼)。

 

備用電話

輸入 Multi-Factor Authentication 在主要電話沒有回應時撥打的電話號碼。

 

備用分機號碼

輸入備用電話號碼的分機。

 

PIN

輸入使用者的 PIN。 或者您也可以按一下 [產生] 按鈕，以產生 PIN。 PIN 必須遵守 PIN 長度下限和弱式 PIN 的規則。

 

[產生] 按鈕

[產生] 按鈕提供隨機產生的 PIN。 產生的 PIN 長度將超過 4 碼或使用者的 PIN 長度下限。

 

確認 PIN

請重新輸入您在 [PIN] 欄位中輸入的 PIN。確認 PIN 必須完全相符。

 

已啟用

指定是否已啟用使用者。 預設會加以選取。

 

注意： 如果未啟用使用者記錄，則使用者在登入時將不會收到 Multi-Factor Authentication 通話，並且會根據公司對於已停用行為的預設值來允許或拒絕存取應用程式。

 

需變更 PIN

指定使用者在下次 Multi-Factor Authentication 通話時必須變更他們的 PIN。

 

注意： 只有當公司設定允許使用者變更其 PIN 時才會顯示 [需變更 PIN] 核取方塊。

 

傳送電子郵件

指定是否應將電子郵件傳送至新增的使用者。

 

注意：只有在設定公司的電子郵件之後才會顯示 [傳送電子郵件] 核取方塊。
