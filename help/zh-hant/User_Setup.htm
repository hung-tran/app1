使用者設定

如果您的帳戶設定不完整，您必須指定帳戶所使用的電話號碼 (若適用) 和/或 PIN (若適用)，否則您可能會收到產生啟用代碼的提示以啟用 Multi-Factor Authentication 行動裝置應用程式。 若公司允許，則可以選擇通話、簡訊、行動裝置應用程式與 OATH 權杖方法。 您所指定的電話號碼將會收到 Multi-Factor Authentication 通話或簡訊，確認此其是否有效，行動裝置應用程式也會收到通知，否則您會收到輸入驗證碼的提示。 若您使用通話方法，視您通話模式之不同，系統會提示您按下 #、輸入您的 PIN 或註冊您的聲紋。 若您使用簡訊方法，將會收到單次密碼 (OTP)，然後可用 OTP 或 OTP + 您的 PIN 進行回覆。 若您使用行動裝置應用程式方法，行動裝置應用程式將會收到通知以提示您進行驗證，視您行動裝置應用程式模式之不同，有可能會需要輸入 PIN。 若使用 OATH 權杖方法，使用者入口網站會提示您輸入驗證碼。 驗證成功之後，系統會提示您建立安全性問題。

 

若要完成通話方法的使用者設定：

  1. 輸入您的國碼 (地區碼) 和電話號碼。 您可能還需要輸入備用電話號碼 (若適用)。

  2. 輸入您的 PIN (若適用)。

  3. 按一下 [立即撥打電話給我以進行驗證] 按鈕。

 

若要完成簡訊方法的使用者設定：

  1. 輸入您的國碼 (地區碼) 和電話號碼。

  2. 輸入您的 PIN (若適用)。

  3. 按一下 [立即傳簡訊給我以進行驗證] 按鈕。

 

若要完成行動裝置應用程式方法的使用者設定：

  1. 在您的手機上安裝行動裝置應用程式。

  2. 按一下 [產生啟用代碼] 按鈕。

  3. 在行動裝置應用程式中輸入啟用代碼和 URL 以完成啟用。

  4. 輸入您的 PIN (若適用)。

  5. 按一下 [立即驗證我的身分] 按鈕。

 

若要完成 OATH 權杖方法的使用者設定：

  1. 在您的手機上安裝行動裝置應用程式。

  2. 按一下 [產生啟用代碼] 按鈕。

  3. 在行動裝置應用程式中輸入啟用代碼和 URL 以完成啟用。

  4. 按一下 [立即驗證我的身分] 按鈕。

或

  1. 輸入印在 OATH 權杖背面的序號。

  2. 輸入目前顯示在 OATH 權杖上的代碼。

  3. 按一下 [驗證 OATH 權杖] 按鈕。

 

 

功能

描述

方法

選取通話、簡訊、行動裝置應用程式或 OATH 權杖方法。 若使用通話方法，您將會收到 Multi-Factor Authentication 通話，提示您按下 #、輸入 PIN 或註冊您的聲紋。 若您使用簡訊方法，將會收到單次密碼 (OTP) 的簡訊，然後請以 OTP 或 OTP + 您的 PIN 進行回覆。 若您使用行動裝置應用程式方法，行動裝置應用程式將會收到通知以提示您進行驗證，您可能需要輸入 PIN。 若使用 OATH 權杖方法，使用者入口網站會提示您輸入驗證碼。

 

注意：   您選取的方法就是您在存取 Multi-Factor Authentication 所保護之應用程式時使用的方法。 您隨時都可以在 [變更方法] 頁面中變更方法。

 

國碼 (地區碼)

在下拉式清單中選取適當的國碼 (地區碼)。如果您已為您的帳戶設定電話號碼，預設值就會是該電話號碼的國碼 (地區碼)。 否則，預設會設定為美國和加拿大 +1。

 

注意：某些國際電話將需要收取費用。 如果您的系統管理員並未實作此選項，則可能無法撥打至某些國碼 (地區碼)。

 

電話

輸入 Multi-Factor Authentication 在驗證期間將撥打或傳送簡訊的電話號碼。這可能是行動電話或桌上型電話的電話號碼，或其他通話模式的號碼。 此電話號碼必須具備傳送簡訊的功能。 電話號碼格式必須符合在 [國碼 (地區碼)] 欄位中選取的格式。

 

範例：若您使用美國和加拿大電話號碼，則必須輸入 10 碼電話號碼 (例如，555-123-4567)。

 

分機號碼

輸入電話接通之後要撥打的分機號碼。 可使用數字、逗號、* 及 #。 逗號可用來暫停一秒，以瀏覽電話系統功能表。 此為選用性欄位。

 

備用國碼 (地區碼)

為備用電話號碼選取適當的國碼 (地區碼)。

 

備用電話

輸入 Multi-Factor Authentication 在主要電話沒有回應時撥打的電話號碼。

 

備用分機號碼

輸入備用電話號碼的分機。

 

PIN

輸入 PIN 號碼。 每次在進行驗證通話時都必須輸入此 PIN，您可以在回覆每封簡訊時輸入單次密碼 (OTP)，或輸入行動裝置應用程式來完成驗證。 您的系統管理員可能已設定 PIN 規則，例如長度下限。

 

序號

輸入印在 OATH 權杖背面的序號。

 

代碼

輸入目前顯示在 OATH 權杖上的代碼。

 

 

若要返回 [登入] 頁面，請按一下 [取消] 按鈕。
