Multi-Factor Authentication 使用者入口網站

Multi-Factor Authentication 使用者入口網站可讓您有一次登入時許可 Multi-Factor Authentication 的機會，以及管理 Multi-Factor Authentication 帳戶設定，包括您的方法、電話號碼、PIN、語言和安全性問題，以啟用 Multi-Factor Authentication 行動裝置應用程式。系統管理員可能會停用其中某些功能。 如需有關如何使用 Multi-Factor Authentication 的資訊，請參閱常見問題集。

 

您可以利用目錄、索引和搜尋來協助您在說明系統中尋找資訊。
