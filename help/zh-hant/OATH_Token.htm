OATH 權杖

[OATH 權杖] 頁面可讓您將所選的使用者與協力廠商 OATH 權杖建立關聯。 在與使用者建立關聯之前，必須先在 Multi-Factor Authentication Server 中定義權杖。 權杖必須支援時間序的 OATH 代碼。 您也可以改為選擇使用 Multi-Factor Authentication 行動裝置應用程式，來輸入OATH 代碼。

 

若要指定 OATH 權杖：

  1. 輸入適當的值。

  2. 按一下 [儲存] 按鈕。

 

功能

描述

序號

輸入協力廠商 OATH 權杖的序號。

 

代碼

輸入目前顯示在協力廠商 OATH 權杖上的代碼。
