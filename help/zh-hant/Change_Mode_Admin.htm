變更方法

[變更方法] 頁面可讓您變更所選使用者帳戶的方法。 若使用通話方法，使用者將會在驗證時收到通話。 若使用簡訊方法，使用者將會在驗證時收到簡訊。 若使用行動裝置應用程式方法，使用者將會在安裝於自己裝置上的 Multi-Factor Authentication 行動應用程式中收到通知。 若使用 OATH 權杖方法，將會提示使用者輸入驗證碼。

 

若要變更所選使用者的方法：

  1. 選取適當的方法。

  2. 按一下 [儲存] 按鈕。

 

功能

描述

方法

在下拉式清單中選取適當的方法。預設值就是使用者目前的方法。

 

傳送電子郵件

指定是否應傳送電子郵件通知使用者有關該項更新。

 

注意：只有在設定公司的電子郵件之後才會顯示 [傳送電子郵件] 核取方塊。
