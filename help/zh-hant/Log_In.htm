登入

若要登入使用者入口網站，請輸入您的使用者名稱和密碼。 將會撥打 Multi-Factor Authentication 電話、發送簡訊至您的電話號碼、傳送通知至 Multi-Factor Authentication 行動裝置應用程式，或提示您輸入驗證碼。 如果您成功完成驗證，將會進入主頁面。

 

如果您的帳戶設定不完整，則會將您引導到 [使用者設定] 頁面。

 

如果這是您第一次登入使用者入口網站，則可能會提示您建立安全性問題。

 

如果 Multi-Factor Authentication 遭到拒絕，則可能會提示您回答安全性問題以完成登入。

 

若要登入：

  1. 輸入您的使用者名稱和密碼。

  2. 按一下 [登入] 按鈕。

 

功能

描述

使用者名稱

您平常用來登入 Multi-Factor Authentication 受保護應用程式的使用者名稱。 通常會是您的 Windows 使用者名稱。

 

密碼

您平常用來登入 Multi-Factor Authentication 受保護應用程式的密碼。 通常會是您的 Windows 密碼。
