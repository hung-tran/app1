移除使用者

[移除使用者] 頁面可讓您永久移除選取的使用者。您可以根據 Multi-Factor Authentication Server 中的應用程式設定，允許或拒絕已移除之使用者的存取。 如果應用程式已設為 [需要比對使用者]，則移除的使用者會因為不在 Multi-Factor Authentication Server 的使用者清單中而被拒絕存取。 如果應用程式未設為 [需要比對使用者]，則使用者不需要 Multi-Factor Authentication 也可以驗證成功。

 

按一下 [確認] 按鈕以永久移除選取的使用者。
