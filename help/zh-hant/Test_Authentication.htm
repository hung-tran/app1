測試驗證

[測試驗證] 頁面可讓您為選取的使用者，測試 Multi-Factor Authentication 通話、簡訊驗證或 Multi-Factor Authentication 行動裝置應用程式驗證。 無論是要進行通話、傳送簡訊或通知 Multi-Factor Authentication 行動裝置應用程式，都取決於使用者的模式而定。

 

按一下 [測試] 按鈕以起始通話、傳送簡訊或通知 Multi-Factor Authentication 行動裝置應用程式。
