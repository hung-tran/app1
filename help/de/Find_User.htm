Suchen von Benutzern

Auf der Seite „Benutzer suchen“ können Administratoren nach einem vorhandenen Benutzerkonto suchen.

 

So suchen Sie einen Benutzer:

  1. Geben Sie Suchkriterien an.

  2. Klicken Sie auf die Schaltfläche Suchen.

 

Wenn ein einzelner Benutzer mit den eingegebenen Kriterien übereinstimmt, wird die Seite Benutzerinformationen für diesen Benutzer angezeigt.  Wenn es mehr als eine Übereinstimmung gibt, wird eine Liste von Benutzerlinks angezeigt.  Klicken Sie auf den entsprechenden Benutzerlink, um zur Seite Benutzerinformationen zu gelangen.

 

Element

Beschreibung

Benutzername

Geben Sie die Suchzeichenfolge zum Abgleichen des Benutzernamens ein.

 

Hinweis: Platzhalterzeichen sind in der Suchzeichenfolge in Form eines Sternchens (*) gestattet, das für beliebige Zeichen steht.

 

Vorname

Geben Sie die Suchzeichenfolge zum Abgleichen des Vornamens ein.

 

Nachname

Geben Sie die Suchzeichenfolge zum Abgleichen des Nachnamens ein.

 

E-Mail-Adresse

Geben Sie die Suchzeichenfolge zum Abgleichen der E-Mail-Adresse ein.

 

Benutzergruppe

Geben Sie die Suchzeichenfolge zum Abgleichen der Benutzergruppe ein.

 

Anschluss

Geben Sie die Suchzeichenfolge zum Abgleichen des Anschlusses ein.

 

Tag

Wählen Sie Alle  aus, um alle Benutzer abzugleichen, oder wählen Sie ein bestimmtes Tag aus, um nur Benutzer mit diesem Tag abzugleichen.
