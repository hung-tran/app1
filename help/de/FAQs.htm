Häufig gestellte Fragen

Wie funktioniert Multi-Factor Authentication™?

Bei Multi-Factor Authentication wird zur Bestätigung ein Anruf ausgeführt, eine SMS gesendet, eine Benachrichtigung in der mobilen Multi-Factor Authentication-Anwendung angezeigt, oder es wird von der mobilen Multi-Factor Authentication-Anwendung oder vom Token zur Eingabe eines Authentifizierungscodes aufgefordert.

 

Schritt 1:

Geben Sie Ihren normalen Benutzernamen und Ihr Kennwort ein.

Schritt 2:

Sie erhalten umgehend einen Anruf, eine SMS oder eine Benachrichtigung in der mobilen Anwendung, oder Sie werden zur Eingabe eines Authentifizierungscodes aufgefordert. Nehmen Sie bei der Telefonanrufmethode den Anruf entgegen, drücken Sie auf das Nummernzeichen (#), und geben Sie Ihre PIN ein bzw. sagen Sie Ihre Stimmabdruckpassphrase an.  Antworten Sie bei der SMS-Methode mit dem Einmalkennwort (OTP) in der Nachricht, oder antworten Sie mit Einmalkennwort und PIN (OTP + PIN).  Bei der Methode mit der mobilen Anwendung drücken Sie in der mobilen Multi-Factor Authentication-Anwendung, die auf Ihrem Gerät installiert ist, auf „Authentifizieren“, oder geben Sie Ihre PIN ein, und drücken Sie auf „Authentifizieren“.  Bei der OATH-Tokenmethode geben Sie den Authentifizierungscode ein, der in der mobilen Multi-Factor Authentication-Anwendung oder im Token angezeigt wird.

Das war schon alles!

 

Dieses einfache Verfahren umfasst zwei Stufen der Authentifizierung über zwei separate Kanäle: Ihr Computer sowie Ihr Telefondienst. Multi-Factor Authentication funktioniert im Telefonanrufmodus mit jedem regulären Telefon und Handy, im SMS-Modus mit jedem SMS-fähigen Telefon sowie mit jedem Gerät, von dem die mobile Multi-Factor Authentication-Anwendung unterstützt wird.

 

Was passiert, wenn mein Telefon verloren geht?

Wählen Sie die Option „Rufnummer ändern“ aus, um eine neue Rufnummer einzugeben. Eine zusätzliche Rufnummer kann auch vom Helpdesk eingerichtet werden (hierzu ist eine umfassende Überprüfung Ihrer Identität erforderlich).

 

Was passiert, wenn mein Handy keinen Empfang hat?

Wählen Sie die Option „Rufnummer ändern“ aus, wenn Ihr Konto auf eine zusätzliche Rufnummer verweisen soll. In einigen Unternehmen ist auch eine Option zu einer Einmalumgehung verfügbar. Damit können Sie sich einmalig ohne Telefonanruf oder SMS anmelden. Wählen Sie die Option zur Einmalumgehung aus, falls verfügbar, um auf diese Funktion zuzugreifen. Sie können sich auch von einem beliebigen Telefon aus an den Helpdesk wenden, um Unterstützung zu erhalten.  Für die OATH-Tokenmethode ist kein Mobilfunkempfang erforderlich.

 

Was ist, wen ich von Multi-Factor Authentication einen Telefonanruf, eine SMS oder eine Benachrichtigung in der mobilen Anwendung erhalte, obwohl ich gar nicht versuche, mich anzumelden?

Dies geschieht nur, wenn jemand anders versucht, sich bei Ihrem Konto anzumelden, und dieser Person Ihr Kennwort bereits bekannt ist. Bedenken Sie, dass Anrufe, SMS und Benachrichtigungen in der mobilen Anwendung nur erfolgen, wenn Benutzername und Kennwort überprüft wurden. In diesem Fall wurden Sie von Multi-Factor Authentication also vor einem unerlaubten Zugriff bewahrt. Sie können Ihr Konto auch vorübergehend für Authentifizierungsversuche sperren. Befolgen Sie dazu bei der Telefonanrufmethode die Anweisungen im Telefonmenü, antworten Sie bei der SMS-Methode mit dem Code zur Meldung von Betrug, den Ihr Unternehmen Ihnen zugewiesen hat, oder drücken Sie in der mobilen Anwendung auf „Ablehnen und Betrug melden“.  Wenn die Betrugswarnung für Ihr Unternehmen aktiviert ist, wird zusätzlich zum Sperren Ihres Kontos umgehend eine Warnung an das IT-Sicherheitsteam Ihres Unternehmens gesendet. Wenden Sie sich an Ihren Helpdesk, um Ihre Kontosperrung aufzuheben.
