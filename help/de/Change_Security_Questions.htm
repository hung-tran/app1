ChangeSecurityQuestions

Mit dem Vorgang „ChangeSecurityQuestions“ werden Sicherheitsfragen und Antworten eines Benutzers aktualisiert.

 

Parameter

Erforderlich/optional

Beschreibung

Eingabe

 

 

username

Erforderlich

Dies ist der Benutzername des zu aktualisierenden Benutzers.

 

userSecurityQuestions SecurityQuestion1

Erforderlich

Geben Sie die erste Sicherheitsfrage des Benutzers an.

 

userSecurityQuestions SecurityAnswer1

Erforderlich

Geben Sie die Antwort auf die erste Sicherheitsfrage des Benutzers an.

 

userSecurityQuestions SecurityQuestion2

Erforderlich

Geben Sie die zweite Sicherheitsfrage des Benutzers an.

 

userSecurityQuestions SecurityAnswer2

Erforderlich

Geben Sie die Antwort auf die zweite Sicherheitsfrage des Benutzers an.

 

userSecurityQuestions SecurityQuestion3

Erforderlich

Geben Sie die dritte Sicherheitsfrage des Benutzers an.

 

userSecurityQuestions SecurityAnswer3

Erforderlich

Geben Sie die Antwort auf die dritte Sicherheitsfrage des Benutzers an.

 

userSecurityQuestions SecurityQuestion4

Erforderlich

Geben Sie die vierte Sicherheitsfrage des Benutzers an.

 

userSecurityQuestions SecurityAnswer4

Erforderlich

Geben Sie die Antwort auf die vierte Sicherheitsfrage des Benutzers an.

 

modifyUsername

Optional

Dies ist der Benutzername des Benutzers, auf den die Änderungen angewendet werden sollen.  Dieser Benutzername wird in den Protokollen für jede Änderung an der Datendatei berücksichtigt.

 

Ausgabe

 

 

Result

 

Es wird „WAHR“ zurückgegeben, wenn der Benutzer erfolgreich aktualisiert wurde, und „FALSCH“, wenn beim Aktualisieren des Benutzers ein Fehler aufgetreten ist.

 

error Code

 

Es wird ein Fehlercode zurückgegeben, wenn beim Aktualisieren des Benutzers ein Fehler auftritt.

 

error Description

 

Es wird eine Beschreibung des Fehlercodes zurückgegeben, wenn beim Aktualisieren des Benutzers ein Fehler auftritt.
