OATH-Token

Mit der Seite „OATH-Token“ können Sie ein Ihrem Konto ein OATH-Token zuordnen.  Dies ist eine Alternative zum Verwenden der mobilen Multi-Factor Authentication-Anwendung für OATH-Codes.

 

So geben Sie ein OATH-Token an:

  1. Geben Sie die entsprechenden Werte ein.

  2. Klicken Sie auf die Schaltfläche Speichern.

 

Element

Beschreibung

Seriennummer

Geben Sie die auf der Rückseite des OATH-Tokens aufgedruckte Seriennummer ein.

 

Code

Geben Sie den Code ein, der derzeit auf dem OATH-Token angezeigt wird.
