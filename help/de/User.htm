Benutzer

Auf der Seite „Benutzer“ werden Informationen zum Benutzer sowie Optionen zum Ändern des Benutzerkontos angezeigt.  Die Seite wird angezeigt, wenn auf der Seite Benutzer suchen ein Benutzer gefunden wird oder wenn auf der Seite Benutzer suchen mehrere Benutzer gefunden werden und auf einen Benutzerlink geklickt wird.

 

Der Benutzername und die Telefonnummer des Benutzers werden in der Kopfzeile angezeigt. Möglicherweise werden die Sicherheitsfragen angezeigt, damit Sie die Identität des Benutzers prüfen können, bevor Änderungen vorgenommen werden.  Geben Sie einfach die Antworten des Benutzers ein, und klicken Sie auf die Schaltfläche „Überprüfen“.  Wenn der Benutzer keine Sicherheitsfragen definiert hat, wird dazu eine Meldung angezeigt.

 

Für Änderungen, die Sie am Konto des Benutzers vornehmen können, werden Links angezeigt.  Je nach den dem Administratorkonto zugewiesenen Berechtigungen und der Multi-Factor Authentication-Gesamtkonfiguration werden möglicherweise nicht alle Links angezeigt.

 

Element

Beschreibung

Einmalumgehung

Klicken Sie auf den LinkEinmalumgehung, um eine  Einmalumgehung für den ausgewählten Benutzer einzuleiten.  Der Benutzer kann sich einmal anmelden, ohne einen Multi-Factor Authentication-Anruf zu erhalten.

 

Methode ändern

Klicken Sie auf den Link Methode ändern, um die Authentifizierungsmethode des ausgewählten Benutzers zu ändern.

 

Modus ändern

Klicken Sie auf den Link Modus ändern, um den Modus des ausgewählten Benutzers zu ändern.

 

Anschluss ändern

Klicken Sie auf den Link Anschluss ändern, um die Telefonnummer des ausgewählten Benutzers zu ändern.

 

PIN ändern

Klicken Sie auf den Link PIN ändern, um die PIN des ausgewählten Benutzers zu ändern.

 

Hinweis: Dieser Link ist nur verfügbar, wenn sich der Benutzer im PIN-Modus für Telefonanruf, im Modus „Bidirektionales OTP + PIN“ für SMS oder im PIN-Modus für die mobile Anwendung befindet.

 

Sprache ändern

Klicken Sie auf den Link Sprache ändern, um die Sprache des ausgewählten Benutzers zu ändern.

 

Stimmabdruck zurücksetzen

Klicken Sie auf den Link Stimmabdruck zurücksetzen, um den Stimmabdruck des ausgewählten Benutzers zurückzusetzen.

 

Benutzerinformationen

Klicken Sie auf den Link Benutzerinformationen, um den Namen oder die E-Mail-Adresse des ausgewählten Benutzers zu ändern.

 

OATH-Token

Klicken Sie auf den Link OATH-Token, um das Drittanbieter-OATH-Token des ausgewählten Benutzers zu ändern.

 

Aktivieren/Deaktivieren

Klicken Sie auf den Link Aktivieren/Deaktivieren, um das Multi-Factor Authentication-Konto des ausgewählten Benutzers zu aktivieren oder zu deaktivieren.

 

Hinweis: Ein deaktivierter Benutzer kann sich möglicherweise ohne Multi-Factor Authentication anmelden.

 

Benutzer sperren/zulassen

Klicken Sie auf den Link Benutzer sperren/zulassen, um den ausgewählten Benutzer zu sperren oder zuzulassen.

 

Benutzer entfernen

Klicken Sie auf den Link Benutzer entfernen, um den ausgewählten Benutzer endgültig zu entfernen.

 

Hinweis: Ein entfernter Benutzer kann sich möglicherweise ohne Multi-Factor Authentication anmelden.

 

Authentifizierung testen

Klicken Sie auf den Link Authentifizierung testen, um für den ausgewählten Benutzer eine Authentifizierung mittels Multi-Factor Authentication-Anruf oder Multi-Factor Authentication-SMS zu testen.

 

Mobile Anwendung aktivieren

Klicken Sie auf den Link Mobile Anwendung aktivieren, um für den ausgewählten Benutzer einen Aktivierungscode für mobile Anwendungen zu generieren.

 

Mobile Anwendung deaktivieren

Klicken Sie auf den Link Mobile Anwendung deaktivieren, um für den ausgewählten Benutzer ein vorhandenes Gerät zu deaktivieren.
