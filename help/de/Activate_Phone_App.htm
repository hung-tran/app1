Mobile Anwendung aktivieren

Auf der Seite „Mobile Anwendung aktivieren“ können Sie einen Aktivierungscode generieren, um die mobile Multi-Factor Authentication-Anwendung auf Ihrem Gerät zu aktivieren.  Wenn Sie über Geräte verfügen, die bereits aktiviert wurden, werden sie aufgelistet und können bei Bedarf deaktiviert werden.

 

So aktivieren Sie die mobile Multi-Factor Authentication-Anwendung auf Ihrem Gerät:

  1. Klicken Sie auf die Schaltfläche Aktivierungscode generieren.

  2. Geben Sie den Aktivierungscode und die URL in die mobile Multi-Factor Authentication-Anwendung auf Ihrem Gerät ein, um die Aktivierung abzuschließen.
    Hinweis:  

    • Wenn ein QR-Code angezeigt wird, können Sie eine Anwendung zum Einlesen von QR-Codes verwenden, um die mobile Multi-Factor Authentication-Anwendung automatisch und mit aufgefülltem Aktivierungscode sowie URL zu starten.

    • Wenn Sie mit einem Browser auf Ihrem Gerät auf die mobile Multi-Factor Authentication-Anwendung zugreifen, wird anstelle des Aktivierungscodes und der URL ein Aktivierungslink angezeigt.  Wenn Sie auf den Aktivierungslink klicken, wird die mobile Multi-Factor Authentication-Anwendung automatisch und mit aufgefülltem Aktivierungscode sowie URL gestartet.

 

So deaktivieren Sie ein Gerät:

  1. Klicken Sie auf den Link Deaktivieren neben dem Gerät.

  2. Bestätigen Sie die Deaktivierung, indem Sie auf OK klicken.
