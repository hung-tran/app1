Anschluss ändern

Auf der Seite „Anschluss ändern“ können Sie die Rufnummer für das ausgewählte Benutzerkonto ändern.  Die Änderung ist dauerhaft, und die neue Rufnummer wird angerufen bzw. es wird eine SMS an sie gesendet, um Multi-Factor Authentication fortzusetzen.

 

So ändern Sie die Rufnummer für den ausgewählten Benutzer:

  1. Wählen Sie die entsprechende Landeskennzahl aus.

  2. Geben Sie eine neue Rufnummer ein.

  3. Klicken Sie auf die Schaltfläche Speichern.

 

Element

Beschreibung

Landeskennzahl

Wählen Sie die entsprechende Landeskennzahl aus der Dropdownliste aus. Die Standardeinstellung ist die aktuelle Landeskennzahl des Benutzers.

 

Hinweis: Einige internationale Anrufe sind kostenpflichtig.  Wenn Ihr Unternehmen diese Option nicht implementiert hat, wird bei einigen Landeskennzahlen kein Anruf ausgeführt.

 

Anschluss

Geben Sie die Rufnummer ein, die bei der Authentifizierung von Multi-Factor Authentication angerufen bzw. an die eine SMS gesendet werden soll. Es kann sich um eine Handynummer, eine Festnetznummer oder um weitere Rufnummern für die Telefonanrufmethode handeln.  Bei der SMS-Methode muss es sich um einen Anschluss handeln, der SMS-fähig ist.  Das Format der Telefonnummer muss mit dem Format übereinstimmen, das im Feld „Landeskennzahl“ ausgewählt wurde.

 

Beispiel: Bei Telefonnummern aus den USA und Kanada müssen Sie eine zehnstellige Nummer eingeben (z. B. 555-123-4567).

 

Durchwahl

Geben Sie die Durchwahl ein, die nach der Rufbeantwortung gewählt werden soll.  Es sind Ziffern, Kommas, * und # zulässig.  Kommas können für 1-sekündige Pausen zum Navigieren in Telefonsystemmenüs verwendet werden.  Das Ausfüllen dieses Felds ist optional.

 

Landeskennzahl der Ersatzrufnummer

Wählen Sie die entsprechende Landeskennzahl der Ersatzrufnummer aus.

 

Ersatzanschluss

Geben Sie die Rufnummer ein, die von Multi-Factor Authentication angerufen wird, wenn der Anruf am primären Telefon nicht angenommen wird.

 

Durchwahl der Ersatzrufnummer

Geben Sie die Durchwahl der Ersatzrufnummer ein.

 

E-Mail senden

Hier wird angegeben, ob eine E-Mail gesendet werden soll, um den Benutzer über die Aktualisierung zu informieren.

 

Hinweis: Das Kontrollkästchen „E-Mail senden“ wird nur angezeigt, wenn E-Mails für Ihr Unternehmen konfiguriert wurden.
