Benutzereinrichtung

Wenn Ihr Konto nicht vollständig eingerichtet ist, müssen Sie ggf. die Telefonnummer und/oder PIN angeben, die mit Ihrem Konto verwendet werden soll, oder Sie werden aufgefordert, zum Aktivieren der mobilen Multi-Factor Authentication-Anwendung einen Aktivierungscode zu generieren.  Wenn es Ihr Unternehmen zulässt, können Sie zwischen der Authentifizierung mittels Telefonanruf, SMS, mobiler Anwendung und OATH-Token wählen.  Die Gültigkeit der von Ihnen angegebenen Telefonnummer wird bestätigt, indem ein Multi-Factor Authentication-Anruf unter dieser Nummer getätigt, eine SMS an diese Nummer gesendet oder eine Benachrichtigung an die mobile Anwendung gesendet wird oder indem Sie aufgefordert werden, einen Authentifizierungscode einzugeben.  Bei der Telefonanrufmethode werden Sie aufgefordert, # zu drücken und je nach dem für Sie gültigen Telefonanrufmodus Ihre PIN einzugeben oder Ihren Stimmabdruck zu registrieren.  Bei der SMS-Methode erhalten Sie ein Einmalkennwort in einer SMS, auf die Sie mit dem Einmalkennwort oder mit dem Einmalkennwort und Ihrer PIN antworten.  Bei der Methode mit der mobilen Anwendung erhalten Sie in der mobilen Anwendung eine Benachrichtigung, mit der Sie aufgefordert werden, sich zu authentifizieren. Hierzu müssen Sie je nach dem Modus der mobilen Anwendung möglicherweise Ihre PIN eingeben.  Bei der OATH-Token-Methode werden Sie vom Benutzerportal aufgefordert, einen Authentifizierungscode einzugeben.  Bei einer erfolgreichen Authentifizierung werden Sie möglicherweise aufgefordert, Sicherheitsfragen zu erstellen.

 

So schließen Sie die Benutzereinrichtung für die Telefonanrufmethode ab:

  1. Geben Sie die Landeskennzahl und die Telefonnummer ein.  Sie können ggf. auch eine Ersatzanschlussnummer eingeben.

  2. Geben Sie ggf. Ihre PIN ein.

  3. Klicken Sie auf die Schaltfläche Jetzt per Sprachanruf authentifizieren.

 

So schließen Sie die Benutzereinrichtung für die SMS-Methode ab:

  1. Geben Sie die Landeskennzahl und die Telefonnummer ein.

  2. Geben Sie ggf. Ihre PIN ein.

  3. Klicken Sie auf die Schaltfläche Jetzt per SMS authentifizieren.

 

So schließen Sie die Benutzereinrichtung für die Methode mit der mobilen Anwendung ab:

  1. Installieren Sie die mobile Anwendung auf Ihrem Telefon.

  2. Klicken Sie auf die Schaltfläche Aktivierungscode generieren.

  3. Geben Sie den Aktivierungscode und die URL in der mobilen Anwendung ein, um die Aktivierung abzuschließen.

  4. Geben Sie ggf. Ihre PIN ein.

  5. Klicken Sie auf die Schaltfläche Jetzt authentifizieren.

 

So schließen Sie die Benutzereinrichtung für die OATH-Token-Methode ab:

  1. Installieren Sie die mobile Anwendung auf Ihrem Telefon.

  2. Klicken Sie auf die Schaltfläche Aktivierungscode generieren.

  3. Geben Sie den Aktivierungscode und die URL in der mobilen Anwendung ein, um die Aktivierung abzuschließen.

  4. Klicken Sie auf die Schaltfläche Jetzt authentifizieren.

ODER

  1. Geben Sie die auf der Rückseite des OATH-Tokens aufgedruckte Seriennummer ein.

  2. Geben Sie den Code ein, der derzeit auf dem OATH-Token angezeigt wird.

  3. Klicken Sie auf die Schaltfläche OATH-Token überprüfen.

 

 

Element

Beschreibung

Methode

Wählen Sie die Authentifizierung mittels Telefonanruf, SMS, mobile Anwendung oder OATH-Token aus.  Bei der Telefonanrufmethode erhalten Sie einen Multi-Factor Authentication-Telefonanruf, mit dem Sie aufgefordert werden, # zu drücken und Ihre PIN einzugeben oder Ihren Stimmabdruck zu registrieren.  Bei der SMS-Methode erhalten Sie ein Einmalkennwort in einer SMS, auf die Sie mit dem Einmalkennwort oder mit dem Einmalkennwort und Ihrer PIN antworten.  Bei der Methode mit der mobilen Anwendung erhalten Sie in der mobilen Anwendung eine Benachrichtigung, mit der Sie aufgefordert werden, sich zu authentifizieren. Hierzu müssen Sie möglicherweise Ihre PIN eingeben.  Bei der OATH-Token-Methode werden Sie vom Benutzerportal aufgefordert, einen Authentifizierungscode einzugeben.

 

Hinweis:   Mit der Methode, die Sie hier auswählen, greifen Sie später auf die geschützte Multi-Factor Authentication-Anwendung zu.  Sie können diese Methode auf der Seite Authentifizierungsmethode ändern später jederzeit ändern.

 

Landeskennzahl

Wählen Sie die entsprechende Landeskennzahl aus der Dropdownliste aus. Wenn für das Konto bereits eine Telefonnummer konfiguriert wurde, ist die Standardeinstellung die Landeskennzahl für diese Telefonnummer.  Andernfalls wird die Standardeinstellung auf „Deutschland +49“ festgelegt.

 

Hinweis: Einige internationale Anrufe sind kostenpflichtig.  Wenn Ihr Administrator diese Option nicht implementiert hat, wird bei einigen Landeskennzahlen kein Anruf ausgeführt.

 

Anschluss

Geben Sie die Rufnummer ein, die bei der Authentifizierung von Multi-Factor Authentication angerufen bzw. an die eine SMS gesendet werden soll. Hierbei kann es sich um eine Mobiltelefonnummer, Festnetznummer oder eine andere Nummer im Telefonanrufmodus handeln.  Das Telefongerät muss zur Verwendung des SMS-Modus über Funktionen zum Empfangen und Senden von SMS verfügen.  Das Format der Telefonnummer muss mit dem Format übereinstimmen, das im Feld „Landeskennzahl“ ausgewählt wurde.

 

Beispiel: Bei Telefonnummern aus den USA und Kanada müssen Sie eine zehnstellige Nummer eingeben (z. B. 555-123-4567).

 

Durchwahl

Geben Sie die Durchwahl ein, die nach der Rufbeantwortung gewählt werden soll.  Es sind Ziffern, Kommas, * und # zulässig.  Kommas können für 1-sekündige Pausen zum Navigieren in Telefonsystemmenüs verwendet werden.  Das Ausfüllen dieses Felds ist optional.

 

Landeskennzahl der Ersatzrufnummer

Wählen Sie die entsprechende Landeskennzahl der Ersatzrufnummer aus.

 

Ersatzanschluss

Geben Sie die Rufnummer ein, die von Multi-Factor Authentication angerufen wird, wenn der Anruf am primären Telefon nicht angenommen wird.

 

Durchwahl der Ersatzrufnummer

Geben Sie die Durchwahl der Ersatzrufnummer ein.

 

PIN

Geben Sie eine PIN ein.  Diese PIN wird bei jedem Authentifizierungsanruf, mit dem Einmalkennwort als Antwort auf jede SMS oder in der mobilen Anwendung zum Abschließen der Authentifizierung eingegeben.  Vom Administrator wurden möglicherweise PIN-Regeln konfiguriert, etwa eine Mindestlänge.

 

Seriennummer

Geben Sie die auf der Rückseite des OATH-Tokens aufgedruckte Seriennummer ein.

 

Code

Geben Sie den Code ein, der derzeit auf dem OATH-Token angezeigt wird.

 

 

Klicken Sie auf die Schaltfläche Abbrechen, um zur Seite Anmelden zurückzukehren.
