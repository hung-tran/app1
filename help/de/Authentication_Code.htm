Authentifizierungscode

Beim OATH-Token-Modus werden Sie über das Benutzerportal aufgefordert, den in der mobilen Multi-Factor Authentication-Anwendung oder im Token angezeigten Authentifizierungscode einzugeben.

 

So schließen Sie die Anmeldung ab:

  1. Geben Sie den in der mobilen Multi-Factor Authentication-Anwendung oder im Token angezeigten Authentifizierungscode ein.

  2. Klicken Sie auf die Schaltfläche Authentifizieren.

 

Klicken Sie auf die Schaltfläche Abbrechen, um zur Seite Anmelden zurückzukehren.
