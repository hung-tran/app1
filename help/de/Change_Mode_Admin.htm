Methode ändern

Auf der Seite „Methode ändern“ können Sie die Methode für das ausgewählte Benutzerkonto ändern.  Bei der Methode „Telefonanruf“ erhält der Benutzer beim Authentifizieren einen Telefonanruf.  Bei der Methode „SMS“ erhält der Benutzer beim Authentifizieren eine SMS.  Bei der Methode „Mobile Anwendung“ erhält der Benutzer eine Benachrichtigung in der mobilen Multi-Factor Authentication-Anwendung, die auf seinem Gerät installiert ist.  Bei der Methode „OATH-Token“ wird der Benutzer zum Eingeben eines Authentifizierungscodes aufgefordert.

 

So ändern Sie die Methode für den ausgewählten Benutzer:

  1. Wählen Sie die gewünschte Methode aus.

  2. Klicken Sie auf die Schaltfläche Speichern.

Methode

Wählen Sie die entsprechende Methode aus der Dropdownliste aus. Die Standardeinstellung ist die aktuelle Methode des Benutzers.

 

E-Mail senden

Hier wird angegeben, ob eine E-Mail gesendet werden soll, um den Benutzer über die Aktualisierung zu informieren.

 

Hinweis: Das Kontrollkästchen „E-Mail senden“ wird nur angezeigt, wenn E-Mails für Ihr Unternehmen konfiguriert wurden.
