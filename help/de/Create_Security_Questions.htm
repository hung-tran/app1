Erstellen von Sicherheitsfragen

Wenn Sie sich zum ersten Mal beim Benutzerportal anmelden, müssen Sie möglicherweise vier Sicherheitsfragen erstellen, bevor Sie den Vorgang fortsetzen können.

 

So erstellen Sie Ihre Sicherheitsfragen:

  1. Wählen Sie die Fragen aus der Dropdownliste aus, und geben Sie die gewünschten Antworten ein.

  2. Klicken Sie auf die Schaltfläche Weiter.

 

Element

Beschreibung

Sicherheitsfrage 1

Wählen Sie in der Dropdownliste die erste Frage aus.

 

Antwort 1

Geben Sie die Antwort zur ersten Sicherheitsfrage ein. Bei diesem Eintrag wird die Groß-/Kleinschreibung nicht berücksichtigt.

 

Sicherheitsfrage 2

Wählen Sie in der Dropdownliste die zweite Frage aus.  Die Frage darf keine Kopie einer vorherigen Frage sein.

 

Antwort 2

Geben Sie die Antwort zur zweiten Sicherheitsfrage ein. Bei diesem Eintrag wird die Groß-/Kleinschreibung nicht berücksichtigt.

 

Sicherheitsfrage 3

Wählen Sie in der Dropdownliste die dritte Frage aus.  Die Frage darf keine Kopie einer vorherigen Frage sein.

 

Antwort 3

Geben Sie die Antwort zur dritten Sicherheitsfrage ein. Bei diesem Eintrag wird die Groß-/Kleinschreibung nicht berücksichtigt.

 

Sicherheitsfrage 4

Wählen Sie in der Dropdownliste die vierte Frage aus.  Die Frage darf keine Kopie einer vorherigen Frage sein.

 

Antwort 4

Geben Sie die Antwort zur vierten Sicherheitsfrage ein. Bei diesem Eintrag wird die Groß-/Kleinschreibung nicht berücksichtigt.

 

 

Klicken Sie auf die Schaltfläche Abbrechen, um zur Seite Anmelden zurückzukehren.
