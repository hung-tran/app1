Token OATH

Stránka Token OATH vám umožní přidružit vybraného uživatele k tokenu OATH třetí strany.  Token musí být definován na serveru Multi-Factor Authentication Server, aby ho bylo možné přidružit k uživateli.  Token musí podporovat kódy OATH s časovou synchronizací.  Jde o alternativu k použití mobilní aplikace Multi-Factor Authentication pro kódy OATH.

 

Zadání tokenu OATH:

  1. Zadejte odpovídající hodnoty.

  2. Klikněte na tlačítko Uložit.
