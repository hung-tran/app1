사용자

[사용자] 페이지에는 사용자에 대한 정보와 사용자 계정을 수정할 수 있는 옵션이 표시됩니다.  이 페이지는 사용자 찾기 페이지에서 한 명의 사용자가 발견된 경우에 표시되고, 여러 명의 사용자가 발견된 경우에는 사용자 찾기 페이지에서 사용자 링크 하나를 클릭하면 표시됩니다.

 

페이지 머리글에 사용자의 사용자 이름과 전화 번호가 표시되고, 변경하기 전에는 사용자의 ID를 확인하기 위해 있도록 보안 질문이 표시될 수도 있습니다.  사용자의 답변을 입력하고 [확인] 단추만 클릭하면 됩니다.  사용자가 보안 질문을 정의하지 않은 경우 해당 내용에 대한 메시지가 표시됩니다.

 

사용자 계정을 수정할 수 있는 링크가 표시됩니다.  관리자 계정에 부여된 권한 및 전체 Multi-Factor Authentication 구성에 따라 일부 링크는 표시되지 않을 수 있습니다.

 

기능

설명

원타임 바이패스

선택한 사용자의 일회성 바이패스를 시작하려면 일회성 바이패스  링크를 선택합니다.  사용자가 한 번은 Multi-Factor Authentication 전화를 받지 않고 로그온할 수 있습니다.

 

방법 변경

선택한 사용자의 방법을 수정하려면 방법 변경 링크를 선택합니다.

 

모드 변경

선택한 사용자의 모드를 수정하려면 모드 변경 링크를 선택합니다.

 

폰 변경

선택한 사용자의 전화 번호를 수정하려면 폰 변경 링크를 선택합니다.

 

PIN 변경

선택한 사용자의 PIN 번호를 수정하려면 PIN 변경 링크를 선택합니다.

 

참고: 이 링크는 사용자가 전화 통화 PIN 모드, 문자 메시지 OTP + PIN 모드 또는 모바일 앱 PIN 모드일 때만 사용할 수 있습니다.

 

언어 변경

선택한 사용자의 언어를 수정하려면 언어 변경 링크를 선택합니다.

 

성문 다시 설정

선택한 사용자의 성문을 다시 설정하려면 성문 다시 설정 링크를 선택합니다.

 

사용자 정보

선택한 사용자의 이름 또는 전자 메일 주소를 수정하려면 사용자 정보 링크를 선택합니다.

 

OATH 토큰

선택한 사용자의 타사 OATH 토큰을 수정하려면 OATH 토큰 링크를 선택합니다.

 

사용/사용 안 함

선택한 사용자의 Multi-Factor Authentication 계정을 사용하거나 사용하지 않도록 설정하려면 사용 또는 사용 안 함 링크를 선택합니다.

 

참고: 사용하지 않도록 설정된 사용자가 Multi-Factor Authentication 없이 로그온할 수도 있습니다.

 

사용자 차단/차단 해제

선택한 사용자를 차단 또는 차단 해제하려면 사용자 차단/차단 해제 링크를 선택합니다.

 

사용자 제거

선택한 사용자를 영구히 제거하려면 사용자 제거 링크를 선택합니다.

 

참고: 제거된 사용자가 Multi-Factor Authentication 없이 로그온할 수도 있습니다.

 

인증 테스트

선택한 사용자에 대해 Multi-Factor Authentication 통화 또는 Multi-Factor Authentication 문자 메시지 인증을 테스트하려면 인증 테스트 링크를 선택합니다.

 

모바일 앱 활성화

선택한 사용자에 대해 모바일 앱 활성화 코드를 생성하려면 모바일 앱 활성화 링크를 선택합니다.

 

모바일 앱 비활성화

선택한 사용자의 기존 장치를 비활성화하려면 모바일 앱 비활성화 링크를 선택합니다.
