로그인

사용자 포털에 로그인하려면 사용자 이름과 암호를 입력합니다.  전화 번호로 Multi-Factor Authentication 전화가 걸려오거나 문자 메시지가 전송되고, 모바일 앱에 알림이 전송되거나, 인증 코드를 묻는 메시지가 표시됩니다.  인증을 완료하면 기본 페이지로 돌아갑니다.

 

계정 설정을 완료하지 않으면 사용자 설정 페이지로 이동됩니다.

 

사용자 포털에 처음 로그인하는 경우 보안 질문을 만들라는 메시지가 나타날 수 있습니다.

 

Multi-Factor Authentication이 거부되면 로그인을 완료하기 위해 보안 질문에 답변하라는 메시지가 나타날 수 있습니다.

 

로그인하려면:

  1. 사용자 이름과 암호를 입력합니다.

  2. 로그인 단추를 클릭합니다.

설명

사용자 이름

일반적으로 Multi-Factor Authentication 보호된 응용 프로그램에 로그인하는 데 사용하는 사용자 이름입니다.  Windows 사용자 이름인 경우가 많습니다.

 

암호

일반적으로 Multi-Factor Authentication 보호된 응용 프로그램에 로그인하는 데 사용하는 암호입니다.  Windows 암호인 경우가 많습니다.
