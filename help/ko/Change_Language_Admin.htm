언어 변경

[언어 변경] 페이지에서는 선택한 사용자의 계정에 대해 전화 통화, 문자 메시지 및 모바일 앱의 언어를 변경할 수 있습니다.  사용 가능한 언어는 회사의 구성에 따라 달라집니다.

 

선택한 사용자의 언어를 변경하려면:

  1. 전화 통화, 문자 메시지 및 모바일 앱에 사용할 적절한 언어를 선택합니다.

  2. 저장 단추를 클릭합니다.

 

기능

설명

전화 통화 언어

드롭다운 목록에서 적절한 언어를 선택합니다. Multi-Factor Authentication 통화 시 음성 안내가 이 언어로 들립니다.  기본값은 사용자의 현재 전화 통화 언어입니다.

 

문자 메시지 언어

드롭다운 목록에서 적절한 언어를 선택합니다.  문자 메시지의 내용이 이 언어로 작성됩니다.  기본값은 사용자의 현재 문자 메시지 언어입니다.

 

모바일 앱 언어

드롭다운 목록에서 적절한 언어를 선택합니다.  인증 시에 Multi-Factor Authentication 모바일 앱에 표시되는 문자가 이 언어로 작성됩니다.  기본값은 사용자의 현재 모바일 앱 언어입니다.
