OATH 토큰

[OATH 토큰] 페이지에서는 OATH 토큰과 사용자의 계정을 연결할 수 있습니다.  이 방법은 OATH 코드를 받기 위해 Multi-Factor Authentication 모바일 앱을 사용하는 대신 사용할 수 있는 방법입니다.

 

OATH 토큰을 지정하려면:

  1. 적절한 값을 입력합니다.

  2. 저장 단추를 클릭합니다.

 

기능

설명

일련 번호

일반적으로 OATH 토큰의 뒷면에 나와 있는 일련 번호를 입력합니다.

 

코드

OATH 토큰에 현재 표시된 코드를 입력합니다.
