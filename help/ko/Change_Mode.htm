방법 변경

[방법 변경] 페이지에서는 방법을 변경할 수 있습니다.  전화 통화 방법의 경우 인증할 때 전화를 받게 됩니다.  문자 메시지 방법의 경우 인증할 때 문자 메시지를 받게 됩니다.  모바일 앱 방법의 경우 장치에 설치된 Multi-Factor Authentication 모바일 앱에서 알림을 받게 됩니다.  OATH 토큰 방법의 경우 인증 코드를 묻는 메시지가 표시됩니다.

 

방법을 변경하려면:

  1. 적절한 방법을 선택합니다.

  2. 저장 단추를 클릭합니다.
