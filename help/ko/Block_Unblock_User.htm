사용자 차단/차단 해제

[사용자 차단/차단 해제] 페이지에서는 사용자를 차단 또는 차단 해제할 수 있습니다.  이 페이지는 사용자의 현재 차단 상태에 따라 달라집니다.  사용자의 차단 상태는 Multi-Factor Authentication POP에서 유지 관리됩니다.  차단된 사용자가 인증하려고 하면 자동으로 거부됩니다.  

 

선택한 사용자를 차단하려면:

  1. 차단 이유를 지정합니다.

  2. 사용자를 차단하려면 확인 단추를 클릭합니다.

 

기능

설명

차단 이유

사용자가 차단되는 이유를 입력합니다.

 

 

선택한 사용자를 차단 해제하려면:

  1. 차단 해제 이유를 지정합니다.

  2. 사용자를 차단 해제하려면 확인 단추를 클릭합니다.

 

기능

설명

차단 해제 이유

사용자가 차단 해제되는 이유를 입력합니다.
