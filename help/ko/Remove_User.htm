사용자 제거

[사용자 제거] 페이지에서는 선택한 사용자를 영구히 제거할 수 있습니다. 제거된 사용자는 Multi-Factor Authentication 서버의 응용 프로그램 구성에 따라 액세스가 허용되거나 거부될 수 있습니다.  응용 프로그램이 [사용자 일치 필요]로 설정된 경우 제거된 사용자는 더 이상 Multi-Factor Authentication 서버의 사용자 목록에 없기 때문에 액세스가 거부됩니다.  응용 프로그램이 [사용자 일치 필요]로 설정되어 있지 않으면 제거된 사용자가 Multi-Factor Authentication 없이 인증에 성공합니다.

 

선택한 사용자를 영구히 제거하려면 확인 단추를 클릭합니다.
