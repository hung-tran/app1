폰 변경

[폰 변경] 페이지에서는 선택한 사용자 계정의 전화 번호를 변경할 수 있습니다.  변경 내용은 영구적이며 이후에는 Multi-Factor Authentication을 위해 새로 지정된 전화 번호로 전화를 걸거나 문자 메시지를 보내게 됩니다.

 

선택한 사용자의 전화 번호를 변경하려면:

  1. 적절한 국가 코드를 선택합니다.

  2. 새 전화 번호를 입력합니다.

  3. 저장 단추를 클릭합니다.

 

기능

설명

국가 코드

드롭다운 목록에서 적절한 국가 코드를 선택합니다. 기본값은 사용자의 현재 국가 코드입니다.

 

참고: 일부 국제 전화의 경우 요금이 부과됩니다.  회사에서 이 옵션을 구현하지 않은 경우에는 일부 국가 코드로 전화를 걸 수 없습니다.

 

전화

인증 시 Multi-Factor Authentication에서 전화를 걸거나 문자를 보낼 전화 번호를 입력합니다. 전화 통화 방법에는 모바일 폰 번호, 일반 전화 번호 또는 다른 몇 가지 번호를 사용할 수 있습니다.  문자 메시지 방법을 사용하려면 폰에 문자 메시지 기능이 있어야 합니다.  전화 번호 형식은 국가 코드 필드에서 선택한 형식과 일치해야 합니다.

 

예: 미국 및 캐나다 전화 번호의 경우 10자리 전화 번호(예: 555-123-4567)를 입력해야 합니다.

 

내선 번호

전화를 받은 후 누를 내선 번호를 입력합니다.  숫자, 쉼표, * 및 #이 허용됩니다.  쉼표는 폰 시스템 메뉴를 탐색하기 위한 1초 일시 중지에 사용될 수 있습니다.  이 필드는 선택 사항입니다.

 

예비 국가 코드

예비 전화 번호에 적절한 국가 코드를 선택합니다.

 

예비 폰

기본 전화에 건 전화를 받지 않을 경우에 Multi-Factor Authentication에서 전화를 걸 전화 번호를 입력합니다.

 

예비 내선 번호

예비 전화 번호의 내선 번호를 입력합니다.

 

전자 메일 보내기

사용자에게 업데이트를 알리는 전자 메일을 보낼지 여부를 지정합니다.

 

참고: [전자 메일 보내기] 확인란은 회사의 전자 메일이 구성된 경우에만 표시됩니다.
