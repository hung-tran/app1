모드 변경

[모드 변경] 페이지에서는 선택한 사용자 계정의 모드를 변경할 수 있습니다.  전화 통화, 문자 메시지, 모바일 앱 방법은 각각 다양한 모드를 지원합니다.

 

참고:  제품 구성에 따라 일부 방법에는 [모드] 드롭다운 목록이 표시되지 않을 수 있습니다.

 

선택한 사용자의 모드를 변경하려면:

  1. 적절한 모드를 선택합니다.

  2. 저장 단추를 클릭합니다.

 

기능

설명

전화 통화 모드

드롭다운 목록에서 적절한 모드를 선택합니다. 기본값은 사용자의 현재 전화 통화 모드입니다.

 

문자 메시지 모드

드롭다운 목록에서 적절한 방향과 모드를 선택합니다.  기본값은 사용자의 현재 문자 메시지 방향과 모드입니다.

 

모바일 앱 모드

드롭다운 목록에서 적절한 모드를 선택합니다.  기본값은 사용자의 현재 모바일 앱 모드입니다.

 

전자 메일 보내기

사용자에게 업데이트를 알리는 전자 메일을 보낼지 여부를 지정합니다.

 

참고: [전자 메일 보내기] 확인란은 회사의 전자 메일이 구성된 경우에만 표시됩니다.
