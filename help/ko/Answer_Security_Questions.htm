보안 질문에 답변

Multi-Factor Authentication이 거부되면 보안 질문에 답변하라는 메시지가 나타납니다.  성공하면 기본 페이지로 돌아갑니다.

 

참고:   4개의 보안 질문 중 임의로 선택된 일부만 나타날 수 있습니다.

 

보안 질문에 답변하고 로그인을 완료하려면:

  1. 각 질문에 대한 답변을 입력합니다.

  2. 로그인 단추를 클릭합니다.

 

기능

설명

보안 질문 1

첫 번째 보안 질문에 답변합니다. 이 항목은 대/소문자를 구분하지 않습니다.

 

보안 질문 2

두 번째 보안 질문에 답변합니다. 이 항목은 대/소문자를 구분하지 않습니다.

 

보안 질문 3

세 번째 보안 질문에 답변합니다. 이 항목은 대/소문자를 구분하지 않습니다.

 

보안 질문 4

네 번째 보안 질문에 답변합니다. 이 항목은 대/소문자를 구분하지 않습니다.

 

 

로그인 페이지로 돌아가려면 취소 단추를 클릭합니다.
