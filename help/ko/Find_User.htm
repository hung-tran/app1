사용자 찾기

[사용자 찾기] 페이지에서는 관리자가 기존 사용자 계정을 검색할 수 있습니다.

 

사용자를 찾으려면:

  1. 검색 조건을 지정합니다.

  2. 찾기 단추를 클릭합니다.

 

입력한 조건과 일치하는 사용자가 한 명이면 해당 사용자의 사용자 정보 페이지가 표시됩니다.  일치하는 사용자가 두 명 이상이면 사용자 목록이 링크로 표시됩니다.  사용자 정보 페이지로 이동하려면 적절한 사용자 링크를 클릭합니다.

 

기능

설명

사용자 이름

일치하는 사용자 이름을 검색하기 위한 검색 문자열을 입력합니다.

 

참고: 검색 문자열에 여러 문자에 해당하는 별표(*)를 지정하여 와일드카드를 사용할 수 있습니다.

 

이름

일치하는 이름을 검색하기 위한 검색 문자열을 입력합니다.

 

성

일치하는 성을 검색하기 위한 검색 문자열을 입력합니다.

 

전자 메일 주소

일치하는 전자 메일 주소를 검색하기 위한 검색 문자열을 입력합니다.

 

사용자 그룹

일치하는 사용자 그룹을 검색하기 위한 검색 문자열을 입력합니다.

 

전화

일치하는 전화 번호를 검색하기 위한 검색 문자열을 입력합니다.

 

태그

모든 사용자를 검색하려면 모두 를 선택하고 특정 태그가 있는 사용자만 검색하려면 특정 태그를 선택합니다.
