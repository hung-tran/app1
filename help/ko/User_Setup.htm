사용자 설정

계정 설정을 완료하지 않은 경우 계정에 사용할 전화 번호(해당하는 경우) 및/또는 PIN(해당하는 경우)을 지정해야 합니다. 그렇지 않으면 Multi-Factor Authentication 모바일 앱을 활성화할 활성화 코드를 생성하라는 메시지가 나타날 수 있습니다.  회사에서 허용하는 경우 전화 통화, 문자 메시지, 모바일 앱, OATH 토큰 방법 중에서 선택할 수 있습니다.  지정한 전화 번호가 올바른지 확인하기 위해 해당 전화 번호로 Multi-Factor Authentication 전화가 걸려오거나 문자 메시지가 전송되고, 모바일 앱에 알림이 전송되거나 인증 코드를 묻는 메시지가 표시됩니다.  전화 통화 방법의 경우 전화 통화 모드에 따라 #를 누르고 PIN을 입력하거나 #를 누르고 성문을 등록하라는 안내를 받게 됩니다.  문자 메시지 방법의 경우 OTP(일회성 암호)를 받으면 OTP 또는 OTP + 사용자의 PIN을 포함하여 회신해야 합니다.  모바일 앱 방법의 경우 모바일 앱으로 알림이 전송되고 인증하라는 메시지가 나타납니다. 모바일 앱 모드에 따라 인증 시 PIN을 입력해야 할 수 있습니다.  OATH 토큰 방법의 경우 사용자 포털에서 인증 코드를 묻는 메시지가 표시됩니다.  성공할 경우 보안 질문을 만들라는 메시지가 나타날 수 있습니다.

 

전화 통화 방법에 대한 사용자 설정을 완료하려면:

  1. 국가 코드와 전화 번호를 입력합니다.  예비 전화 번호(해당하는 경우)를 입력할 수도 있습니다.

  2. PIN(해당 하는 경우)을 입력합니다.

  3. 지금 전화로 인증 받기 단추를 클릭합니다.

 

문자 메시지 방법에 대한 사용자 설정을 완료하려면:

  1. 국가 코드와 전화 번호를 입력합니다.

  2. PIN(해당 하는 경우)을 입력합니다.

  3. 지금 문자로 인증 받기 단추를 클릭합니다.

 

모바일 앱 방법에 대한 사용자 설정을 완료하려면:

  1. 폰에서 모바일 앱을 설치합니다.

  2. 활성화 코드 생성 단추를 클릭합니다.

  3. 모바일 앱에 활성화 코드와 URL을 입력하여 활성화를 완료합니다.

  4. PIN(해당 하는 경우)을 입력합니다.

  5. 지금 인증 단추를 클릭합니다.

 

OATH 토큰 방법에 대한 사용자 설정을 완료하려면:

  1. 폰에서 모바일 앱을 설치합니다.

  2. 활성화 코드 생성 단추를 클릭합니다.

  3. 모바일 앱에 활성화 코드와 URL을 입력하여 활성화를 완료합니다.

  4. 지금 인증 단추를 클릭합니다.

또는

  1. 일반적으로 OATH 토큰의 뒷면에 나와 있는 일련 번호를 입력합니다.

  2. OATH 토큰에 현재 표시된 코드를 입력합니다.

  3. OATH 토큰 확인 단추를 클릭합니다.

방법

전화 통화, 문자 메시지, 모바일 앱, OATH 토큰 방법 중에서 선택합니다.  전화 통화 방법의 경우 #를 누르고 PIN을 입력하거나 #를 누르고 성문을 등록하라는 Multi-Factor Authentication 전화를 받게 됩니다.  문자 메시지 방법의 경우 문자 메시지로 OTP(일회성 암호)가 전송되고 OTP 또는 OTP + 사용자의 PIN으로 회신하게 됩니다.  모바일 앱 방법의 경우 모바일 앱으로 알림이 전송되고 인증하라는 메시지가 나타납니다. 인증 시 PIN을 입력해야 할 수 있습니다.  OATH 토큰 방법의 경우 사용자 포털에서 인증 코드를 묻는 메시지가 표시됩니다.

 

참고:    선택한 방법은 Multi-Factor Authentication 보호된 응용 프로그램에 액세스할 때 사용됩니다.  방법은 나중에 언제든지 방법 변경 페이지에서 변경할 수 있습니다.

 

국가 코드

드롭다운 목록에서 적절한 국가 코드를 선택합니다. 계정에 대해 이미 전화 번호가 구성된 경우 기본값은 해당 전화 번호의 국가 코드입니다.  아직 전화 번호가 구성되지 않은 경우 기본값은 [미국 및 캐나다 +1]로 설정되어 있습니다.

 

참고: 일부 국제 전화의 경우 요금이 부과됩니다.  관리자가 이 옵션을 구현하지 않은 경우에는 일부 국가 코드로 전화를 걸 수 없습니다.

 

전화

인증 시 Multi-Factor Authentication에서 전화를 걸거나 문자를 보낼 전화 번호를 입력합니다. 전화 통화 모드에서는 모바일 폰 번호, 일반 전화 번호 또는 다른 몇 가지 번호를 사용할 수 있습니다.  문자 메시지 모드를 지정하려면 폰에 문자 메시지 기능이 있어야 합니다.  전화 번호 형식은 국가 코드 필드에서 선택한 형식과 일치해야 합니다.

 

예: 미국 및 캐나다 전화 번호의 경우 10자리 전화 번호(예: 555-123-4567)를 입력해야 합니다.

 

내선 번호

전화를 받은 후 누를 내선 번호를 입력합니다.  숫자, 쉼표, * 및 #이 허용됩니다.  쉼표는 폰 시스템 메뉴를 탐색하기 위한 1초 일시 중지에 사용될 수 있습니다.  이 필드는 선택 사항입니다.

 

예비 국가 코드

예비 전화 번호에 적절한 국가 코드를 선택합니다.

 

예비 폰

기본 전화에 건 전화를 받지 않을 경우에 Multi-Factor Authentication에서 전화를 걸 전화 번호를 입력합니다.

 

예비 내선 번호

예비 전화 번호의 내선 번호를 입력합니다.

 

PIN

PIN 번호를 입력합니다.  이 PIN은 인증을 완료하기 위해 각 인증 통화 시에 입력하거나, 각 문자 메시지에 대한 회신에 OTP(일회성 암호)와 함께 입력하거나, 모바일 앱에 입력하게 됩니다.  관리자가 PIN 규칙(예: 최소 길이)을 구성했을 수 있습니다.

 

일련 번호

일반적으로 OATH 토큰의 뒷면에 나와 있는 일련 번호를 입력합니다.

 

코드

OATH 토큰에 현재 표시된 코드를 입력합니다.

 

 

로그인 페이지로 돌아가려면 취소 단추를 클릭합니다.
