모바일 앱 활성화

[모바일 앱 활성화] 페이지에서는 장치의 Multi-Factor Authentication 모바일 앱을 활성화하기 위해 활성화 코드를 생성할 수 있습니다.  이미 장치를 활성화한 경우 목록에 나열되며 필요한 경우 비활성화할 수 있습니다.

 

장치의 Multi-Factor Authentication 모바일 앱을 활성화하려면:

  1. 활성화 코드 생성 단추를 클릭합니다.

  2. 장치의 Multi-Factor Authentication 모바일 앱에 활성화 코드와 URL을 입력하여 활성화를 완료합니다.
    참고:  

    • QR 코드가 표시되면 QR 코드 스캐너 앱을 사용하여 활성화 코드와 URL이 입력된 Multi-Factor Authentication 모바일 앱을 자동으로 시작할 수 있습니다.

    • 장치의 브라우저에서 Multi-Factor Authentication 사용자 포털에 액세스하면 활성화 코드와 URL 대신 [활성화] 링크가 표시됩니다.  [활성화] 링크를 클릭하면 활성화 코드와 URL이 입력된 Multi-Factor Authentication 모바일 앱이 자동으로 시작됩니다.

 

장치를 비활성화하려면:

  1. 장치 옆에 있는 비활성화 링크를 클릭합니다.

  2. 확인을 클릭하여 비활성화를 확인합니다.
