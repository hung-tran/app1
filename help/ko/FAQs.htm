Multi-Factor Authentication™의 작동 원리는 어떻게 됩니까?

Multi-Factor Authentication은 확인 전화를 걸거나, 문자 메시지를 보내거나, 로그인할 때 Multi-Factor Authentication 모바일 앱에 알림을 보내거나, Multi-Factor Authentication 모바일 앱 또는 토큰에서 인증 코드를 묻는 메시지를 표시하는 방식으로 작동합니다.

 

1단계:

평소의 사용자 이름과 암호를 입력하십시오.

2단계:

즉시 전화가 걸려오거나, 문자 메시지를 받거나, 모바일 앱 알림을 받거나, 인증 코드를 묻는 메시지가 나타납니다. 전화 통화 방법의 경우 전화를 받고 #를 누른 다음, PIN을 입력하거나 성문 암호를 말합니다.  문자 메시지 방법의 경우 문자 메시지에 OTP(일회성 암호)를 포함하여 회신하거나 OTP + 사용자의 PIN을 포함하여 회신합니다.  모바일 앱 방법의 경우 장치에 설치된 Multi-Factor Authentication 모바일 앱에서 [인증]을 누르거나 PIN을 입력하고 [인증]을 누릅니다.  OATH 토큰 방법의 경우 Multi-Factor Authentication 모바일 앱 또는 토큰에 표시되어 있는 인증 코드를 입력합니다.

이것이 전부입니다!

 

이 간단한 프로세스를 통해 별도의 두 가지 인증 방법을 제공할 수 있으며, 컴퓨터와 폰 서비스라는 별도의 두 채널이 사용됩니다. 전화 통화 모드에서는 일반 전화기 또는 모바일 폰에서 작동하고, 문자 메시지 모드에서는 문자 메시지 기능이 있는 폰에서 작동하며, 모바일 앱 모드에서는 Multi-Factor Authentication 모바일 앱을 지원하는 장치에서 작동합니다.

 

내 폰을 분실하는 경우 어떻게 합니까?

[전화 번호 변경] 옵션을 선택하고 새 전화 번호를 입력하십시오. ID가 확실히 설정되면 지원 센터에 전화하여 대체 번호를 설정할 수도 있습니다.

 

휴대폰 신호 수신지에서 벗어나는 경우 어떻게 해야 합니까?

[전화 번호 변경] 옵션을 선택하여 대체 전화 번호를 가리키도록 계정을 변경할 수 있습니다. 일부 회사에서는 전화 통화나 문자 메시지 없이 한 번 로그인할 수 있는 [일회성 바이패스] 옵션도 제공합니다. 가능한 경우 이 기능을 사용하려면 [일회성 바이패스] 옵션을 선택하십시오. 지원이 필요한 경우 아무 전화기나 사용하여 지원 센터에 전화할 수 있습니다.  OATH 토큰 방법은 휴대폰 신호 수신지가 필요하지 않습니다.

 

로그인하려는 상황이 아닌데 Multi-Factor Authentication에서 전화를 받거나, 문자 메시지를 받거나, 모바일 앱 알림을 받는 경우 어떻게 해야 합니까?

이는 암호를 이미 알고 있는 다른 사람이 사용자의 계정으로 로그인을 시도하는 경우에만 발생합니다. 사용자 이름과 암호가 확인된 후에만 전화 통화가 걸려오고, 문자 메시지가 전송되고, 모바일 앱 알림이 전송된다는 점에 유의하십시오. 따라서 이러한 상황이 발생하는 경우 Multi-Factor Authentication이 불법 액세스로부터 사용자 계정을 보호합니다! 계정에 대한 인증 시도를 일시적으로 차단하려면 전화 통화 방법의 경우 폰 메뉴에서 음성 안내에 따르고, 문자 메시지 방법의 경우 회사에서 할당한 사기 행위 코드가 포함된 문자에 회신하고, 모바일 앱의 경우 [거부 및 사기 행위 보고]를 누르십시오.  회사에서 [사기 행위 보고]를 사용하도록 설정한 경우 사용자의 계정을 차단할 뿐 아니라 회사의 IT 보안 팀에 즉시 경고를 보냅니다. 계정을 차단 해제하려면 지원 센터에 문의하십시오.
