사용/사용 안 함

[사용/사용 안 함] 페이지에서는 선택한 사용자를 사용하거나 사용하지 않도록 설정할 수 있습니다. 사용하도록 설정된 사용자의 구성이 완전하면 해당 사용자는 Multi-Factor Authentication 전화를 받기 시작합니다.  사용하도록 설정된 사용자의 구성이 불완전하면 사용자 포털에서 해당 계정을 활성화할 수 있습니다.  사용하지 않도록 설정된 사용자는 더 이상 Multi-Factor Authentication 전화를 받지 않게 되며, 사용자의 사용 안 함 동작에 따라 액세스가 허용되거나 거부될 수 있습니다.  전자 메일이 구성된 경우 사용자를 사용하도록 설정할 때 사용자에게 전자 메일을 보내는 옵션이 표시됩니다.

 

선택한 사용자를 사용하거나 사용하지 않도록 설정하려면 확인 단추를 클릭합니다.
