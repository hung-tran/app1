사용자 추가

[사용자 추가] 페이지에서는 새 Multi-Factor Authentication 계정을 만들 수 있습니다.

 

참고:  방법(전화 통화, 문자 메시지 또는 모바일 앱)을 비롯한 여러 사용자 설정은 Multi-Factor Authentication 서버의 [회사 설정] 섹션에서 지정한 사용자 기본값에 따라 기본적으로 설정됩니다.

 

새 사용자를 추가하려면:

  1. 사용자 정보를 입력합니다.

  2. 추가 단추를 클릭합니다.

 

기능

설명

사용자 이름

사용자 이름은 일반적으로 적절한 전자 메일 형식의 사용자 전자 메일입니다.

 

예: chuck@abc.com

 

이름

사용자의 이름을 입력합니다.

 

성

사용자의 성을 입력합니다.

 

전자 메일 주소

사용자의 전자 메일 주소를 입력합니다.  이 전자 메일 주소로 전자 메일을 보냅니다.  전자 메일 주소를 지정하지 않은 경우 사용자 이름이 적절한 전자 메일 형식이면 사용자 이름으로 전자 메일을 보냅니다.

 

사용자 그룹

사용자의 사용자 그룹을 입력합니다.

 

국가 코드

드롭다운 목록에서 적절한 국가 코드를 선택합니다. 기본값은 [미국 및 캐나다 +1]로 설정되어 있습니다.

 

참고: 일부 국제 전화의 경우 요금이 부과됩니다.  회사에서 이 옵션을 구현하지 않은 경우에는 일부 국가 코드로 전화를 걸 수 없습니다.

 

전화

인증 시 Multi-Factor Authentication에서 전화를 걸거나 문자를 보낼 전화 번호를 입력합니다. 전화 통화 방법에는 모바일 폰 번호, 일반 전화 번호 또는 다른 몇 가지 번호를 사용할 수 있습니다.  문자 메시지 방법을 사용하려면 폰에 문자 메시지 기능이 있어야 합니다.  전화 번호 형식은 국가 코드 필드에서 선택한 형식과 일치해야 합니다.

 

예: 미국 및 캐나다 전화 번호의 경우 10자리 전화 번호(예: 555-123-4567)를 입력해야 합니다.

 

내선 번호

전화를 받은 후 누를 내선 번호를 입력합니다.  숫자, 쉼표, * 및 #이 허용됩니다.  쉼표는 폰 시스템 메뉴를 탐색하기 위한 1초 일시 중지에 사용될 수 있습니다.  이 필드는 선택 사항입니다.

 

예비 국가 코드

예비 전화 번호에 적절한 국가 코드를 선택합니다.

 

예비 폰

기본 전화에 건 전화를 받지 않을 경우에 Multi-Factor Authentication에서 전화를 걸 전화 번호를 입력합니다.

 

예비 내선 번호

예비 전화 번호의 내선 번호를 입력합니다.

 

PIN

사용자의 PIN을 입력합니다.  또는 [생성] 단추를 클릭하여 PIN을 생성할 수도 있습니다.  PIN은 PIN 최소 길이 및 약한 PIN에 대한 규칙을 준수해야 합니다.

 

[생성] 단추

[생성] 단추는 임의로 생성된 PIN을 제공합니다.  생성된 PIN의 길이는 4자리나 사용자의 최소 PIN 길이보다 큽니다.

 

PIN 확인

[PIN] 필드에 입력했던 PIN을 다시 입력합니다. 확인하기 위해 입력하는 PIN은 정확하게 일치해야 합니다.

 

사용

사용자를 사용할지 여부를 지정합니다.  이 옵션은 기본적으로 선택되어 있습니다.

 

참고: 사용자 레코드를 사용하지 않도록 설정한 경우 해당 사용자가 로그온할 때 Multi-Factor Authentication 전화를 받지 못하고, 사용 안 함 동작에 대한 회사 기본값에 따라 응용 프로그램에 대한 액세스가 허용되거나 거부될 수 있습니다.

 

PIN 변경 필요

다음 Multi-Factor Authentication 통화 시 사용자가 PIN을 변경해야 하도록 지정합니다.

 

참고: [PIN 변경 필요] 확인란은 회사 구성에서 사용자가 PIN을 변경할 수 있도록 허용된 경우에만 표시됩니다.

 

전자 메일 보내기

추가된 사용자에게 전자 메일을 보낼지 여부를 지정합니다.

 

참고: [전자 메일 보내기] 확인란은 회사의 전자 메일이 구성된 경우에만 표시됩니다.
