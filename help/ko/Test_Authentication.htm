인증 테스트

[인증 테스트] 페이지에서는 선택한 사용자에 대해 Multi-Factor Authentication 통화, 문자 메시지 인증 또는 Multi-Factor Authentication 모바일 앱 인증을 테스트할 수 있습니다.  전화를 걸지, 문자 메시지를 보낼지 또는 Multi-Factor Authentication 모바일 앱에 알림을 보낼지는 사용자의 모드에 따라 결정됩니다.

 

통화를 시작하거나, 문자 메시지를 보내거나, Multi-Factor Authentication 모바일 앱에 알림을 보내려면 테스트 단추를 클릭합니다.
