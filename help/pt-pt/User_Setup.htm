Configuração do Utilizador

Se a configuração da cona estiver incompleta, deve especificar o Número de Telefone (caso se aplique) e/ou PIN (caso se aplique) para usar com a conta, ou poderá ser-lhe pedido para gerar um código de ativação para ativar a aplicação móvel da Multi-Factor Authentication.  Se a empresa permitir, pode escolher entre os métodos chamada telefónica, mensagem de texto, aplicação móvel e token OATH.  Será realizada uma chamada do Multi-Factor Authentication ou será enviada uma mensagem de texto para o número de telefone que especificar para confirmar que este é válido, será enviada uma notificação para a aplicação móvel ou ser-lhe-á solicitado um código de Autenticação.  Para o método de chamada telefónica, ser-lhe-á pedido para premir #, introduzir o PIN ou inscrever-se com a impressão de voz, dependendo do modo de chamada telefónica.  Para o método de mensagem de texto, ser-lhe-á enviada uma palavra-passe monouso (OTP) e responderá com o OTP ou o OTP + o PIN.  Para o método de aplicação móvel, ser-lhe-á enviada uma notificação para a aplicação móvel que irá solicitar que se autentique, o que pode exigir que introduza o PIN, dependendo do modo de aplicação móvel.  Para o método de token OATH, o Portal de Utilizador solicitar-lhe-á um código de Autenticação.  Se tiver êxito, poderá ser-lhe solicitado para criar as suas perguntas de segurança.

 

Para concluir a configuração do utilizador para o método de chamada telefónica:

  1. Introduza o indicativo do país e o número de telefone.  Também pode introduzir um número de telefone alternativo (se aplicável).

  2. Introduza o PIN (se aplicável).

  3. Clique no botão Telefonar-me Agora para Autenticar.

 

Para concluir a configuração do utilizador para o método de mensagem de texto:

  1. Introduza o indicativo do país e o número de telefone.

  2. Introduza o PIN (se aplicável).

  3. Clique no botão Enviar-me Agora SMS para Autenticar.

 

Para concluir a configuração do utilizador para o método de aplicação móvel:

  1. Instale a aplicação móvel no telefone.

  2. Clique no botão Gerar Código de Ativação.

  3. Introduza o código de ativação e o URL da aplicação móvel para concluir a ativação.

  4. Introduza o PIN (se aplicável).

  5. Clique no botão Autenticar Agora.

 

Para concluir a configuração do utilizador para o método token OATH:

  1. Instale a aplicação móvel no telefone.

  2. Clique no botão Gerar Código de Ativação.

  3. Introduza o código de ativação e o URL da aplicação móvel para concluir a ativação.

  4. Clique no botão Autenticar Agora.

OU

  1. Introduza o número de série, impresso na parte de trás do token OATH.

  2. Introduza o código atualmente apresentado no token OATH.

  3. Clique no botão Verificar Token OATH.

 

 

Funcionalidade

Descrição

Método

Selecione o método Chamada Telefónica, Mensagem de Texto, Aplicação Móvel ou token OATH.  Para o método de chamada telefónica, irá receber uma chamada do Multi-Factor Authentication, pedindo-lhe que prima #, introduza o PIN ou se inscreva com a impressão de voz.  Para o método de mensagem de texto, ser-lhe-á enviada uma palavra-passe monouso (OTP) numa mensagem de texto e responderá com o OTP ou o OTP + o PIN.  Para o método de aplicação móvel, ser-lhe-á enviada uma notificação para a aplicação móvel que irá solicitar que se autentique, o que pode exigir que introduza o PIN.  Para o método de token OATH, o Portal de Utilizador solicitar-lhe-á um código de Autenticação.

 

Nota:   O método selecionado é o método que irá utilizar quando aceder à aplicação protegida por Multi-Factor Authentication.  O método pode sempre ser alterado posteriormente, na página Alterar Método.

 

Indicativo de País

Selecione o indicativo de país adequado na lista pendente. Se já tiver sido configurado um número de telefone para a conta, a predefinição é o indicativo de país desse número de telefone.  Caso contrário, a predefinição está definida como Estados Unidos e Canadá + 1.

 

Nota: Algumas chamadas internacionais poderão ser cobradas.  Se o administrador não tiver implementado esta opção, não será efetuada uma chamada para alguns indicativos de país.

 

Telefone

Introduza o número de telefone que a Multi-Factor Authentication irá contactar por chamada ou texto durante a autenticação. Isto pode ser um número de telemóvel, número de telefone fixo ou outro número para o método de chamada telefónica.  Tem de ser um telefone com capacidades para mensagens de texto no modo de mensagem de texto.  O formato de número de telefone tem de corresponder ao formato selecionado no campo Indicativo de País.

 

Exemplo: Para números de telefone dos EUA e Canadá, tem de introduzir um número de telefone de 10 dígitos (por exemplo, 555-123-4567).

 

Extensão

Introduza a extensão que será marcada depois de a chamada ser atendida.  São permitidos dígitos, vírgulas, * e #.  É possível utilizar vírgulas para uma pausa de um segundo para navegar nos menus do sistema telefónico.  Este campo é opcional.

 

Indicativo de País Alternativo

Selecione o indicativo de país adequado para o número de telefone alternativo.

 

Telefone Alternativo

Introduza o número de telefone que o Multi-Factor Authentication deverá ligar se a chamada para o telefone principal não for atendida.

 

Extensão Alternativa

Introduza a extensão do número de telefone alternativo.

 

PIN

Introduza um número PIN.  Este PIN será introduzido durante cada chamada de autenticação, introduzido com a palavra-passe monouso (OTP) em resposta a cada mensagem de texto ou introduzido na aplicação móvel para concluir a autenticação.  O administrador poderá ter configurado regras PIN, tais como um comprimento mínimo.

 

Número de Série

Introduza o número de série, impresso na parte de trás do token OATH.

 

Código

Introduza o código atualmente apresentado no token OATH.

 

 

Para voltar à página de Início de Sessão, clique no botão Cancelar.
