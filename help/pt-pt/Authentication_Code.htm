Código de Autenticação

Para o modo Token OATH, ser-lhe-á pedido no Portal de utilizador o código de autenticação apresentado na aplicação móvel ou token do Multi-Factor Authentication.

 

Para concluir o início de sessão:

  1. Introduza o código de autenticação apresentado na aplicação móvel ou token da Multi-Factor Authentication.

  2. Clique no botão Autenticar.

 

Para voltar à página de Início de Sessão, clique no botão Cancelar.
