Início de Sessão

Para iniciar sessão no Portal de Utilizador, introduza o nome de utilizador e palavra-passe.  Será realizada uma chamada telefónica de Multi-Factor Authentication, será enviada uma mensagem de texto para o seu número de telemóvel, será enviada uma notificação para a aplicação móvel de Multi-Factor Authentication ou ser-lhe-á pedido o código de autenticação.  Se concluir a autenticação com êxito, será conduzido para a página principal.

 

Se o programa de configuração de conta estiver incompleto, será direcionado para a página Configuração do Utilizador.

 

Se esta for a primeira vez que inicia sessão no Portal de Utilizador, poderá ser-lhe solicitado para criar as suas perguntas de segurança.

 

Se a Multi-Factor Authentication for recusada, ser-lhe-á pedido para responder a perguntas de segurança para completar o início de sessão.

 

Para iniciar sessão:

  1. Introduza o nome de utilizador e a palavra-passe.

  2. Clique no botão Iniciar Sessão.

 

Funcionalidade

Descrição

Nome de utilizador

O nome de utilizador que normalmente utiliza para iniciar sessão na aplicação protegida de Multi-Factor Authentication.  Muitas vezes, este é o nome de utilizador do Windows.

 

Palavra-passe

A palavra-passe que normalmente utiliza para iniciar sessão na aplicação protegida de Multi-Factor Authentication.  Muitas vezes, esta é a palavra-passe de utilizador do Windows.
