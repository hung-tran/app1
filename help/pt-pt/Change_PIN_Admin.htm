Alterar PIN

Na página Alterar PIN pode alterar o PIN da conta do utilizador seleccionado.  A alteração é permanente e o PIN recentemente especificado será utilizado daqui em diante no Multi-Factor Authentication.

 

Para alterar o PIN do utilizador selecionado:

  1. Introduza um PIN novo.

  2. Introduza o novo PIN novamente para confirmar.

  3. Clique no botão Guardar.

 

Funcionalidade

Descrição

Novo PIN

Escreva um PIN novo.  Ou pode gerar o PIN ao clicar no botão Gerar.  O PIN tem de respeitar as regras do Comprimento Mínimo do PIN e de PINs fracos.

 

Botão Gerar

O botão Gerar fornece um PIN gerado aleatoriamente.  O comprimento do PIN gerado será o maior de 4 dígitos ou o comprimento mínimo do PIN do utilizador.

 

Confirmar PIN

Volte a escrever o novo PIN para confirmar a alteração. O PIN de confirmação tem de corresponder exatamente.

 

Necessário Alterar PIN

Especifica que um utilizador tem de alterar o seu PIN durante a sua próxima Multi-Factor Authentication.

 

Nota: A caixa de verificação Necessário Alterar PIN só será apresentada se o utilizador tiver permissão para alterar o respectivo PIN.

 

Enviar Correio Eletrónico

Especifica se deve ser enviada uma mensagem de correio eletrónico para informar o utilizador da atualização.

 

Nota: A caixa de verificação Enviar Correio Eletrónico só será apresentada se a mensagem de correio eletrónico foi configurada para a sua empresa.
