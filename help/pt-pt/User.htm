Utilizador

A página do Utilizador apresenta informações sobre o utilizador e opções para modificar a conta de utilizador.  A página é apresentada quando é encontrado um único utilizador na página Localizar Utilizador ou quando se clica numa ligação de utilizador na página Localizar Utilizador quando são localizados vários utilizadores.

 

São apresentados o nome de utilizador e número de telefone no cabeçalho da página e as suas perguntas de segurança podem ser apresentadas para que possa verificar a identidade do utilizador antes de fazer alterações.  Basta introduzir as respostas do utilizador e clicar no botão Verificar.  Se o utilizador não tiver definido as respetivas perguntas de segurança, irá ser apresentada uma mensagem para esse efeito.

 

São apresentadas hiperligações para as modificações que pode efetuar na conta do utilizador.  Nem todas as ligações poderão ser apresentadas, dependendo das permissões concedidas à sua conta de administrador e da configuração de Multi-Factor Authentication global.

 

Funcionalidade

Descrição

Omissão de Uso Individual

Selecione a ligação Omissão de Uso Individual, para ignorar uma vez o utilizador selecionado.  O utilizador conseguirá iniciar sessão uma única vez sem receber uma chamada do Multi-Factor Authentication.

 

Alterar Método

Selecione a hiperligação Alterar Método para alterar o método do utilizador selecionado.

 

Alterar Modo

Selecione a hiperligação Alterar Modo para alterar os modos do utilizador selecionado.

 

Alterar Telefone

Selecione a hiperligação Alterar Telefone para alterar o número de telefone do utilizador selecionado.

 

Alterar PIN

Selecione a hiperligação Alterar PIN para alterar o número PIN do utilizador selecionado.

 

Nota: Esta hiperligação apenas estará disponível quando o utilizador estiver em modo de chamada telefónica com PIN, mensagem de texto OTP + modo PIN, ou aplicação móvel com modo PIN.

 

Alterar Idioma

Selecione a hiperligação Alterar Idioma para alterar o idioma do utilizador selecionado.

 

Repor Impressão de Voz

Selecione a hiperligação Repor Impressão de Voz para repor a impressão de voz do utilizador seleccionado.

 

Informações do Utilizador

Selecione a hiperligação Informações do Utilizador para alterar o utilizador nome ou endereço eletrónico do utilizador.

 

Token OATH

Selecione a hiperligação Token OATH para alterar o token OATH de terceiros do utilizador.

 

Ativar/desativar

Selecione a hiperligação Ativar ou Desativar para ativar ou desativar a conta de Multi-Factor Authentication do utilizador selecionado.

 

Nota: Um utilizador desativado poderá conseguir iniciar sessão sem Multi-Factor Authentication.

 

Bloquear/desbloquear Utilizador

Seleccione a hiperligação Bloquear/Desbloquear Utilizador para bloquear ou desbloquear o utilizador seleccionado.

 

Remover Utilizador

Selecione a hiperligação Remover Utilizador para remover permanentemente o utilizador selecionado.

 

Nota: Um utilizador removido poderá conseguir iniciar sessão sem Multi-Factor Authentication.

 

Testar Autenticação

Selecione a hiperligação Testar Autenticação para testar uma chamada de Multi-Factor Authentication ou uma mensagem de texto de Multi-Factor Authentication para o utilizador selecionado.

 

Ativar Aplicação Móvel

Selecione a hiperligação Ativar Aplicação Móvel para gerar o código de ativação da aplicação móvel para o utilizador selecionado.

 

Desativar Aplicação Móvel

Selecione a hiperligação Desativar Aplicação Móvel para desativar um dispositivo existente para o utilizador selecionado.
