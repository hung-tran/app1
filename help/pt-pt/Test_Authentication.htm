Testar Autenticação

A página Testar Autenticação permite-lhe testar uma chamada de Multi-Factor Authentication, autenticação de mensagens de texto ou aplicação móvel da Multi-Factor Authentication para o utilizador selecionado.  O modo do utilizador irá definir se será efetuada uma chamada, enviada uma mensagem de texto ou se a aplicação móvel de Multi-Factor Authentication é notificada.

 

Clique no botão Teste para iniciar a chamada, enviar a mensagem de texto ou notificar a aplicação móvel da Multi-Factor Authentication.
