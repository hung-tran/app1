Ativar Aplicação Móvel

A página Ativar Aplicação Móvel, permite-lhe gerar o código de ativação da aplicação móvel para ativar no dispositivo a aplicação móvel de Multi-Factor Authentication.  Se já tiver dispositivos ativados, estes serão apresentados e podem ser desativados conforme necessário.

 

Para ativar a aplicação móvel de Multi-Factor Authentication no dispositivo:

  1. Clique no botão Gerar Código de Ativação.

  2. Para concluir a ativação, introduza o código de ativação e o URL na aplicação móvel da Multi-Factor Authentication no seu dispositivo.
    Nota:  

    • Se for apresentado um código QR, poderá usar uma aplicação de leitura do código QR para iniciar automaticamente a aplicação móvel da Multi-Factor Authentication com o código de ativação e o URL já preenchidos.

    • Se estiver a aceder a Multi-Factor Authentication no Portal de Utilizador usando um browser no seu dispositivo, será apresentada uma hiperligação em vez do código de ativação e URL.  Clicar na ligação Ativar iniciará automaticamente a aplicação móvel de Multi-Factor Authentication com o código de ativação e o URL preenchidos.

 

Para desativar um dispositivo:

  1. Clique na hiperligação Desativar junto do dispositivo.

  2. Confirmar a desativação, clicando em OK.
