Adicionar Utilizador

A página Adicionar Utilizador permite-lhe criar uma nova conta de Multi-Factor Authentication.

 

Nota:  Várias definições de utilizador, tal como método (Chamada Telefónica, Mensagem de Texto ou Aplicação Móvel), são predefinidas com base nas predefinições do utilizador especificadas na seção Definições da Empresa do Servidor Multi-Factor Authentication.

 

Para adicionar um novo utilizador:

  1. Introduza os detalhes de utilizador.

  2. Clique no botão Adicionar.

 

Funcionalidade

Descrição

Nome de utilizador

O nome de utilizador é, geralmente, o correio eletrónico do utilizador no formato de correio eletrónico correto.

 

Exemplo: Chuck@abc.com

 

Nome Próprio

Introduza o nome próprio do utilizador.

 

Apelido

Introduza o apelido do utilizador.

 

Endereço de Correio Eletrónico

Introduza o endereço de correio eletrónico do utilizador.  As mensagens de correio eletrónico serão enviadas para este endereço de correio eletrónico.  Se não for especificado, as mensagens de correio eletrónico serão enviadas para o nome de utilizador, se este estiver no formato de correio eletrónico correto.

 

Grupo do Utilizador

Introduza o grupo de utilizadores do utilizador.

 

Indicativo de País

Selecione o indicativo de país adequado na lista pendente. A predefinição está definida como Estados Unidos e Canadá + 1.

 

Nota: Algumas chamadas internacionais poderão ser cobradas.  Se a sua empresa não tiver implementado esta opção, não será efetuada uma chamada para alguns indicativos de país.

 

Telefone

Introduza o número de telefone que a Multi-Factor Authentication irá contactar por chamada ou texto durante a autenticação. Este pode ser um número de telemóvel, número de telefone fixo ou outro número para o método de chamada telefónica.  Para o método de mensagens de texto, é necessário que o telefone tenha capacidades de mensagens de texto.  O formato de número de telefone tem de corresponder ao formato selecionado no campo Indicativo de País.

 

Exemplo: Para números de telefone dos EUA e Canadá, tem de introduzir um número de telefone de 10 dígitos (por exemplo, 555-123-4567).

 

Extensão

Introduza a extensão que será marcada depois de a chamada ser atendida.  São permitidos dígitos, vírgulas, * e #.  É possível utilizar vírgulas para uma pausa de um segundo para navegar nos menus do sistema telefónico.  Este campo é opcional.

 

Indicativo de País Alternativo

Selecione o indicativo de país adequado para o número de telefone alternativo.

 

Telefone Alternativo

Introduza o número de telefone que o Multi-Factor Authentication deverá ligar se a chamada para o telefone principal não for atendida.

 

Extensão Alternativa

Introduza a extensão do número de telefone alternativo.

 

PIN

Escreva o PIN do utilizador.  Ou pode gerar o PIN ao clicar no botão Gerar.  O PIN tem de respeitar as regras do Comprimento Mínimo do PIN e de PINs fracos.

 

Botão Gerar

O botão Gerar fornece um PIN gerado aleatoriamente.  O comprimento do PIN gerado será o maior de 4 dígitos ou o comprimento mínimo do PIN do utilizador.

 

Confirmar PIN

Escreva novamente o PIN que introduziu no campo PIN. O PIN de confirmação tem de corresponder exatamente.

 

Ativado

Especifica se o utilizador está ativado.  A predefinição é marcada.

 

Nota:Se um registo de utilizador não estiver ativado, então esse utilizador não irá receber chamadas de Multi-Factor Authentication quando iniciar sessão, e poder ter acesso ou o acesso poderá ser recusado à aplicação, dependendo das predefinições da empresa relativamente a comportamentos para desativado.

 

Necessário Alterar PIN

Especifica que um utilizador tem de alterar o seu PIN durante a sua próxima chamada de Multi-Factor Authentication.

 

Nota: A caixa de verificação Necessário Alterar PIN só será apresentada se a configuração da empresa permitir aos utilizadores alterarem o respectivo PIN.

 

Enviar Correio Eletrónico

Especifica se deve ser enviada uma mensagem de correio eletrónico para o utilizador adicionado.

 

Nota: A caixa de verificação Enviar Correio Eletrónico só será apresentada se a mensagem de correio eletrónico foi configurada para a sua empresa.
