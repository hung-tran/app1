Alterar Método

Na página Alterar Método pode alterar o Método da conta do utilizador selecionado.  Para o método de chamada telefónica, o utilizador irá receber uma chamada telefónica quando se autenticar.  Para o método de mensagem de texto, o utilizador irá receber uma mensagem de texto quando se autenticar.  Para o método de aplicação móvel, o utilizador receberá uma notificação na aplicação móvel de Multi-Factor Authentication instalada no respetivo dispositivo.  Para o método de token OATH, será solicitado ao utilizador um código de Autenticação.

 

Para alterar o Método do utilizador selecionado:

  1. Selecione o método apropriado.

  2. Clique no botão Guardar.

 

Funcionalidade

Descrição

Método

Selecione o método adequado na lista pendente. A predefinição é o atual método do utilizador.

 

Enviar Correio Eletrónico

Especifica se deve ser enviada uma mensagem de correio eletrónico para informar o utilizador da atualização.

 

Nota: A caixa de verificação Enviar Correio Eletrónico só será apresentada se a mensagem de correio eletrónico foi configurada para a sua empresa.
