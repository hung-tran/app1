Localizar Utilizador

A página Localizar Utilizador permite que um administrador procure uma conta de utilizador existente.

 

Para localizar um utilizador:

  1. Especificar critérios de procura.

  2. Clique no botão Localizar.

 

Se existir um único utilizador que corresponda ao critério introduzido, a página Informações do Utilizador será apresentada para esse utilizador.  Se corresponder a mais do que um utilizador, será apresentada uma lista de utilizadores como ligações.  Clique na ligação de utilizador adequada para ir para a página Informações do Utilizador.

 

Funcionalidade

Descrição

Nome de utilizador

Introduza a cadeia de procura para a correspondência do nome de utilizador.

 

Nota: são permitidos carateres universais na cadeia de procura, incluindo um asterisco (*), para corresponder a vários carateres.

 

Nome Próprio

Introduza a cadeia de procura para a correspondência do nome próprio.

 

Apelido

Introduza a cadeia de procura para a correspondência do apelido.

 

Endereço de Correio Eletrónico

Introduza a cadeia de procura para a correspondência do endereço de correio eletrónico.

 

Grupo do Utilizador

Introduza a cadeia de procura para a correspondência do grupo do utilizador.

 

Telefone

Introduza a cadeia de procura para a correspondência do telefone.

 

Tag

Selecione Todos  para corresponder a todos os utilizadores ou selecione uma tag específica para fazer corresponder apenas os utilizadores com essa tag.
