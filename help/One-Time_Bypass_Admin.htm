One-Time Bypass

The One-Time Bypass page allows you to bypass Multi-Factor Authentication for the selected user's next sign on. When you initiate a bypass, the user will be able to authenticate without Multi-Factor Authentication a single time.  The bypass expires if not used.

 

To initiate a one-time bypass for the selected user:

  1. Specify bypass seconds.

  2. Click the Confirm button to bypass Multi-Factor Authentication for the selected user's next sign on.

 

Feature

Description

Bypass Seconds

Enter the number of seconds during which the bypass can be used before it expires.
