Authentication Code

For OATH Token mode, the User Portal will prompt for the authentication code displayed in the Multi-Factor Authentication mobile app or token.

 

To complete your log in:

  1. Enter the authentication code displayed in the Multi-Factor Authentication mobile app or token.

  2. Click the Authenticate button.

 

To return to the Log In page, click the Cancel button.
