User Setup

If your account setup is incomplete, you must specify the Phone Number (if applicable) and/or PIN (if applicable) to use with your account or you may be prompted to generate an activation code to activate the Multi-Factor Authentication mobile app.  If your company allows it, you can choose between phone call, text message, mobile app, and OATH token methods.  A Multi-Factor Authentication call will be placed or a text message sent to the phone number you specify to confirm it is valid, a notification will be sent to the mobile app, or you will be prompted for an Authentication code.  For the phone call method, you will be prompted to press #, enter your PIN or enroll your voiceprint depending on your phone call mode.  For the text message method, you will be sent a one-time passcode (OTP) and will either reply with the OTP or the OTP + your PIN.  For the mobile app method, a notification will be sent to the mobile app and it will prompt you to authenticate which may require you enter your PIN depending on your mobile app mode.  For the OATH token method, you will be prompted for an Authentication code by the User Portal.  If successful, you may be prompted to create your security questions.

 

To complete your user setup for the phone call method:

  1. Enter your country code and phone number.  You may also enter a backup phone number (if applicable).

  2. Enter your PIN (if applicable).

  3. Click the Call Me Now to Authenticate button.

 

To complete your user setup for the text message method:

  1. Enter your country code and phone number.

  2. Enter your PIN (if applicable).

  3. Click the Text Me Now to Authenticate button.

 

To complete your user setup for the mobile app method:

  1. Install the mobile app on your phone.

  2. Click the Generate Activation Code button.

  3. Enter the activation code and URL in the mobile app to complete the activation.

  4. Enter your PIN (if applicable).

  5. Click the Authenticate Me Now button.

 

To complete your user setup for the OATH token method:

  1. Install the mobile app on your phone.

  2. Click the Generate Activation Code button.

  3. Enter the activation code and URL in the mobile app to complete the activation.

  4. Click the Authenticate Me Now button.

OR

  1. Enter the serial number printed on the back of the OATH token.

  2. Enter the code currently displayed on the OATH token.

  3. Click the Verify OATH Token button.

 

 

Feature

Description

Method

Select either Phone Call, Text Message, Mobile App, or OATH Token method.  For the phone call method, you will receive a Multi-Factor Authentication call prompting you to press #, enter your PIN or enroll your voiceprint.  For the text message method, you will be sent a one-time passcode (OTP) in a text message and will either reply with the OTP or the OTP + your PIN.  For the mobile app method, a notification will be sent to the mobile app and it will prompt you to authenticate which may require you enter your PIN.  For the OATH token method, you will be prompted for an Authentication code by the User Portal.

 

Note:  The method you select is the method you will use when accessing your Multi-Factor Authentication protected application.  The method can always be changed later on the Change Method page.

 

Country Code

Select the appropriate country code from the drop-down list. If a phone number has already been configured for your account, the default is the country code for that phone number.  Otherwise, the default is set to United States & Canada +1.

 

Note: Some international calls result in a charge.  If your administrator has not implemented this option, a call will not be made to some country codes.

 

Phone

Enter the phone number that Multi-Factor Authentication will call or text during authentication. This may be a mobile phone number, desk phone number, or some other number in phone call mode.  It must be a phone with text message capabilities in text message mode.  The phone number format must match the format selected in the Country Code field.

 

Example: For U.S. and Canadian phone numbers, you must enter a 10-digit phone number (e.g., 555-123-4567).

 

Extension

Enter the extension that will be dialed after the call is answered.  Digits, commas, *, and # are allowed.  Commas can be used for a one second pause to navigate phone system menus.  This field is optional.

 

Backup Country Code

Select the appropriate country code for the backup phone number.

 

Backup Phone

Enter the phone number that Multi-Factor Authentication will call if the call to the primary phone is not answered.

 

Backup Extension

Enter the extension for the backup phone number.

 

PIN

Enter a PIN number.  This PIN will be entered during each authentication call, entered with the one-time passcode (OTP) in reply to each text message, or entered in the mobile app in order to complete the authentication.  Your administrator may have configured PIN rules such as a minimum length.

 

Serial Number

Enter the serial number printed on the back of the OATH token.

 

Code

Enter the code currently displayed on the OATH token.

 

 

To return to the Log In page, click the Cancel button.
