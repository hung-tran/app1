Пользователь

На странице "Пользователь" отображаются сведения о пользователе и параметры изменения учетной записи пользователя.  Эта страница отображается, если на странице Find User (Поиск пользователя) найден один пользователь, или когда пользователь щелкнул ссылку на странице Find User (Поиск пользователя), если было найдено несколько пользователей.

 

Имя пользователя и его номер телефона отображаются в заголовке страницы; также могут отображаться вопросы безопасности, что позволяет проверить личность пользователя перед внесением изменений.  Просто введите ответы пользователя и нажмите кнопку Verify (Проверить).  Если пользователь не указал свои контрольные вопросы, на экране появится соответствующее сообщение.

 

Любые изменения, доступные для учетной записи пользователя, отображаются в виде ссылок.  Могут отображаться не все ссылки — это зависит от полномочий, предоставленных вашей учетной записи администратора и общими параметрами настройки Multi-Factor Authentication.

 

Компонент

Описание

Одноразовый обход проверки

Выберите Одноразовый обход проверки, чтобы инициализировать  одноразовый обход проверки для выбранного пользователя.  Пользователь получит возможность один раз выполнить вход без получения телефонного вызова от Multi-Factor Authentication.

 

Изменение метода

Перейдите по ссылке Изменение метода, чтобы изменить метод выбранного пользователя.

 

Изменение режима

Перейдите по ссылке Изменение режима, чтобы изменить режим выбранного пользователя.

 

Изменение телефона

Перейдите по ссылке Изменение телефона, чтобы изменить номер телефона выбранного пользователя.

 

Изменение ПИН-кода

Перейдите по ссылке Изменение ПИН-кода, чтобы изменить ПИН-код выбранного пользователя.

 

Примечание. Эта ссылка доступна только в том случае, если для пользователя действует режим ПИН-кода телефонного звонка или ПИН-кода мобильного приложения.

 

Изменение языка

Перейдите по ссылке Изменение языка, чтобы изменить язык выбранного пользователя.

 

Сброс голосового отпечатка

Перейдите по ссылке Сброс голосового отпечатка , чтобы сбросить голосовой отпечаток выбранного пользователя.

 

Сведения о пользователе

Перейдите по ссылке Сведения о пользователе, чтобы изменить имя пользователя или адрес его электронной почты.

 

Токен OATH

Перейдите по ссылке Токен OATH, чтобы изменить сторонний токен OATH выбранного пользователя.

 

Включение/отключение

Перейдите по ссылке Включение/отключение, чтобы включить или отключить учетную запись Multi-Factor Authentication выбранного пользователя.

 

Примечание. Отключенный пользователь сможет выполнить вход без Multi-Factor Authentication.

 

Блокировка/разблокировка пользователя

Перейдите по ссылке Блокировка/разблокировка пользователя, чтобы заблокировать или разблокировать выбранного пользователя.

 

Удаление пользователя

Перейдите по ссылке Удаление пользователя, чтобы удалить выбранного пользователя без возможности восстановления.

 

Примечание. Удаленный пользователь сможет выполнить вход без Multi-Factor Authentication.

 

Тестирование проверки подлинности

Перейдите по ссылке Тестирование проверки подлинности, чтобы протестировать телефонный вызов или текстовое сообщение Multi-Factor Authentication для проверки подлинности выбранного пользователя.

 

Активация мобильного приложения

Щелкните ссылку Активация мобильного приложения, чтобы создать код активации мобильного приложения для выбранного пользователя.

 

Деактивация мобильного приложения

Ссылка Деактивация мобильного приложения позволяет деактивировать существующее устройство для выбранного пользователя.
