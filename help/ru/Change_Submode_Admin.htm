Изменение режима

Страница "Изменение режима" позволяет изменить режимы для выбранной учетной записи пользователя.  Методы, связанные с использованием телефонных вызовов, SMS и мобильных приложений поддерживают разные режимы.

 

Примечание.  В зависимости от конфигурации раскрывающиеся списки режимов могут не отображаться для некоторых методов.

 

Чтобы изменить режимы выбранного пользователя, выполните следующие действия.

  1. Выберите соответствующие режимы.

  2. Нажмите кнопку Сохранить.

 

Компонент

Описание

Режим телефонного вызова

Выберите подходящий режим из раскрывающегося списка. По умолчанию используется текущий режим телефонного вызова пользователя.

 

Режим текстового сообщения

Выберите из раскрывающегося списка соответствующее направление текста и режим.  По умолчанию указаны текущее направление SMS и режим пользователя.

 

Режим мобильного приложения

Выберите подходящий режим из раскрывающегося списка.  По умолчанию используется текущий режим мобильного приложения пользователя.

 

Отправить сообщение

Указывает, что пользователю нужно отправить сообщение электронной почты с уведомлением об обновлении.

 

Примечание. Флажок "Отправить сообщение" будет отображаться только в том случае, если для вашей компании настроен адрес электронной почты.
