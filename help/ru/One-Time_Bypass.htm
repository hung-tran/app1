Одноразовый обход проверки

Страница "Однократный обход проверки" позволяет обходить Multi-Factor Authentication при следующем входе в систему. При инициализации обхода проверки этот пользователь сможет однократно пройти проверку подлинности без использования Multi-Factor Authentication.  Действие права на обход проверки завершается, если оно не используется.

 

Чтобы инициировать однократный обход, выполните следующие действия.

  1. Укажите длительность действия обхода в секундах.

  2. Нажмите кнопку Подтвердить, чтобы выбранный пользователь обошел Multi-Factor Authentication при следующем входе в систему.
