Портал пользователей Multi-Factor Authentication

Портал пользователей Multi-Factor Authentication позволяет обходить Multi-Factor Authentication для одного входа, управлять параметрами учетной записи Multi-Factor Authentication, включая метод проверки, номер телефона, ПИН-код, язык и параметры безопасности, а также активировать мобильное приложение Multi-Factor Authentication. Некоторые из этих функций могут быть отключены системным администратором.  Дополнительные сведения об использовании Multi-Factor Authentication см. в разделе Часто задаваемые вопросы.

 

Содержание, алфавитный указатель и средство поиска облегчают поиск информации в справочной системе.
