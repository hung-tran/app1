Поиск пользователя

Страница "Поиск пользователя" позволяет администратору выполнить поиск существующей учетной записи пользователя.

 

Чтобы найти пользователя, выполните следующие действия.

  1. Укажите критерии поиска.

  2. Нажмите кнопку Найти.

 

Если указанному критерию отвечает один пользователь, то на экране появится страница Сведения о пользователе для этого пользователя.  Если обнаружено несколько совпадений, то на экране появится список пользователей в виде ссылок.  Щелкните соответствующую ссылку на пользователя, чтобы перейти к странице User Information (Сведения о пользователе).

Имя пользователя

Примечание. В строке поиска разрешено использовать подстановочный знак "звездочка" (*), соответствующий нескольким символам.

 

Имя

Введите строку поиска для сопоставления имени.

 

Фамилия

Введите строку поиска для сопоставления фамилии.

 

Адрес электронной почты

Введите строку поиска для сопоставления адреса электронной почты.

 

Группа пользователей

Введите строку поиска для сопоставления группы пользователей.

 

Телефон

Введите строку поиска для соответствия телефона.

 

Тег

Выберите Все , чтобы сопоставлять всех пользователей, или выберите отдельный тег, чтобы сопоставлять только пользователей с этим тегом.
