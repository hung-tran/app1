Вход

Чтобы войти на портал пользователей, введите имя пользователя и пароль.  Сервер Multi-Factor Authentication сделает телефонный вызов, на пользовательский номер телефона будет отправлено SMS, на мобильное приложение Multi-Factor Authentication будет направлено уведомление или пользователю будет выдан запрос на ввод кода проверки подлинности.  Пользователь, успешно прошедший проверку подлинности, перенаправляется на главную страницу.

 

Если настройка учетной записи не завершена, пользователь перенаправляется на страницу Настройка пользователя.

 

Если пользователь входит на портал пользователей впервые, может быть выдан запрос на создание вопросов безопасности.

 

Если Multi-Factor Authentication завершилась отказом в доступе, то для пользователя может отобразиться запрос на ввод ответов на вопросы безопасности, что позволит завершить вход.

 

Чтобы войти в систему, выполните следующие действия.

  1. Введите имя пользователя и пароль.

  2. Нажмите кнопку Войти.

 

Компонент

Описание

Имя пользователя

Имя пользователя, обычно используемое для входа в приложение, защищенное с помощью Multi-Factor Authentication.  Часто это имя пользователя Windows.

 

Пароль

Пароль, обычно используемый для входа в приложение, защищенное с помощью Multi-Factor Authentication.  Часто это пароль для входа в Windows.
