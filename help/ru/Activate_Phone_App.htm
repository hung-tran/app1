Активация мобильного приложения

Страница "Активация мобильного приложения" позволяет создать код активации мобильного приложения для активации мобильного приложения Multi-Factor Authentication на вашем устройстве.  Если устройства уже активированы, появится список устройств и вы сможете их отключить при необходимости.

 

Чтобы активировать мобильное приложение Multi-Factor Authentication на устройстве, выполните следующие действия.

  1. Нажмите кнопку Создать код активации.

  2. Введите код и URL-адрес активации в мобильное приложение Multi-Factor Authentication на вашем устройстве, чтобы завершить активацию.
    Примечание.  

    • Если на экране отображается QR код, можно воспользоваться приложением для сканирования QR кодов, чтобы автоматически запустить мобильное приложение Multi-Factor Authentication с уже введенным кодом и URL-адресом активации.

    • В случае доступа к порталу пользователей Multi-Factor Authentication с браузера на вашем устройстве вместо кода и URL-адреса активации отображается ссылка "Активировать".  Переход по ссылке Activate (Активировать) приводит к автоматическому запуску мобильного приложения Multi-Factor Authentication с уже введенным кодом и URL-адресом активации.

 

Чтобы деактивировать устройство, выполните следующие действия.

  1. Щелкните ссылку Деактивировать рядом с устройством.

  2. Подтвердите отключение, нажав ОК.
