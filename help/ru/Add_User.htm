Добавление пользователя

Страница "Добавление пользователя" позволяет создать новую учетную запись Multi-Factor Authentication.

 

Примечание.  Многие параметры пользователя, такие как метод (телефонный вызов, текстовое сообщение и мобильное приложение) получают значения по умолчанию на основании пользовательских значений по умолчанию, которые указаны в разделе "Параметры компании" сервера Multi-Factor Authentication.

 

Чтобы добавить нового пользователя, выполните следующие действия.

  1. Введите сведения о пользователе.

  2. Нажмите кнопку Добавить.

 

Компонент

Описание

Имя пользователя

Как правило, имя пользователя представляет собой адрес электронной почты пользователя в правильном формате.

 

Пример. Chuck@ABC.com

 

Имя

Введите имя пользователя.

 

Фамилия

Введите фамилию пользователя.

 

Адрес электронной почты

Введите адрес электронной почты пользователя.  На этот адрес электронной почты будут отправляться сообщения электронной почты.  Если адрес не указан, то сообщения электронной почты будут отправляться на имя пользователя, если оно представляет собой адрес электронной почты в правильном формате.

 

Группа пользователей

Введите группу пользователей, в которую входит пользователь.

 

Код страны

В раскрывающемся списке выберите код соответствующей страны. По умолчанию для США и Канады установлен код +1.

 

Примечание. Некоторые международные вызовы тарифицируются.  Если компания не реализовала поддержку этого варианта, то телефонные вызовы не будут совершаться на некоторые коды стран.

 

Телефон

Введите номер телефона, на который сервер Multi-Factor Authentication сделает вызов или направит текстовое сообщение в ходе проверки подлинности. Этот номер может принадлежать мобильному телефону, настольному телефону или иному переговорному устройству.  Это устройство должно представлять собой телефон с поддержкой приема и отправки SMS, которые необходимы для метода на основе SMS.  Формат телефонного номера должен совпадать с форматом, выбранным в поле Country Code (Код страны).

 

Пример. Для телефонных номеров в США и Канаде необходимо ввести 10-значный номер телефона (например, 555-123-4567).

 

Добавочный номер

Введите добавочный номер, который будет набираться после ответа на звонок.  Разрешены цифры, запятые, знаки "*" и "#".  Запятые могут использоваться для секундной паузы при переходе по системным меню телефона.  Это поле является необязательным.

 

Код страны для резервного номера телефона

Выберите код соответствующей страны для резервного номера телефона.

 

Резервный номер телефона

Введите номер телефона, на который Multi-Factor Authentication сделает вызов в случае, если по основному номеру телефона не будет получен ответ.

 

Добавочный номер резервного телефона

Введите добавочный номер для резервного телефона.

 

ПИН-код

Введите ПИН-код пользователя.  Также можно создать новый ПИН-код, нажав кнопку Generate (Создать).  Этот ПИН-код должен соответствовать правилам относительно минимальной длины ПИН-кода и удовлетворять требованиям по сложности к ПИН-кодам.

 

Кнопка "Создать"

Кнопка "Создать" позволяет создать случайный ПИН-код.  Длина созданного ПИН-кода будет больше 4 символов или будет превышать минимальную длину ПИН-кода, разрешенную для пользователя.

 

Подтвердите ПИН-код

Введите ПИН-код, введенный в поле ПИН-код. ПИН-код должен точно соответствовать введенному ранее значению.

 

Включено

Указывает, включен ли пользователь.  По умолчанию флажок установлен.

 

Примечание. Если запись пользователя не включена, он не будет получать вызовы Multi-Factor Authentication при входе; кроме того, ему может быть разрешен или запрещен доступ к приложению, что зависит от поведения при работе с отключенными пользователями, настроенного по умолчанию для компании.

 

Необходимо изменить ПИН-код

Указывает на необходимость изменения ПИН-кода пользователем в ходе следующего телефонного вызова Multi-Factor Authentication.

 

Примечание. Флажок "Необходимо изменить ПИН-код" будет отображаться только в том случае, если настройки компании разрешают пользователю изменять свой ПИН-код.

 

Отправить сообщение

Указывает, следует ли отправлять сообщение электронной почты добавленному пользователю.

 

Примечание. Флажок "Отправить сообщение" будет отображаться только в том случае, если для вашей компании настроен адрес электронной почты.
