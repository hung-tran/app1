Активация мобильного приложения

Страница "Активация мобильного приложения" позволяет создать код активации мобильного приложения для выбранного пользователя.

 

Чтобы создать код активации мобильного приложения для выбранного пользователя, выполните следующие действия.

  1. Нажмите кнопку Создать код активации.

  2. Предоставьте код активации и URL-адрес пользователю, чтобы он мог ввести эти сведения в мобильное приложение Multi-Factor Authentication на своем устройстве для завершения активации.
