Изменение телефона

Страница "Изменить телефон" позволяет изменить номер телефона, настроенный для вашей учетной записи.  Это изменение носит постоянный характер. Последующие телефонные вызовы и SMS Multi-Factor Authentication будут направляться на указанный новый номер телефона.

 

Чтобы изменить номер телефона, выполните следующие действия.

  1. Выберите код соответствующей страны.

  2. Введите новый номер телефона.

  3. Нажмите кнопку Сохранить.

 

Компонент

Описание

Код страны

В раскрывающемся списке выберите код соответствующей страны. По умолчанию используется текущий код страны.

 

Примечание. Некоторые международные вызовы тарифицируются.  Если системный администратор не реализовал поддержку этого варианта, то телефонные вызовы не будут совершаться на некоторые коды стран.

 

Телефон

Введите номер телефона, на который сервер Multi-Factor Authentication сделает вызов или направит текстовое сообщение в ходе проверки подлинности. Этот номер может принадлежать мобильному телефону, настольному телефону или иному переговорному устройству.  Это устройство должно представлять собой телефон с поддержкой приема и отправки SMS, которые необходимы для метода на основе SMS.  Формат телефонного номера должен совпадать с форматом, выбранным в поле Country Code (Код страны).

 

Пример. Для телефонных номеров в США и Канаде необходимо ввести 10-значный номер телефона (например, 555-123-4567).

 

Добавочный номер

Введите добавочный номер, который будет набираться после ответа на звонок.  Разрешены цифры, запятые, знаки "*" и "#".  Запятые могут использоваться для секундной паузы при переходе по системным меню телефона.  Это поле является необязательным.

 

Код страны для резервного номера телефона

Выберите код соответствующей страны для резервного номера телефона.

 

Резервный номер телефона

Введите номер телефона, на который Multi-Factor Authentication сделает вызов в случае, если по основному номеру телефона не будет получен ответ.

 

Добавочный номер резервного телефона

Введите добавочный номер для резервного телефона.
