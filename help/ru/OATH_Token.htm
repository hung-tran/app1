Токен OATH

Страница "Токен OATH" позволяет привязать выбранного пользователя к стороннему токену OATH.  Прежде чем привязывать пользователя, этот токен необходимо создать на сервере Multi-Factor Authentication.  Этот токен должен поддерживать OATH-коды с учетом времени.  Это альтернатива работе с OATH-кодами с помощью мобильного приложения Multi-Factor Authentication.

 

Чтобы указать маркер OATH, выполните следующие действия.

  1. Введите соответствующие значения.

  2. Нажмите кнопку Сохранить.
