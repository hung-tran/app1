Find User

The Find User page allows an administrator to search for an existing user account.

 

To find a user:

  1. Specify search criteria.

  2. Click the Find button.

 

If a single user matches the criteria entered, the User Information page will be displayed for that user.  If more than one user matches, a list of users will be displayed as links.  Click the appropriate user link to be taken to the User Information page.
