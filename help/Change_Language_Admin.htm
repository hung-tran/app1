Change Language

The Change Language page allows you to change the language for phone call, text message, and mobile app for the selected user's account.  The available languages depend on your company's configuration.

 

To change the selected user's Language:

  1. Select the appropriate language for phone call, text message, and mobile app.

  2. Click the Save button.

 

Feature

Description

Phone Call Language

Select the appropriate language from the drop-down list. The prompts in the Multi-Factor Authentication call will be spoken in this language.  The default is the user's current phone call language.

 

Text Message Language

Select the appropriate language from the drop-down list.  The content of the text message will be written in this language.  The default is the user's current text message language.

 

Mobile App Language

Select the appropriate language from the drop-down list.  The text displayed in the Multi-Factor Authentication mobile app during an authentication will be written in this language.  The default is the user's current mobile app language.
