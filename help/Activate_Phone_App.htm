Activate Mobile App

The Activate Mobile App page allows you to generate a mobile app activation code to activate the Multi-Factor Authentication mobile app on your device.  If you have already activated devices, they will be listed and can be deactivated as needed.

 

To activate the Multi-Factor Authentication mobile app on your device:

  1. Click the Generate Activation Code button.

  2. Enter the activation code and URL in the Multi-Factor Authentication mobile app on your device to complete the activation.
    Note:  

    • If a QR code is displayed, you may use a QR code scanner app to automatically launch the Multi-Factor Authentication mobile app with the activation code and URL filled in.

    • If you are accessing the Multi-Factor Authentication User Portal in a browser on your device, an Activate link will be displayed instead of the activation code and URL.  Clicking the Activate link will automatically launch the Multi-Factor Authentication mobile app with the activation code and URL filled in.

 

To deactivate a device:

  1. Click the Deactivate link next to the device.

  2. Confirm the deactivation by clicking OK.
