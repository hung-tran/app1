Ta bort användare

På sidan för att ta bort användare kan du permanent ta bort en vald användare. Den borttagna användaren beviljas eller nekas åtkomst beroende på programmets konfiguration på Multi-Factor Authentication-servern.  If the application is set to Require User Match, then the removed user will be denied access since they are no longer found in the Multi-Factor Authentication Server's user list.  If the application is not set to Require User Match, then the user will succeed authentication without Multi-Factor Authentication.

 

Klicka på knappen Bekräfta om du vill ta bort den valda användaren permanent.
