Byta språk

På sidan för att byta språk kan du ändra språk för telefonsamtal, SMS och mobilappar för det valda användarkontot.  The available languages depend on your company's configuration.

 

Så här ändrar du språk för den valda användaren:

  1. Välj det aktuella språket för telefonsamtal, SMS och mobilappar.

  2. Klicka på knappen Spara.

 

Funktion

Beskrivning

Telefonsamtalsspråk

Välj det aktuella språket i listrutan. Meddelanden i Multi-Factor Authentication-samtalet uttalas på det här språket.  The default is the user's current phone call language.

 

SMS-språk

Välj det aktuella språket i listrutan.  The content of the text message will be written in this language.  The default is the user's current text message language.

 

Mobilappspråk

Välj det aktuella språket i listrutan.  The text displayed in the Multi-Factor Authentication mobile app during an authentication will be written in this language.  The default is the user's current mobile app language.
