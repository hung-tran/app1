Ändra telefon

På sidan Byt telefonnummer kan du byta telefonnumret som har konfigurerats för ditt konto.  The change is permanent and the newly specified phone number will be called or texted for Multi-Factor Authentication going forward.

 

Så här byter du telefonnummer:

  1. Välj den aktuella landskoden.

  2. Ange ett nytt telefonnummer.

  3. Klicka på knappen Spara.

 

Funktion

Beskrivning

Landskod

Välj den aktuella landskoden i listrutan. Som standard används det aktuella landsnumret.

 

Obs! Vissa internationella samtal kostar pengar.  Om inte din administratör har implementerat det här alternativet går det inte att ringa till vissa landskoder.

 

Telefonnummer

Ange telefonnumret som Multi-Factor Authentication ska ringa eller SMS:a vid användarautentisering. Det kan vara numret till en mobiltelefon, en fast telefon eller något annat nummer för samtalsmetoden.  It must be a phone with text message capabilities for the text message method.  The phone number format must match the format selected in the Country Code field.

 

Exempel: För telefonnummer i USA och Kanada måste du ange ett telefonnummer bestående av 10 siffror, t.ex. 555-123-4567.

 

Anknytning

Ange numret till den anknytning som ska ringas när samtalet har besvarats.  Digits, commas, *, and # are allowed.  Commas can be used for a one second pause to navigate phone system menus.  This field is optional.

 

Sekundär landskod

Ange den aktuella landskoden för det sekundära telefonnumret.

 

Sekundär telefon

Ange telefonnumret som Multi-Factor Authentication ska ringa om ingen svarar på det primära telefonnumret.

 

Sekundär anknytning

Ange anknytningen för det sekundära telefonnumret.
