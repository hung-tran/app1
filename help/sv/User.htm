Användare

På användarsidan visas information om användaren och alternativ för att ändra användarkontot.  Sidan visas när bara en användare hittas på sidan där du söker efter användare, eller när du klickar på en användarlänk på sidan där du söker efter användare, efter att du hittat flera användare.

 

Användarens användarnamn och telefonnummer visas i sidrubriken. Det går också att visa användarens säkerhetsfrågor så att du kan verifiera användarens identitet innan du gör några ändringar.  Ange bara användarens svar och klicka på knappen för att verifiera.  Om inte användaren har definierat några säkerhetsfrågor visas ett meddelande om detta.

 

Länkar visas för de ändringar du kan göra i användarens konto.  Eventuellt visas inte alla länkar beroende på vilka behörigheter ditt administratörskonto har beviljats och den övergripande Multi-Factor Authentication-konfigurationen.

 

Funktion

Beskrivning

Engångsförbikoppling

Klicka på länken för engångsförbikoppling när du vill initiera en engångsförbikoppling för den valda användaren.  Användaren kan då logga in en gång utan att ta emot ett Multi-Factor Authentication-samtal.

 

Ändra metod

Klicka på länken för att byta metod när du vill byta metod för den valda användaren.

 

Ändra läge

Klicka på länken för att ändra läge när du vill ändra lägen för den valda användaren.

 

Ändra telefon

Klicka på länken för att byta telefonnummer när du vill byta telefonnummer för den valda användaren.

 

Byta pinkod

Klicka på länken för att byta pinkod när du vill byta pinkod för den valda användaren.

 

Obs! Den här länken är bara tillgänglig när användaren är i telefonsamtalspinkodsläge, SMS-OTP + pinkodsläge eller i mobilappläge.

 

Byta språk

Klicka på länken för att byta språk när du vill byta språk för den valda användaren.

 

Återställ röstprofil

Klicka på länken för att återställa röstprofil när du vill återställa röstprofilen för den valda användaren.

 

Användarinformation

Klicka på länken för användarinformation när du vill ändra användarens namn eller e-postadress.

 

OATH-token

Klicka på länken för OATH-token när du vill byta användarens OATH-token från tredje part.

 

Aktivera/inaktivera

Klicka på länken för att aktivera eller inaktivera när du vill aktivera eller inaktivera den valda användarens Multi-Factor Authentication-konto.

 

Obs! Eventuellt kan en inaktiverad användare logga in utan Multi-Factor Authentication.

 

Blockera/avblockera användare

Klicka på länken för att blockera/avblockera användare när du vill blockera eller avblockera den valda användaren.

 

Ta bort användare

Klicka på länken för att ta bort användare när du vill ta bort den valda användaren permanent.

 

Obs! Eventuellt kan en borttagen användare logga in utan Multi-Factor Authentication.

 

Testa autentisering

Klicka på länken för att testa autentisering när du behöver testa en Multi-Factor Authentication-samtalsautentisering eller Multi-Factor Authentication-SMS-autentisering för den valda användaren.

 

Aktivera mobilapp

Klicka på länken för att aktivera mobilapp när du vill generera en mobilappaktiveringskod för den valda användaren.

 

Inaktivera mobilapp

Klicka på länken för att inaktivera mobilapp när du vill inaktivera en befintlig enhet för den valda användaren.
