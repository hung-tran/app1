Paramétrage de l’utilisateur

Si le paramétrage de votre compte est incomplet, vous devez spécifier le Numéro de téléphone (si applicable) et/ou le PIN (si applicable) à utiliser avec votre compte, ou vous serez invité à générer un code d’activation pour activer l’application mobile Multi-Factor Authentication.  Si votre société l’autorise, vous pouvez choisir entre les méthodes d’appel téléphonique, de message texte, d’application mobile et de jeton OATH.  Un appel Multi-Factor Authentication sera effectué ou un message texte envoyé au numéro de téléphone que vous avez spécifié pour confirmer sa validité, une notification sera envoyée à l’application mobile, ou vous serez invité à entrer un code d’authentification.  Pour la méthode d’appel téléphonique, vous devrez appuyer sur la touche #, entrer votre PIN ou inscrire votre empreinte vocale en fonction de votre mode d’appel téléphonique.  Pour la méthode de message texte, vous recevrez un code secret à usage unique (OTP) et répondrez soit avec l’OTP, soit avec l’OTP + votre PIN.  Pour la méthode d’application mobile, une notification sera envoyée à l’application mobile et vous invitera à vous authentifier, avec entrée ou non de votre PIN suivant votre mode d’application mobile.  Pour la méthode de jeton OATH, vous serez invité par le portail de l’utilisateur à entrer un code d’authentification.  Si la méthode réussit, vous serez invité à créer vos questions de sécurité.

 

Pour effectuer le paramétrage de l’utilisateur pour la méthode d’appel téléphonique :

  1. Entrez votre indicatif de pays et numéro de téléphone.  Vous pouvez également entrer un numéro de téléphone de secours (le cas échéant).

  2. Entrez votre PIN (si applicable).

  3. Cliquez sur le bouton M’appeler maintenant pour m’authentifier.

 

Pour effectuer le paramétrage de l’utilisateur pour la méthode de message texte :

  1. Entrez votre indicatif de pays et numéro de téléphone.

  2. Entrez votre PIN (si applicable).

  3. Cliquez sur le bouton M’envoyer un message texte maintenant pour m’authentifier.

 

Pour effectuer le paramétrage de l’utilisateur pour la méthode d’application mobile :

  1. Installez l’application mobile sur votre téléphone.

  2. Cliquez sur le bouton Générer le code d’activation.

  3. Entrez le code d’activation et l’URL de l’application mobile pour effectuer l’activation.

  4. Entrez votre PIN (si applicable).

  5. Cliquez sur le bouton M’authentifier maintenant.

 

Pour effectuer le paramétrage de l’utilisateur pour la méthode de jeton OATH :

  1. Installez l’application mobile sur votre téléphone.

  2. Cliquez sur le bouton Générer le code d’activation.

  3. Entrez le code d’activation et l’URL de l’application mobile pour effectuer l’activation.

  4. Cliquez sur le bouton M’authentifier maintenant.

OU

  1. Entrez le numéro de série imprimé à l’arrière du jeton OATH.

  2. Entrez le code actuellement affiché sur le jeton OATH.

  3. Cliquez sur le bouton Vérifier le jeton OATH.

 

 

Fonctionnalité

Description

Méthode

Sélectionnez la méthode Appel téléphonique, Message texte, Application Mobile ou Jeton OATH.  Pour la méthode d’appel téléphonique, vous recevrez un appel Multi-Factor Authentication vous invitant à appuyer sur la touche #, entrer votre PIN ou inscrire votre empreinte vocale.  Pour la méthode de message texte, vous recevrez un code secret à usage unique (OTP) dans un message texte et répondrez soit avec l’OTP, soit avec l’OTP + votre PIN.  Pour la méthode d’application mobile, une notification sera envoyée à l’application mobile et vous invitera à vous authentifier, avec entrée possible de votre PIN.  Pour la méthode de jeton OATH, vous serez invité par le portail de l’utilisateur à entrer un code d’authentification.

 

Remarque :    la méthode que vous sélectionnez est la méthode que vous utiliserez lorsque vous accéderez à votre application Multi-Factor Authentication protégée.  Cette méthode peut toujours être modifiée ultérieurement dans la page Modifier la méthode.

 

Indicatif du pays

Sélectionnez l’indicatif du pays approprié dans la liste déroulante. Si un numéro de téléphone a déjà été configuré pour votre compte, la valeur par défaut est l’indicatif de pays de ce numéro de téléphone.  Sinon, la valeur par défaut est définie sur États-Unis et Canada + 1.

 

Remarque : certains appels internationaux peuvent entraîner des frais.  Si votre administrateur n’a pas implémenté cette option, l’appel ne sera pas passé pour certains indicatifs de pays.

 

Téléphone

Entrez le numéro de téléphone que Multi-Factor Authentication appellera ou auquel il enverra un message texte lors de l’authentification. Il peut s’agir d’un numéro de téléphone mobile, de téléphone de bureau ou de tout autre numéro du mode d’appel téléphonique.  En mode de message texte, le numéro doit être celui d’un téléphone équipé de la fonctionnalité de message texte.  Le format du numéro de téléphone doit correspondre au format sélectionné dans le champ Indicatif du pays.

 

Exemple : pour les numéros de téléphone aux États-Unis et au Canada, vous devez entrer un numéro à 10 chiffres (par ex., 555-123-4567).

 

Extension

Entrez l'extension à composer une fois l'appel reçu.  Sont autorisés les chiffres, les virgules, * et #.  Les virgules peuvent être utilisées pour marquer une pause d'une seconde dans le but de parcourir les menus du système téléphonique.  Ce champ est facultatif.

 

Indicatif de pays de sauvegarde

Spécifiez l’indicatif du pays approprié pour le numéro de téléphone de secours.

 

Téléphone de secours

Entrez le numéro de téléphone appelé par Multi-Factor Authentication si l’appel vers le téléphone principal reste sans réponse.

 

Extension de sauvegarde

Entrez l’extension du numéro de téléphone de secours.

 

PIN

Entrez un numéro de PIN.  Ce PIN sera entré au cours de chaque appel d’authentification, entré avec le code secret à usage unique (OTP) en réponse à chaque message texte, ou entré dans l’application mobile afin d’effectuer l’authentification.  Votre administrateur peut avoir configuré les règles de PIN, notamment la longueur minimale à respecter.

 

Numéro de série

Entrez le numéro de série imprimé à l’arrière du jeton OATH.

 

Code

Entrez le code actuellement affiché sur le jeton OATH.

 

 

Pour revenir à la page Se connecter, cliquez sur le bouton Annuler.
