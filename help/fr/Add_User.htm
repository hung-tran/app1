Ajouter un utilisateur

La page Ajouter un utilisateur permet de créer un nouveau compte Multi-Factor Authentication.

 

Remarque :  de nombreux paramètres d’utilisateur, tels que la méthode (Appel téléphonique, Message texte ou Application mobile) ont la valeur par défaut en fonction des paramètres par défaut de l’utilisateur spécifiés dans la section Paramètres de la société du serveur Multi-Factor Authentication.

 

Pour ajouter un nouvel utilisateur :

  1. Entrez les informations de l’utilisateur.

  2. Cliquez sur le bouton Ajouter.

 

Fonctionnalité

Description

Nom d’utilisateur

Le nom d’utilisateur correspond généralement à l’e-mail de l’utilisateur dans le format d’e-mail correct.

 

Exemple : chuck@abc.com

 

Prénom

Entrez le prénom de l’utilisateur.

 

Nom

Entrez le nom de famille de l’utilisateur.

 

Adresse de messagerie

Entrez l’adresse de messagerie de l’utilisateur.  Les e-mails seront envoyés à cette adresse de messagerie.  Si elle n’est pas spécifiée, les e-mails seront envoyés au nom d’utilisateur s’il est dans le format d’e-mail correct.

 

Groupe d’utilisateurs

Entrez le groupe d’utilisateurs de l’utilisateur.

 

Indicatif du pays

Sélectionnez l’indicatif du pays approprié dans la liste déroulante. La valeur par défaut est définie sur États-Unis et Canada + 1.

 

Remarque : certains appels internationaux peuvent entraîner des frais.  Si votre société n’a pas implémenté cette option, l’appel ne sera pas passé pour certains indicatifs de pays.

 

Téléphone

Entrez le numéro de téléphone que Multi-Factor Authentication appellera ou auquel il enverra un message texte lors de l’authentification. Il peut s’agir d’un numéro de téléphone mobile, de téléphone de bureau ou de tout autre numéro de la méthode d’appel téléphonique.  Pour la méthode de message texte, le téléphone doit être équipé de la fonctionnalité de message texte.  Le format du numéro de téléphone doit correspondre au format sélectionné dans le champ Indicatif du pays.

 

Exemple : pour les numéros de téléphone aux États-Unis et au Canada, vous devez entrer un numéro à 10 chiffres (par ex., 555-123-4567).

 

Extension

Entrez l'extension à composer une fois l'appel reçu.  Sont autorisés les chiffres, les virgules, * et #.  Les virgules peuvent être utilisées pour marquer une pause d'une seconde dans le but de parcourir les menus du système téléphonique.  Ce champ est facultatif.

 

Indicatif de pays de sauvegarde

Spécifiez l’indicatif du pays approprié pour le numéro de téléphone de secours.

 

Téléphone de secours

Entrez le numéro de téléphone appelé par Multi-Factor Authentication si l’appel vers le téléphone principal reste sans réponse.

 

Extension de sauvegarde

Entrez l’extension du numéro de téléphone de secours.

 

PIN

Tapez le PIN de l’utilisateur.  Ou, générez le PIN en cliquant sur le bouton Générer.  Le PIN doit respecter les règles de longueur minimale du PIN et de PIN faibles.

 

Bouton Générer

Le bouton Générer fournit un PIN généré de manière aléatoire.  La longueur du PIN généré sera égale à la plus grande longueur de 4 chiffres ou à la longueur minimale du PIN de l’utilisateur.

 

Confirmer le PIN

Retapez le PIN que vous avez saisi dans le champ PIN. Le code PIN confirmé doit être identique.

 

Activé

Spécifie si l’utilisateur est activé.  Par défaut, la case à cocher est sélectionnée.

 

Remarque : si un enregistrement d’utilisateur n’est pas activé, l’utilisateur ne reçoit pas d’appel Multi-Factor Authentication lorsqu’il se connecte et peut se voir accorder ou refuser l’accès à l’application en fonction des paramètres par défaut de la société pour le comportement désactivé.

 

Modification de PIN requise

Spécifie qu’un utilisateur doit modifier son PIN lors de son prochain appel Multi-Factor Authentication.

 

Remarque : la case à cocher Modification de PIN requise s’affiche uniquement si la configuration de la société permet aux utilisateurs de modifier leur PIN.

 

Envoyer un courrier électronique

Spécifie si un e-mail doit être envoyé à l’utilisateur ajouté.

 

Remarque : la case à cocher Envoyer un courrier électronique s’affiche uniquement si l’e-mail a été configuré pour votre société.
