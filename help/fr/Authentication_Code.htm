Code d’authentification

Pour le mode Jeton OATH, le portail de l’utilisateur invitera à entrer le code d’authentification affiché dans l’application mobile Multi-Factor Authentication ou le jeton.

 

Pour terminer votre connexion :

  1. Entrez le code d’authentification affiché dans l’application mobile Multi-Factor Authentication ou le jeton.

  2. Cliquez sur le bouton Authentifier.

 

Pour revenir à la page Se connecter, cliquez sur le bouton Annuler.
