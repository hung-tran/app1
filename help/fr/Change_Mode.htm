Modifier la méthode

La page Modifier la méthode permet de modifier votre méthode.  Pour la méthode d’appel téléphonique, vous recevrez un appel téléphonique lorsque vous vous authentifierez.  Pour la méthode message texte, vous recevrez un message texte lorsque vous vous authentifierez.  Pour la méthode d’application mobile, vous recevrez une notification dans l’application mobile Multi-Factor Authentication installée sur votre périphérique.  Pour la méthode jeton OATH, vous serez invité à entrer un code d’authentification.

 

Pour modifier votre méthode :

  1. Sélectionnez la méthode appropriée.

  2. Cliquez sur le bouton Enregistrer.

Méthode

Sélectionnez la méthode appropriée dans la liste déroulante. La valeur par défaut est votre méthode actuelle.
