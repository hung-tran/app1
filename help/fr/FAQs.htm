Foire aux questions

Fonctionnement de Multi-Factor Authentication™

L’authentification multifacteur fonctionne en passant un appel de confirmation, en envoyant un message texte, en notifiant l’application mobile Multi-Factor Authentication lors de la connexion, ou en invitant à entrer un code d’authentification à partir de l’application mobile Multi-Factor Authentication ou un jeton.

 

Étape 1 :

Entrez vos nom d’utilisateur et mot de passe habituels.

Étape 2 :

Vous recevez instantanément un appel téléphonique, un message texte ou une notification d’application mobile, ou vous serez invité à entrer un code d’authentification. Pour la méthode d’appel téléphonique, vous pouvez soit répondre et appuyer sur la touche #, soit entrer votre PIN, soit énoncer votre phrase d’empreinte vocale.  Pour la méthode de message texte, vous pouvez soit répondre avec le code secret à usage unique (OTP) dans le message, soit répondre avec l’OTP + votre PIN.  Pour la méthode d’application mobile, vous pouvez soit appuyer sur Authentifier, soit entrer votre PIN et appuyer sur Authentifier dans l’application mobile Multi-Factor Authentication installée sur votre périphérique.  Pour la méthode de jeton OATH, entrez le code d’authentification affiché dans l’application mobile Multi-Factor Authentication ou le jeton.

C’est fini !

 

Ce processus simple fournit deux facteurs d’authentification distincts via deux canaux séparés (votre ordinateur et votre service téléphonique). Il fonctionne en mode d’appel téléphonique avec n’importe quel téléphone standard ou mobile, en mode de message texte avec n’importe quel téléphone équipé de la fonctionnalité de message texte, ou avec un périphérique qui prend en charge l’application mobile Multi-Factor Authentication.

 

Que se passe-t-il si je perds mon téléphone ?

Sélectionnez l’option Modifier le numéro de téléphone pour entrer un nouveau numéro de téléphone. Vous pouvez également obtenir un autre numéro en appelant le support technique une fois que votre identité est fortement établie.

 

Que se passe-t-il si je perds la couverture de mon téléphone cellulaire dans une certaine zone ?

Vous pouvez modifier votre compte afin qu’il pointe vers un autre numéro de téléphone, en sélectionnant l’option Modifier le numéro de téléphone. Certaines sociétés offrent également une option de Contournement à usage unique qui permet de se connecter une seule fois sans appel téléphonique ou message texte. Si elle est disponible, sélectionnez l’option Contournement à usage unique pour accéder à cette fonctionnalité. Ou, appelez le support technique à partir de n’importe quel téléphone pour obtenir de l’aide.  La méthode de jeton OATH ne nécessite pas de couverture pour le téléphone cellulaire.

 

Que se passe-t-il si je reçois un appel téléphonique, un message texte ou une notification d’application de Multi-Factor Authentication, alors que je n’ai pas essayé de me connecter ?

Si cela se produit, c’est uniquement parce qu’une personne essaye de se connecter à votre compte, et qu’elle connaît déjà votre mot de passe. N’oubliez pas que, l’exécution d’un appel téléphonique, l’envoi d’un message texte ou l’envoi d’une notification d’application mobile ne sont possibles qu’après vérification de votre nom d’utilisateur et de votre mot de passe. Par conséquent, si cela se produit, cela signifie que Multi-Factor Authentication vient juste de protéger votre compte contre un accès illicite ! Afin de bloquer temporairement toute tentative d’authentification avec votre compte, suivez les invites dans le menu du téléphone pour la méthode d’appel téléphonique, répondez au texte avec le code fraude attribué par votre société pour la méthode de message texte, ou appuyez sur Refuser et signaler la fraude dans l’application mobile.  Si l’option Alerte fraude est sélectionnée pour votre société, en plus de bloquer votre compte, une alerte sera envoyée instantanément à l’équipe de sécurité informatique de votre société. Contactez votre support technique pour obtenir de l’aide pour débloquer votre compte.
