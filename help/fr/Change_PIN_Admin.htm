Modifier le PIN

La page Modifier le PIN permet de modifier le PIN pour le compte de l’utilisateur sélectionné.  La modification est définitive et le PIN qui vient d’être spécifié sera utilisé à l’avenir dans Multi-Factor Authentication.

 

Pour modifier le PIN de l’utilisateur sélectionné :

  1. Entrez un nouveau PIN.

  2. Entrez-le de nouveau pour le confirmer.

  3. Cliquez sur le bouton Enregistrer.

 

Fonctionnalité

Description

Nouveau PIN

Tapez un nouveau PIN.  Ou, générez le PIN en cliquant sur le bouton Générer.  Le PIN doit respecter les règles de longueur minimale du PIN et de PIN faibles.

 

Bouton Générer

Le bouton Générer fournit un PIN généré de manière aléatoire.  La longueur du PIN généré sera égale à la plus grande longueur de 4 chiffres ou à la longueur minimale du PIN de l’utilisateur.

 

Confirmer le PIN

Tapez encore une fois le nouveau code PIN pour confirmer la modification. Le code PIN confirmé doit être identique.

 

Modification de PIN requise

Spécifie qu’un utilisateur doit modifier son PIN lors de sa prochaine authentification multifacteur.

 

Remarque : la case à cocher Modification de PIN requise s’affiche uniquement si l’utilisateur est autorisé à modifier son PIN.

 

Envoyer un courrier électronique

Spécifie si un e-mail doit être envoyé pour informer l’utilisateur de la mise à jour.

 

Remarque : la case à cocher Envoyer un courrier électronique s’affiche uniquement si l’e-mail a été configuré pour votre société.
