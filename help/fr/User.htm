Utilisateur

La page Utilisateur affiche des informations sur l’utilisateur et les options permettant de modifier le compte de l’utilisateur.  La page s’affiche lorsqu’un utilisateur unique est trouvé dans la page Rechercher un utilisateur, ou en cliquant sur un lien d’utilisateur dans la page Rechercher un utilisateur où sont affichés plusieurs utilisateurs trouvés.

 

Le nom d’utilisateur et le mot de passe de l’utilisateur sont affichés dans l’en-tête de la page. Il est possible d’afficher également ses questions de sécurité pour vous permettre de vérifier l’identité de l’utilisateur avant d’apporter des modifications.  Pour ce faire, entrez les réponses de l’utilisateur et cliquez sur le bouton Vérifier.  Si l’utilisateur n’a pas défini ses questions de sécurité, un message s’affichera à cet effet.

 

Des liens sont affichés pour les modifications que vous pouvez apporter au compte d’utilisateur.  Tous les liens pourront ne pas être affichés en fonction des autorisations accordées à votre compte d’administrateur et de la configuration générale de l’authentification multifacteur.

 

Fonctionnalité

Description

Contournement à usage unique

Sélectionnez le lien Contournement à usage unique pour initier un contournement à usage unique pour l’utilisateur sélectionné.  L’utilisateur pourra se connecter une seule fois sans recevoir d’appel Multi-Factor Authentication.

 

Modifier la méthode

Sélectionnez le lien Modifier la méthode pour modifier la méthode de l’utilisateur sélectionné.

 

Modifier le mode

Sélectionnez le lien Modifier le mode pour modifier les modes de l’utilisateur sélectionné.

 

Modifier le téléphone

Sélectionnez le lien Modifier le téléphone pour modifier le numéro de téléphone de l’utilisateur sélectionné.

 

Modifier le PIN

Sélectionnez le lien Modifier le PIN pour modifier le PIN de l’utilisateur sélectionné.

 

Remarque : ce lien est disponible uniquement lorsque l’utilisateur est en mode PIN de l’appel téléphonique, mode code secret à usage unique + PIN du message texte, ou mode PIN de l’application mobile.

 

Modifier la langue

Sélectionnez le lien Modifier la langue pour modifier la langue de l’utilisateur sélectionné.

 

Réinitialiser l’empreinte vocale

Sélectionnez le lien Réinitialiser l’empreinte vocale pour réinitialiser l’empreinte vocale de l’utilisateur sélectionné.

 

Informations sur l’utilisateur

Sélectionnez le lien Informations sur l’utilisateur pour modifier le nom ou l’adresse de messagerie de l’utilisateur.

 

Jeton OATH

Sélectionnez le lien Jeton OATH pour modifier le jeton OATH tiers de l’utilisateur.

 

Activer ou désactiver

Sélectionnez le lien Activer ou désactiver pour activer ou désactiver le compte Multi-Factor Authentication de l’utilisateur sélectionné.

 

Remarque : un utilisateur désactivé peut avoir la possibilité de se connecter sans authentification multifacteur.

 

Bloquer/débloquer l’utilisateur

Sélectionnez le lien Bloquer/débloquer l’utilisateur pour bloquer ou débloquer l’utilisateur sélectionné.

 

Supprimer un utilisateur

Sélectionnez le lien Supprimer un utilisateur pour supprimer définitivement l’utilisateur sélectionné.

 

Remarque : un utilisateur supprimé peut avoir la possibilité de se connecter sans authentification multifacteur.

 

Tester l’authentification

Sélectionnez le lien Tester l’authentification pour tester l’authentification avec un appel Multi-Factor Authentication ou un message texte Multi-Factor Authentication pour l’utilisateur sélectionné.

 

Activer l’application mobile

Sélectionnez le lien Activer l’application mobile pour générer un code d’activation d’application mobile pour l’utilisateur sélectionné.

 

Désactiver l’application mobile

Sélectionnez le lien Désactiver l’application mobile pour désactiver un périphérique existant pour l’utilisateur sélectionné.
