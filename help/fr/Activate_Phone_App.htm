Activer l’application mobile

La page Activer l’application mobile permet de générer un code d’activation pour activer l’application mobile Multi-Factor Authentication sur votre périphérique.  Si vous avez déjà activé des périphériques, ils seront répertoriés et pourront être désactivés si besoin.

 

Pour activer l’application mobile Multi-Factor Authentication sur votre périphérique :

  1. Cliquez sur le bouton Générer le code d’activation.

  2. Entrez le code d’activation et l’URL dans l’application mobile Multi-Factor Authentication sur votre périphérique pour effectuer l’activation.
    Remarque :  

    • Si un code QR s’affiche, vous pouvez utiliser une application de scanneur de code QR pour lancer automatiquement l’application mobile Multi-Factor Authentication avec le code d’activation et l’URL complétés.

    • Si vous accédez au portail de l’utilisateur de Multi-Factor Authentication dans un navigateur sur votre appareil, un lien Activer s’affichera au lieu du code d’activation et de l’URL.  En cliquant sur le lien Activer, vous lancerez automatiquement l’application mobile Multi-Factor Authentication avec le code d’activation et l’URL complétés.

 

Pour désactiver un périphérique :

  1. Cliquez sur le lien Désactiver à côté du périphérique.

  2. Confirmez la désactivation en cliquant sur OK.
