Modifier les questions de sécurité

La page Modifier les questions de sécurité permet de remplacer vos questions et réponses de sécurité actuelles par des nouvelles.

 

Pour modifier vos questions de sécurité :

  1. Sélectionnez les questions dans la liste déroulante, puis tapez les réponses à vos 4 questions de sécurité.

  2. Cliquez sur le bouton Enregistrer.
