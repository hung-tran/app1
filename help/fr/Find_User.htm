Rechercher un utilisateur

La page Rechercher un utilisateur permet à l’administrateur de rechercher un compte d’utilisateur existant.

 

Pour trouver un utilisateur :

  1. Spécifiez les critères de recherche.

  2. Cliquez sur le bouton Rechercher.

 

Si un seul utilisateur correspond aux critères saisis, la page Informations sur l’utilisateur s’affichera pour cet utilisateur.  Si plusieurs utilisateurs correspondent aux critères saisis, une liste d’utilisateurs s’affichera sous forme de liens.  Cliquez sur le lien d’utilisateur approprié pour ouvrir la page Informations sur l’utilisateur.

 

Fonctionnalité

Description

Nom d’utilisateur

Entrez la chaîne de recherche du nom d’utilisateur correspondant.

 

Remarque : les caractères génériques sont autorisés dans la chaîne de recherche en spécifiant un astérisque (*) pour remplacer plusieurs caractères.

 

Prénom

Entrez la chaîne de recherche du prénom correspondant.

 

Nom

Entrez la chaîne de recherche du nom de famille correspondant.

 

Adresse de messagerie

Entrez la chaîne de recherche de l’adresse de messagerie correspondante.

 

Groupe d’utilisateurs

Entrez la chaîne de recherche du groupe d’utilisateurs correspondant.

 

Téléphone

Entrez la chaîne de recherche du téléphone correspondant.

 

Balise

Sélectionnez Tout  pour rechercher tous les utilisateurs ou sélectionnez une balise spécifique pour rechercher uniquement les utilisateurs avec cette balise.
