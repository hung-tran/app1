Modifier le mode

La page Modifier le mode permet de modifier les modes pour le compte de l’utilisateur sélectionné.  Les méthodes d’appel téléphonique, de message texte et d’application mobile ont différents modes pris en charge.

 

Remarque :  selon la configuration du produit, les listes déroulantes Mode pourront ne pas s’afficher pour certaines méthodes.

 

Pour modifier les modes de l’utilisateur sélectionné :

  1. Sélectionnez les modes appropriés.

  2. Cliquez sur le bouton Enregistrer.

 

Fonctionnalité

Description

Mode d’appel téléphonique

Sélectionnez le mode approprié dans la liste déroulante. La valeur par défaut est le mode d’appel téléphonique actuel de l’utilisateur.

 

Mode de message texte

Sélectionnez la direction et le mode appropriés dans les listes déroulantes.  La valeur par défaut est la direction et le mode de message texte actuels de l’utilisateur.

 

Mode d’application mobile

Sélectionnez le mode approprié dans la liste déroulante.  La valeur par défaut est le mode d’application mobile actuel de l’utilisateur.

 

Envoyer un courrier électronique

Spécifie si un e-mail doit être envoyé pour informer l’utilisateur de la mise à jour.

 

Remarque : la case à cocher Envoyer un courrier électronique s’affiche uniquement si l’e-mail a été configuré pour votre société.
