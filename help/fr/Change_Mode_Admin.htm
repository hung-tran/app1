Modifier la méthode

La page Modifier la méthode permet de modifier la méthode pour le compte de l’utilisateur sélectionné.  Pour la méthode d’appel téléphonique, l’utilisateur recevra un appel téléphonique lorsqu’il s’authentifiera.  Pour la méthode de message texte, l’utilisateur recevra un message texte lorsqu’il s’authentifiera.  Pour la méthode d’application mobile, l’utilisateur recevra une notification dans l’application mobile Multi-Factor Authentication installée sur son périphérique.  Pour la méthode de jeton OATH, l’utilisateur sera invité à entrer un code d’authentification.

 

Pour modifier la méthode de l’utilisateur sélectionné :

  1. Sélectionnez la méthode appropriée.

  2. Cliquez sur le bouton Enregistrer.

 

Fonctionnalité

Description

Méthode

Sélectionnez la méthode appropriée dans la liste déroulante. La valeur par défaut est la méthode actuelle de l’utilisateur.

 

Envoyer un courrier électronique

Spécifie si un e-mail doit être envoyé pour informer l’utilisateur de la mise à jour.

 

Remarque : la case à cocher Envoyer un courrier électronique s’affiche uniquement si l’e-mail a été configuré pour votre société.
