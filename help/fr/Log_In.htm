Se connecter

Pour vous connecter au portail de l’utilisateur, entrez votre nom d’utilisateur et votre mot de passe.  Vous recevrez un appel Multi-Factor Authentication ou un message texte à votre numéro de téléphone, une notification sur l’application mobile Multi-Factor Authentication, ou vous serez invité à entrer un code d’authentification.  Si vous terminez correctement l’authentification, vous serez redirigé vers la page principale.

 

Si la configuration de votre compte est incomplète, vous serez redirigé vers la page Paramétrage de l’utilisateur.

 

Si vous vous connectez au portail de l’utilisateur pour la première fois, vous pouvez être invité à créer vos questions de sécurité.

 

Si l’authentification Multi-Factor est refusée, vous pouvez être invité à répondre à vos questions de sécurité pour terminer la connexion.

 

Pour vous connecter :

  1. Entrez vos nom d’utilisateur et mot de passe.

  2. Cliquez sur le bouton Se connecter.

 

Fonctionnalité

Description

Nom d’utilisateur

Nom d’utilisateur que vous utilisez normalement pour vous connecter à votre application Multi-Factor Authentication protégée.  Il s’agit souvent de votre nom d’utilisateur Windows.

 

Mot de passe

Mot de passe que vous utilisez normalement pour vous connecter à votre application Multi-Factor Authentication protégée.  Il s’agit souvent de votre mot de passe Windows.
