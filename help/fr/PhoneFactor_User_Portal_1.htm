Portail de l’utilisateur Multi-Factor Authentication

Le portail de l’utilisateur Multi-Factor Authentication permet de contourner l’authentification multifacteur pour une connexion unique, de gérer les paramètres de votre compte Multi-Factor Authentication, d’inclure vos méthode, numéro de téléphone, PIN, langue et questions de sécurité, et d’activer l’application mobile Multi-Factor Authentication. Votre administrateur peut désactiver certaines de ces fonctionnalités.  Pour plus d’informations sur l’utilisation de Multi-Factor Authentication, voir Foire aux questions.

 

Une table des matières, un index et des fonctionnalités de recherche sont disponibles pour vous aider à trouver des informations dans le système d’aide.
