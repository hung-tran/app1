Test Authentication

The Test Authentication page allows you to test a Multi-Factor Authentication call, text message authentication, or Multi-Factor Authentication mobile app authentication for the selected user.  Whether a call is made, a text message is sent, or the Multi-Factor Authentication mobile app is notified is based on the user's mode.

 

Click the Test button to initiate the call, send the text message, or notify the Multi-Factor Authentication mobile app.

 

When the user is in One-Way SMS Text mode, you will be prompted to authenticate the user's One-Time Passcode (OTP) or OTP + PIN.

 

To complete a One-Way Text Message test authentication:

  1. Enter the one-time passcode the user received in the text message.
  2. Enter the user's PIN in the same field if instructed to do so.
  3. Click the Authenticate button.
