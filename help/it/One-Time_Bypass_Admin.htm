Bypass monouso

La pagina Bypass monouso consente di eseguire il bypass di Multi-Factor Authentication per l'accesso successivo dell'utente selezionato. Quando si attiva il bypass, l'utente potrà eseguire l'autenticazione senza Multi-Factor Authentication una sola volta.  Il bypass scade se non viene usato.

 

Per attivare un bypass monouso per l'utente selezionato:

  1. Specificare i secondi di bypass.

  2. Scegliere il pulsante Conferma per eseguire il bypass di Multi-Factor Authentication per l'accesso successivo dell'utente selezionato.

 

Funzionalità

Descrizione

Secondi bypass

Immettere il numero di secondi durante i quali è possibile usare il bypass prima che scada.
