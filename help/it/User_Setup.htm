Configurazione utenti

Se la configurazione dell'account non è completa, è necessario specificare il numero di telefono (se applicabile) e/o il PIN (se applicabile) da usare con l'account oppure è possibile che venga richiesto di generare un codice di attivazione per attivare l'app mobile Multi-Factor Authentication.  Se è consentito dalla società, è possibile scegliere tra i metodi telefonata, SMS, app mobile e token OATH.  A seconda del metodo scelto, sarà effettuata una chiamata di Multi-Factor Authentication o sarà inviato un SMS al numero di telefono specificato per confermarne la validità, sarà inviata una notifica all'app mobile o sarà richiesta l'immissione di un codice di autenticazione.  A seconda della modalità scelta per il metodo telefonata, sarà richiesto di premere #, di immettere il PIN o di registrare l'impronta vocale.  Per il metodo SMS, l'utente riceverà un passcode monouso (OTP) e dovrà rispondere con il passcode monouso o il passcode monouso più il PIN.  Per il metodo app mobile, l'utente riceverà una notifica sull'app mobile per eseguire l'autenticazione e a seconda della modalità scelta è possibile che sia necessario immettere il PIN.  Per il metodo token OATH, sarà richiesta l'immissione di un codice di autenticazione dal portale per gli utenti.  Se l'operazione ha esito positivo, è possibile che venga richiesto di creare le domande di sicurezza.

 

Per completare la configurazione utente per il metodo telefonata:

  1. Immettere il codice paese e il numero di telefono.  È anche possibile immettere un numero di telefono di backup (se applicabile).

  2. Immettere il PIN (se applicabile).

  3. Scegliere il pulsante Chiama per autenticare utente corrente.

 

Per completare la configurazione utente per il metodo SMS:

  1. Immettere il codice paese e il numero di telefono.

  2. Immettere il PIN (se applicabile).

  3. Scegliere il pulsante Invia SMS per autenticare utente corrente.

 

Per completare la configurazione utente per il metodo app mobile:

  1. Installare l'app mobile sul telefono.

  2. Scegliere il pulsante Genera codice di attivazione.

  3. Immettere il codice di attivazione e l'URL nell'app mobile per completare l'attivazione.

  4. Immettere il PIN (se applicabile).

  5. Scegliere il pulsante Esegui autenticazione utente corrente.

 

Per completare la configurazione utente per il metodo token OATH:

  1. Installare l'app mobile sul telefono.

  2. Scegliere il pulsante Genera codice di attivazione.

  3. Immettere il codice di attivazione e l'URL nell'app mobile per completare l'attivazione.

  4. Scegliere il pulsante Esegui autenticazione utente corrente.

In alternativa

  1. Immettere il numero di serie stampato sul retro del token OATH.

  2. Immettere il codice attualmente visualizzato sul token OATH.

  3. Scegliere il pulsante Verifica token OATH.

 

 

Funzionalità

Descrizione

Metodo

Selezionare il metodo Telefonata, SMS, App mobile o token OATH.  Per il metodo telefonata, l'utente riceverà una chiamata di Multi-Factor Authentication in cui viene richiesto di premere #, di immettere il PIN o di registrare l'impronta vocale.  Per il metodo SMS l'utente riceverà un SMS con un passcode monouso (OTP) e dovrà rispondere con il passcode monouso o il passcode monouso più il PIN.  Per il metodo app mobile, l'utente riceverà una notifica sull'app mobile per eseguire l'autenticazione ed è possibile che sia necessario immettere il PIN.  Per il metodo token OATH, sarà richiesta l'immissione di un codice di autenticazione dal portale per gli utenti.

 

Nota:   il metodo selezionato è il metodo usato per accedere all'applicazione protetta Multi-Factor Authentication.  È possibile cambiare il metodo successivamente nella pagina Cambia metodo.

 

Codice paese

Selezionare il codice paese appropriato dall'elenco a discesa. Se per l'account è già stato configurato un numero di telefono, il valore predefinito sarà il codice paese di tale numero di telefono.  In caso contrario, per impostazione predefinita viene usato Stati Uniti & Canada +1.

 

Nota: alcune chiamate internazionali prevedono un addebito.  Se l'amministratore non ha implementato questa opzione, per alcuni codici paese non saranno effettuate chiamate.

 

Telefono

Immettere il numero di telefono che verrà usato da Multi-Factor Authentication per l'invio di telefonate o SMS durante l'autenticazione. È possibile usare un numero di telefono cellulare, un numero di telefono fisso o un altro numero per la modalità telefonata.  Per la modalità SMS usare un telefono con funzionalità SMS.  Il formato del numero di telefono deve corrispondere al formato selezionato nel campo Codice paese.

 

Esempio: per gli Stati Uniti e il Canada è necessario immettere un numero di telefono a 10 cifre, ad esempio 555-123-4567.

 

Interno

Immettere l'interno che verrà composto dopo che la chiamata è stata risposta.  Sono consentiti cifre, virgole, * e #.  È possibile usare le virgole per una pausa di un secondo per scorrere i menu di sistema del telefono.  Questo campo è facoltativo.

 

Codice paese di backup

Selezionare il codice paese appropriato per il numero di telefono di backup.

 

Telefono di backup

Immettere il numero di telefono che verrà usato da Multi-Factor Authentication se la chiamata al telefono primario non viene risposta.

 

Estensione di backup

Immettere l'interno per il numero di telefono di backup.

 

PIN

Immettere il PIN.  Per completare l'autenticazione l'utente dovrà immettere questo PIN durante la chiamata di autenticazione, con il passcode monouso in risposta a ogni SMS o nell'app mobile.  È possibile che l'amministratore abbia configurato alcune regole PIN, ad esempio la lunghezza minima.

 

Numero di serie

Immettere il numero di serie stampato sul retro del token OATH.

 

Codice

Immettere il codice attualmente visualizzato sul token OATH.

 

 

Per tornare alla pagina Accedi, fare clic sul pulsante Annulla.
