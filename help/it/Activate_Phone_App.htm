Attiva app mobile

La pagina Attiva app mobile consente di generare un codice di attivazione dell'app mobile per attivare l'app mobile Multi-Factor Authentication sul dispositivo.  Se sono già disponibili dispositivi attivati, essi saranno elencati nella pagina e potranno essere disattivati a seconda delle necessità.

 

Per attivare l'app mobile Multi-Factor Authentication sul dispositivo:

  1. Scegliere il pulsante Genera codice di attivazione.

  2. Immettere il codice di attivazione e l'URL nell'app mobile Multi-Factor Authentication sul dispositivo per completare l'attivazione.
    Nota:  

    • Se viene visualizzato un codice QR, è possibile usare un'app per l'analisi del codice QR in modo da avviare automaticamente l'app mobile Multi-Factor Authentication con il codice di attivazione e l'URL inseriti.

    • Se si accede al portale utenti Multi-Factor Authentication in un browser sul dispositivo, verrà visualizzato il collegamento Attiva anziché il codice di attivazione e l'URL.  Fare clic sul collegamento Attiva per avviare automaticamente l'app mobile Multi-Factor Authentication con il codice di attivazione e l'URL inseriti.

 

Per disattivare un dispositivo:

  1. Fare clic sul collegamento Disattiva accanto al dispositivo.

  2. Confermare la disattivazione facendo clic su OK.
