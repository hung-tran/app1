Utente

Nella pagina Utente vengono visualizzate le informazioni sull'utente e le opzioni per modificare l'account utente.  La pagina viene visualizzata quando viene trovato un singolo utente nella pagina Trova utente o, se vengono trovati più utenti, quando si fa clic su un collegamento utente nella pagina Trova utente.

 

Nell'intestazione della pagina vengono visualizzati il nome e il numero di telefono dell'utente. È inoltre possibile che vengano visualizzate le domande di sicurezza per verificare l'identità dell'utente prima di effettuare modifiche.  Immettere le risposte dell'utente e fare clic sul pulsante Verifica.  Se l'utente non ha definito le domande di sicurezza, sarà visualizzato un messaggio in tal senso.

 

I collegamenti vengono visualizzati per le modifiche che è possibile effettuare all'account dell'utente.  I collegamenti visualizzati dipendono dalle autorizzazioni concesse all'account amministratore e dalla configurazione generale di Multi-Factor Authentication.

 

Funzionalità

Descrizione

Bypass monouso

Selezionare il collegamento Bypass monouso per avviare un  bypass monouso per l'utente selezionato.  L'utente potrà accedere una sola volta senza ricevere una chiamata di Multi-Factor Authentication.

 

Cambia metodo

Selezionare il collegamento Cambia metodo per modificare il metodo dell'utente selezionato.

 

Cambia modalità

Selezionare il collegamento Cambia modalità per modificare le modalità dell'utente selezionato.

 

Cambia telefono

Selezionare il collegamento Cambia telefono per modificare il numero di telefono dell'utente selezionato.

 

Cambia PIN

Selezionare il collegamento Cambia PIN per modificare il PIN dell'utente selezionato.

 

Nota: questo collegamento è disponibile solo se l'utente è in modalità PIN per la telefonata, in modalità OTP + PIN per l'SMS o in modalità PIN per l'app mobile.

 

Cambia lingua

Selezionare il collegamento Cambia lingua per modificare la lingua dell'utente selezionato.

 

Reimposta impronta vocale

Selezionare il collegamento Reimposta impronta vocale per reimpostare l'impronta vocale dell'utente selezionato.

 

Informazioni utente

Selezionare il collegamento informazioni utente per modificare il nome o l'indirizzo di posta elettronica dell'utente.

 

Token OATH

Selezionare il collegamento Token OATH per modificare il token OATH di terze parti dell'utente.

 

Abilita/Disabilita

Selezionare il collegamento Abilita o Disabilita per abilitare o disabilitare l'account Multi-Factor Authentication dell'utente selezionato.

 

Nota: un utente disabilitato potrà accedere senza Multi-Factor Authentication.

 

Blocca/Sblocca utente

Selezionare il collegamento Blocca/Sblocca utente per bloccare o sbloccare l'utente selezionato.

 

Rimuovi utente

Selezionare il collegamento Rimuovi utente per rimuovere definitivamente l'utente selezionato.

 

Nota: un utente rimosso potrà accedere senza Multi-Factor Authentication.

 

Test di autenticazione

Selezionare il collegamento Test di autenticazione per testare l'autenticazione con una chiamata o un SMS di Multi-Factor Authentication per l'utente selezionato.

 

Attiva app mobile

Selezionare il collegamento Attiva app mobile per generare un codice di attivazione dell'app mobile per l'utente selezionato.

 

Disattiva app mobile

Selezionare il collegamento Disattiva app mobile per disattivare un dispositivo esistente per l'utente selezionato.
