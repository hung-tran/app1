User

The User page displays information about the user and options for modifying the user account.  The page is displayed when a single user is found on the Find User page or a user link is clicked on the Find User page when multiple users are found.

 

The user's username and phone number are displayed in the page header and their security questions may be displayed to allow you to verify the user's identity prior to making changes.  Just enter the user's answers and click the Verify button.  If the user has not defined their security questions, a message to that effect will be displayed.

 

Links are displayed for modifications you can make to the user's account.  Not all of the links may be displayed depending on the permissions granted your administrator account and the overall Multi-Factor Authentication configuration.

 

Feature

Description

One-Time Bypass

Select the One-Time Bypass link to initiate a  one-time bypass for the selected user.  The user will be able to sign on a single time without receiving a Multi-Factor Authentication call.

 

Change Method

Select the Change Method link to modify the selected user's method.

 

Change Mode

Select the Change Mode link to modify the selected user's modes.

 

Change Phone

Select the Change Phone link to modify the selected user's phone number.

 

Change PIN

Select the Change PIN link to modify the selected user's PIN number.

 

Note: This link is only available when the user is in phone call PIN mode, text message OTP + PIN mode, or mobile app PIN mode.

 

Change Language

Select the Change Language link to modify the selected user's language.

 

Reset Voiceprint

Select the Reset Voiceprint link to reset the selected user's voiceprint.

 

User Information

Select the User Information link to modify the user's name or email address.

 

OATH Token

Select the OATH Token link to modify the user's third-party OATH token.

 

Enable/Disable

Select the Enable or Disable link to enable or disable the selected user's Multi-Factor Authentication account.

 

Note: A disabled user may be able to sign on without Multi-Factor Authentication.

 

Block/Unblock User

Select the Block/Unblock User link to block or unblock the selected user.

 

Remove User

Select the Remove User link to permanently remove the selected user.

 

Note: A removed user may be able to sign on without Multi-Factor Authentication.

 

Test Authentication

Select the Test Authentication link to test a Multi-Factor Authentication call or Multi-Factor Authentication text message authentication for the selected user.

 

Activate Mobile App

Select the Activate Mobile App link to generate a mobile app activation code for the selected user.

 

Deactivate Mobile App

Select the Deactivate Mobile App link to deactivate an existing device for the selected user.
