Cambiar modo

La página Cambiar modo permite cambiar los modos de la cuenta de usuario seleccionada.  Los métodos de llamada de teléfono, mensaje de texto y aplicación móvil admiten varios modos distintos.

 

Nota:  Es posible que no se muestren las listas desplegables de modos según la configuración del producto.

 

Para cambiar los modos del usuario seleccionado:

  1. Seleccione los modos correspondientes.

  2. Haga clic en el botón Guardar.

 

Característica

Descripción

Modo de llamada de teléfono

Seleccione el modo correspondiente de la lista desplegable. El valor predeterminado es el modo de llamada de teléfono actual del usuario.

 

Modo de mensaje de texto

Seleccione el modo y la dirección correspondientes de las listas desplegables.  El valor predeterminado es el modo y la dirección actuales de mensajes de texto del usuario.

 

Modo de aplicación móvil

Seleccione el modo correspondiente de la lista desplegable.  El valor predeterminado es el modo de aplicación móvil actual del usuario.

 

Enviar correo electrónico

Especifica si se va a enviar un mensaje de correo electrónico para informar al usuario de la actualización.

 

Nota: la casilla Enviar correo electrónico solo se mostrará si se configuró el correo electrónico para su empresa.
