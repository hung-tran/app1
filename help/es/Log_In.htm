Iniciar sesión

Para iniciar sesión en el Portal de usuarios, escriba su nombre de usuario y su contraseña.  A continuación, se realizará una llamada de Multi-Factor Authentication, se enviará un mensaje de texto a su número de teléfono, se enviará una notificación a la aplicación móvil Multi-Factor Authentication o se le solicitará un código de autenticación.  Si completa la autenticación correctamente, se mostrará la página principal.

 

Si la configuración de la cuenta está incompleta, se mostrará la página Configuración de usuario.

 

Si es la primera vez que inicia sesión en el Portal de usuarios, deberá crear las preguntas de seguridad.

 

Si se deniega la autenticación Multi-Factor Authentication, es posible que tenga que responder a las preguntas de seguridad para completar el inicio de sesión.

 

Para iniciar sesión:

  1. Escriba su nombre de usuario y su contraseña.

  2. Haga clic en el botón Iniciar sesión.

 

Característica

Descripción

Nombre de usuario

Nombre de usuario que suele usar para iniciar sesión en su aplicación Multi-Factor Authentication protegida.  Suele ser el nombre de usuario de Windows.

 

Contraseña

Contraseña que suele usar para iniciar sesión en su aplicación Multi-Factor Authentication protegida.  Suele ser la contraseña de Windows.
