Buscar usuario

La página Buscar usuario permite al administrador buscar una cuenta de usuario existente.

 

Para buscar un usuario:

  1. Especifique los criterios de búsqueda.

  2. Haga clic en el botón Buscar.

 

Si un usuario coincide con los criterios especificados, se mostrará la página Información de usuario para dicho usuario.  Si coincide más de un usuario, se mostrará una lista de usuarios como vínculos.  Haga clic en el vínculo del usuario correspondiente para ir a la página Información del usuario.

 

Característica

Descripción

Nombre de usuario

Escriba la cadena de búsqueda para mostrar el nombre de usuario coincidente.

 

Nota: es posible usar caracteres comodín en la cadena de búsqueda. Para ello, escriba un asterisco (*) como equivalente de varios caracteres.

 

Nombre

Escriba la cadena de búsqueda para mostrar el nombre coincidente.

 

Apellidos

Escriba la cadena de búsqueda para mostrar los apellidos coincidentes.

 

Dirección de correo electrónico

Escriba la cadena de búsqueda para mostrar la dirección de correo electrónico coincidente.

 

Grupo de usuarios

Escriba la cadena de búsqueda para mostrar el grupo de usuarios coincidente.

 

Teléfono

Escriba la cadena de búsqueda para mostrar el teléfono coincidente.

 

Etiqueta

Seleccione Todos  para coincidir con todos los usuarios o seleccione una etiqueta específica para que coincidan solo los usuarios con dicha etiqueta.
