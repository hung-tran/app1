Configuración de usuario

Si la configuración de la cuenta no está completa, deberá especificar el número de teléfono (si corresponde) y/o el código PIN (si corresponde) que se van a usar con la cuenta o deberá generar un código de activación para activar la Aplicación móvil Multi-Factor Authentication.  Si su empresa lo permite, puede elegir entre los métodos de llamada de teléfono, mensaje de texto, aplicación móvil y token OATH.  Dependiendo del modo seleccionado, se realizará una llamada de Multi-Factor Authentication o se enviará un mensaje de texto al número de teléfono especificado como válido, se enviará una notificación a la aplicación móvil o se solicitará que especifique un código de autenticación.  Si seleccionó el método de llamada de teléfono, deberá presionar la tecla almohadilla (#), escribir el código PIN o inscribir su huella de voz, según el modo de llamada de teléfono seleccionado.  Si optó por el método de mensaje de texto, se le enviará un código de acceso de un solo uso (OTP) al que deberá responder con el OTP o el OTP con el código PIN.  Si seleccionó el método de aplicación móvil, se enviará una notificación a la aplicación móvil en la que deberá autenticarse. Según el modo de aplicación móvil seleccionado, es posible que tenga que escribir su código PIN.  Al seleccionar el método de token OATH, el Portal de usuarios le pedirá un Código de autenticación.  Si realiza el proceso correctamente, deberá crear las preguntas de seguridad.

 

Para completar la configuración de usuario para el método de llamada de teléfono:

  1. Escriba el código de país y el número de teléfono.  También puede especificar un número de teléfono de reserva (si corresponde).

  2. Escriba el código PIN (si corresponde).

  3. Haga clic en el botón Llamarme ahora para autenticar.

 

Para completar la configuración de usuario para el método de mensaje de texto:

  1. Escriba el código de país y el número de teléfono.

  2. Escriba el código PIN (si corresponde).

  3. Haga clic en el botón Enviarme un SMS ahora para autenticar.

 

Para completar la configuración de usuario para el método de aplicación móvil:

  1. Instale la aplicación móvil en su teléfono.

  2. Haga clic en el botón Generar código de activación.

  3. Escriba el código de activación y la URL en la aplicación móvil para completar la activación.

  4. Escriba el código PIN (si corresponde).

  5. Haga clic en el botón Autenticarme ahora.

 

Para completar la configuración de usuario para el método de token OATH:

  1. Instale la aplicación móvil en su teléfono.

  2. Haga clic en el botón Generar código de activación.

  3. Escriba el código de activación y la URL en la aplicación móvil para completar la activación.

  4. Haga clic en el botón Autenticarme ahora.

O bien

  1. Escriba el número de serie impreso en la parte posterior del token OATH.

  2. Escriba el código que se muestra actualmente en el token OATH.

  3. Haga clic en el botón Comprobar token OATH.

 

 

Característica

Descripción

Método

Seleccione el método de Llamada de teléfono, Mensaje de texto, Aplicación móvil o Token OATH.  Para el método de llamada de teléfono, recibirá una llamada de la Multi-Factor Authentication que le solicitará que presione la tecla almohadilla (#), que indique su código PIN o que inscriba su huella de voz.  Si optó por el método de mensaje de texto, se le enviará un código de acceso de un solo uso (OTP) en un mensaje de texto al que deberá responder con el OTP o el OTP con el código PIN.  En el caso del método de aplicación móvil, se enviará una notificación a la aplicación móvil con la que deberá autenticarse. Es posible que tenga que especificar el código PIN.  Al seleccionar el método de token OATH, el Portal de usuarios le pedirá un Código de autenticación.

 

Nota:   el método seleccionado es el método que usará para tener acceso a la aplicación protegida mediante Multi-Factor Authentication.  Es posible cambiar este método más adelante en la página Cambiar método.

 

Código de país

Seleccione el código de país correspondiente en la lista desplegable. Si ya configuró un número de teléfono para su cuenta, de forma predeterminada se tomará el código de país de dicho número de teléfono.  De lo contrario, se establecerá el valor predeterminado Estados Unidos y Canadá +1.

 

Nota: algunas llamadas internacionales pueden suponer costes.  Si su administrador no implementó esta opción, no se realizarán llamadas a determinados códigos de país.

 

Teléfono

Escriba el número de teléfono al que Multi-Factor Authentication llamará o enviará mensajes de texto durante la autenticación. Este número puede ser un número de teléfono móvil, un número de teléfono de escritorio u otro número especificado para el modo de llamada de teléfono.  Si optó por el modo de mensaje de texto, debe usar un teléfono con capacidades de mensaje de texto.  El formato de número de teléfono debe coincidir con el formato seleccionado en el campo Código de país.

 

Ejemplo: para números de teléfono de Estados Unidos y Canadá, debe especificar un número de teléfono de 10 dígitos (por ejemplo, 555-123-4567).

 

Extensión

Escriba la extensión que se marcará después de que se conteste a la llamada.  Solo se admiten dígitos, comas, * y #.  Las comas se pueden utilizar para incorporar una pausa de un segundo para navegar en menús de sistemas telefónicos.  El campo es opcional.

 

Código de país de reserva

Seleccione el código de país correspondiente para el número de teléfono de reserva.

 

Teléfono de reserva

Escriba el número de teléfono al que llamará Multi-Factor Authentication si no se responde la llamada al teléfono principal.

 

Extensión de reserva

Escriba la extensión del número de teléfono de reserva.

 

PIN

Escriba un código PIN.  Para completar la autenticación, deberá especificar este código PIN en todas las llamadas de autenticación, en las respuestas a cada mensaje de texto junto con el código de acceso de un solo uso (OTP) o en la aplicación móvil.  Es posible que el administrador haya configurado reglas de PIN como, por ejemplo, de longitud mínima.

 

Número de serie

Escriba el número de serie impreso en la parte posterior del token OATH.

 

Código

Escriba el código que se muestra actualmente en el token OATH.

 

 

Para volver a la página Iniciar sesión, haga clic en el botón Cancelar.
