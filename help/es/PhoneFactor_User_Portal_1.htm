Portal de usuarios de Multi-Factor Authentication

El Portal de usuarios de Multi-Factor Authentication permite omitir la autenticación Multi-Factor Authentication en un inicio de sesión, administrar la configuración de la cuenta de Multi-Factor Authentication, incluido el método, el número de teléfono, el código PIN, el idioma y las preguntas de seguridad. Asimismo, permite activar la aplicación móvil de Multi-Factor Authentication. Es posible que el administrador haya deshabilitado algunas de estas características.  Vea las Preguntas más frecuentes para obtener información acerca del uso de Multi-Factor Authentication.

 

La tabla de contenido, el índice y la característica de búsqueda le ayudarán a localizar la información en el sistema de ayuda.
