Token OATH

La página Token OATH permite asociar el usuario seleccionado con un token OATH de terceros.  El token debe definirse en el Servidor Multi-Factor Authentication antes de asociarlo a un usuario.  El token debe admitir códigos OATH por tiempo.  Se trata de una alternativa al uso de la aplicación móvil Multi-Factor Authentication para códigos OATH.

 

Para especificar un token OATH:

  1. Escriba los valores correspondientes.

  2. Haga clic en el botón Guardar.

 

Característica

Descripción

Número de serie

Escriba el número de serie del token OATH de terceros.

 

Código

Escriba el código que se muestra actualmente en el token OATH de terceros.
