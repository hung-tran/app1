Preguntas más frecuentes

¿Cómo funciona la autenticación Multi-Factor Authentication™?

La autenticación Multi-Factor Authentication se completa cuando se realiza una llamada de teléfono, se envía un mensaje de texto, se notifica a la aplicación móvil Multi-Factor Authentication al iniciar sesión o al solicitar un token o código de autenticación a la aplicación móvil Multi-Factor Authentication.

 

Paso 1:

Escriba su nombre de usuario habitual y su contraseña.

Paso 2:

Al instante, recibirá una llamada de teléfono, un mensaje de texto o una notificación de aplicación móvil, o bien se le pedirá que especifique un código de autenticación. Para el método de llamada de teléfono, responda a la llamada y presione la almohadilla (#), especifique el código PIN o diga su frase de huella de voz.  Para el método de mensaje de texto, responda al mensaje con el código de acceso de un solo uso (OTP) o responda con el OTP + código PIN.  Para el método de aplicación móvil, presione Autenticar o escriba el código PIN y presione Autenticar en la aplicación móvil Multi-Factor Authentication instalada en el dispositivo.  Para el método de token OATH, escriba el token o el código de autenticación que se muestra en la aplicación móvil Multi-Factor Authentication.

¡Eso es todo!

 

Este sencillo proceso proporciona dos factores independientes de autenticación a través de dos canales distintos (el equipo y el servicio telefónico). Funciona en modo de llamada de teléfono con cualquier teléfono o teléfono móvil, en modo de mensaje de texto con cualquier teléfono con capacidades de mensaje de texto o con cualquier dispositivo compatible con la aplicación móvil Multi-Factor Authentication.

 

¿Qué sucede si pierdo mi teléfono?

Seleccione la opción Cambiar número de teléfono para especificar un nuevo número de teléfono. También puede configurar un número alternativo mediante una llamada al servicio de soporte técnico tras establecer su identidad.

 

¿Qué sucede si pierdo la cobertura del teléfono móvil en un área determinada?

Puede cambiar la cuenta para que apunte a un número de teléfono alternativo seleccionando la opción Cambiar número de teléfono. Algunas empresas también ofrecen la opción Omisión por única vez, que permite iniciar sesión una vez sin llamada de teléfono ni mensaje de texto. Si está disponible, seleccione la opción Omisión por única vez para tener acceso a esta característica. También puede llamar al servicio de soporte técnico desde cualquier teléfono para obtener ayuda.  El método de token OATH no requiere cobertura del teléfono móvil.

 

¿Qué sucede si recibo una llamada telefónica, un mensaje de texto o una notificación en la aplicación móvil de Multi-Factor Authentication si no estoy intentando iniciar sesión?

Esto solo podría ocurrir si otra persona que conoce su contraseña tratara de iniciar sesión en su cuenta. Recuerde que solo se realizan llamadas de teléfono, se envían mensajes de texto o se envían notificaciones a la aplicación móvil tras comprobar el nombre de usuario y la contraseña. Por lo tanto, incluso en este caso, con la autenticación Multi-Factor Authentication su cuenta seguiría estando protegida frente al acceso ilícito. Para bloquear temporalmente los intentos de acceso a su cuenta, siga las indicaciones del menú del teléfono (método de llamada de teléfono), responda al mensaje de texto con el código de fraude asignado por la empresa (método de mensaje de texto) o presione Denegar y notificar fraude en la aplicación móvil.  Si está habilitada la Alerta de fraude para su empresa, además de bloquear su cuenta, se enviará de forma instantánea una alerta al equipo de seguridad de TI de su empresa. Póngase en contacto con servicio de soporte técnico para obtener ayuda acerca del desbloqueo de su cuenta.
