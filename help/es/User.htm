Usuario

La página Usuario muestra información sobre el usuario, así como opciones para modificar la cuenta de usuario.  Esta página aparece cuando solo se encuentra a un usuario en la página Buscar usuario o al hacer clic en un vínculo de usuario de la página Buscar usuario cuando se encuentran varios usuarios.

 

En el encabezado de la página se muestran el nombre y el número de teléfono del usuario. También es posible que se muestren las preguntas de seguridad del usuario, lo que le ayudará a comprobar la identidad del usuario antes de realizar cambios.  Escriba las respuestas del usuario y haga clic en el botón Verificar.  Si el usuario no definió sus preguntas de seguridad, se mostrará un mensaje a tal efecto.

 

Los vínculos mostrados corresponden a las modificaciones que puede realizar en la cuenta del usuario.  Es posible que no se muestren todos los vínculos, ya que ello depende de los permisos concedidos a la cuenta de su administrador y de la configuración general de Multi-Factor Authentication.

 

Característica

Descripción

Omisión por única vez

Seleccione el vínculo Omisión por única vez para iniciar una  omisión por única vez para el usuario seleccionado.  El usuario podrá iniciar sesión una sola vez sin recibir una llamada de Multi-Factor Authentication.

 

Cambiar método

Seleccione el vínculo Cambiar método para modificar el método del usuario seleccionado.

 

Cambiar modo

Seleccione el vínculo Cambiar modo para modificar el modo del usuario seleccionado.

 

Cambiar teléfono

Seleccione el vínculo Cambiar teléfono para modificar el número de teléfono del usuario seleccionado.

 

Cambiar PIN

Seleccione el vínculo Cambiar PIN para modificar el código PIN del usuario seleccionado.

 

Nota: este vínculo solo estará disponible cuando el usuario tenga seleccionado el modo PIN para la llamada de teléfono, el modo OTP + PIN para el mensaje de texto o el modo PIN para la aplicación móvil.

 

Cambiar idioma

Seleccione el vínculo Cambiar idioma para modificar el idioma del usuario seleccionado.

 

Restablecer huella de voz

Seleccione el vínculo Restablecer huella de voz vínculo para restablecer la huella de voz del usuario seleccionado.

 

Información de usuario

Seleccione el vínculo Información de usuario vínculo para modificar la dirección de correo electrónico o el nombre de usuario.

 

Token OATH

Seleccione el vínculo Token OATH para modificar el token OATH de terceros del usuario.

 

Habilitar/deshabilitar

Seleccione el vínculo Habilitar o Deshabilitar para habilitar o deshabilitar la cuenta Multi-Factor Authentication del usuario seleccionado.

 

Nota: un usuario deshabilitado puede iniciar una sesión sin Multi-Factor Authentication.

 

Bloquear/desbloquear usuario

Seleccione el vínculo Bloquear/desbloquear usuario para bloquear o desbloquear el usuario seleccionado.

 

Quitar usuario

Seleccione el vínculo Quitar usuario vínculo para quitar definitivamente el usuario seleccionado.

 

Nota: un usuario quitado puede iniciar una sesión sin Multi-Factor Authentication.

 

Probar autenticación

Seleccione el vínculo Probar autenticación para probar la autenticación mediante llamada de Multi-Factor Authentication o mensaje de texto de Multi-Factor Authentication para el usuario seleccionado.

 

Activar aplicación móvil

Seleccione el vínculo Activar aplicación móvil para generar un código de activación de aplicación móvil para el usuario seleccionado.

 

Desactivar aplicación móvil

Seleccione el vínculo Desactivar aplicación móvil para desactivar un dispositivo existente para el usuario seleccionado.
