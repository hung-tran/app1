Activar aplicación móvil

La página Activar aplicación móvil permite generar un código de activación de aplicación móvil para activar la aplicación móvil Multi-Factor Authentication en el dispositivo.  Si ya ha activado dispositivos, estos se mostrarán y se podrán desactivar según sea necesario.

 

Para activar la aplicación móvil Multi-Factor Authentication en el dispositivo:

  1. Haga clic en el botón Generar código de activación.

  2. Escriba el código de activación y la URL en la aplicación móvil Multi-Factor Authentication de su dispositivo para completar la activación.
    Nota:  

    • si se muestra un código QR, es posible que tenga que usar una aplicación de escáner de códigos QR para iniciar automáticamente la aplicación móvil Multi-Factor Authentication con el código de activación y la URL cumplimentados.

    • Si visita el Portal de usuarios de Multi-Factor Authentication en un explorador del dispositivo, se mostrará el vínculo Activar en lugar del código de activación y la URL.  Al hacer clic en el vínculo Activar se iniciará automáticamente la aplicación móvil Multi-Factor Authentication con el código de activación y la URL cumplimentados.

 

Para desactivar un dispositivo:

  1. Haga clic en el vínculo Desactivar situado junto al dispositivo.

  2. Confirme la desactivación haciendo clic en Aceptar.
