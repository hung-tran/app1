Activar aplicación móvil

La página Activar aplicación móvil permite generar un código de activación de la aplicación móvil para el usuario seleccionado.

 

Para generar un código de activación de la aplicación móvil para el usuario seleccionado:

  1. Haga clic en el botón Generar código de activación.

  2. Proporcione el código de activación y la URL al usuario para que pueda especificarlos en la aplicación móvil Multi-Factor Authentication de su dispositivo para completar la activación.
