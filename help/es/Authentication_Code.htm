Código de autenticación

Para el modo de token OATH, el Portal de usuarios solicitará el código de autenticación que se muestra en el token o en la aplicación móvil Multi-Factor Authentication.

 

Para completar el inicio de sesión:

  1. Escriba el código de autenticación que se muestra en el token o en la aplicación móvil Multi-Factor Authentication.

  2. Haga clic en el botón Autenticar.

 

Para volver a la página Iniciar sesión, haga clic en el botón Cancelar.
