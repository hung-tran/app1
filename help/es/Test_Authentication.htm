Probar autenticación

La página Probar autenticación permite probar una llamada de Multi-Factor Authentication, una autenticación mediante mensaje o una autenticación mediante Multi-Factor Authentication para el usuario seleccionado.  Si se realiza una llamada, se envía un mensaje de texto o se notifica a la aplicación móvil Multi-Factor Authentication según el modo del usuario.

 

Haga clic en el botón Probar para iniciar la llamada, enviar el mensaje de texto o notificar a la aplicación móvil de Multi-Factor Authentication.
