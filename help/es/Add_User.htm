Agregar usuario

La página Agregar usuario permite crear una nueva cuenta de Multi-Factor Authentication.

 

Nota:  muchas opciones de usuario como, por ejemplo, el método (Llamada de teléfono, Mensaje de texto o Aplicación móvil), se establecen de forma predeterminada según los valores predeterminados del usuario especificados en la sección Configuración de la empresa del Servidor Multi-Factor Authentication.

 

Para agregar un nuevo usuario:

  1. Escriba los detalles del usuario.

  2. Haga clic en el botón Agregar.

 

Característica

Descripción

Nombre de usuario

El Nombre de usuario suele ser el correo electrónico del usuario en el formato de correo electrónico correcto.

 

Ejemplo: chuck@abc.com

 

Nombre

Escriba el nombre del usuario.

 

Apellidos

Escriba los apellidos del usuario.

 

Dirección de correo electrónico

Escriba la dirección de correo electrónico del usuario.  Los mensajes de correo electrónico se enviarán a esta dirección de correo electrónico.  Si no se especifica, los mensajes de correo electrónico se enviarán al nombre de usuario siempre que este tenga el formato de correo electrónico correcto.

 

Grupo de usuarios

Escriba el grupo de usuarios del usuario.

 

Código de país

Seleccione el código de país correspondiente en la lista desplegable. El valor predeterminado establecido es Estados Unidos y Canadá +1.

 

Nota: algunas llamadas internacionales pueden suponer costes.  Si su empresa no implementó esta opción, no se realizarán llamadas a determinados códigos de país.

 

Teléfono

Escriba el número de teléfono al que Multi-Factor Authentication llamará o enviará mensajes de texto durante la autenticación. Este número puede ser un número de teléfono móvil, un número de teléfono de escritorio u otro número especificado para el método de llamada.  Asimismo, si se seleccionó el método de mensaje de texto, el teléfono debe tener capacidades de mensaje de texto.  El formato de número de teléfono debe coincidir con el formato seleccionado en el campo Código de país.

 

Ejemplo: para números de teléfono de Estados Unidos y Canadá, debe especificar un número de teléfono de 10 dígitos (por ejemplo, 555-123-4567).

 

Extensión

Escriba la extensión que se marcará después de que se conteste a la llamada.  Solo se admiten dígitos, comas, * y #.  Las comas se pueden utilizar para incorporar una pausa de un segundo para navegar en menús de sistemas telefónicos.  El campo es opcional.

 

Código de país de reserva

Seleccione el código de país correspondiente para el número de teléfono de reserva.

 

Teléfono de reserva

Escriba el número de teléfono al que llamará Multi-Factor Authentication si no se responde la llamada al teléfono principal.

 

Extensión de reserva

Escriba la extensión del número de teléfono de reserva.

 

PIN

Escriba el código PIN del usuario.  O bien, puede generar el código PIN haciendo clic en el botón Generar.  El código PIN debe ser conforme a las reglas de longitud mínima de código PIN y de seguridad del código PIN.

 

Botón Generar

El botón Generar proporciona un código PIN generado de forma aleatoria.  El código PIN generado tendrá una longitud superior a 4 dígitos o a la longitud mínima del código PIN del usuario.

 

Confirmar PIN

Vuelva a escribir el código PIN especificado en el campo PIN. El PIN de confirmación debe coincidir.

 

Habilitado

Especifica si el usuario está habilitado.  La opción está activada de manera predeterminada.

 

Nota: si un registro de usuario no está habilitado, dicho usuario no recibirá llamadas de Multi-Factor Authentication al inicia sesión y es posible que se permita o deniegue el acceso a la aplicación según el comportamiento predeterminado de la empresa para usuarios deshabilitados.

 

Se requiere cambiar el PIN

Especifica que un usuario debe cambiar su código PIN en la siguiente llamada de Multi-Factor Authentication.

 

Nota: la casilla Se requiere cambiar el PIN solo se mostrará si la configuración de la empresa permite a los usuarios cambiar su código PIN.

 

Enviar correo electrónico

Especifica si se debe enviar un correo electrónico al usuario que se agregó.

 

Nota: la casilla Enviar correo electrónico solo se mostrará si se configuró el correo electrónico para su empresa.
