ユーザーのセットアップ

アカウントの Phone アプリが完了していない場合、アカウントに使用する電話番号 (該当する場合)、PIN (該当する場合)、またはその両方を指定する必要があります。または、Multi-Factor Authentication モバイル アプリをアクティブ化するためにアクティブ化コードの生成を求められる場合があります。 会社で許可されている場合、電話、テキスト メッセージ、モバイル アプリ、および OATH トークンの認証方法から選択できます。 指定した電話番号に Multi-Factor Authentication の電話がかかってくるか、テキスト メッセージが送信され、有効であることが確認されます。または、通知がモバイル アプリに送信されるか、認証コードの入力が求められます。 電話による認証方法の場合、電話モードによって、# を押して PIN を入力するか、ボイスプリントを登録するように求められます。 テキスト メッセージによる認証方法の場合、ワンタイム パスコード (OTP) が送信され、その OTP または OTP と自分の PIN を入力して返信します。 モバイル アプリによる認証方法の場合、通知がモバイル アプリに送信され、認証するように求められます。このとき、モバイル アプリ モードによっては PIN を入力する必要があります。 OATH トークンによる認証方法の場合、ユーザー ポータルから認証コードの入力が求められます。 成功すると、セキュリティの質問を作成するように求められます。

 

電話による認証方法のユーザー セットアップを完了するには:

  1. 国コードと電話番号を入力します。 予備の電話番号 (該当する場合) を入力する場合もあります。

  2. PIN (該当する場合) を入力します。

  3. [今すぐ電話で認証] ボタンをクリックします。

 

テキスト メッセージによる認証方法のユーザー セットアップを完了するには:

  1. 国コードと電話番号を入力します。

  2. PIN (該当する場合) を入力します。

  3. [今すぐテキスト メッセージで認証] ボタンをクリックします。

 

モバイル アプリによる認証方法のユーザー セットアップを完了するには:

  1. 携帯電話にモバイル アプリをインストールします。

  2. [アクティブ化コードの生成] ボタンをクリックします。

  3. モバイル アプリにアクティブ化コードと URL を入力して、アクティブ化を完了します。

  4. PIN (該当する場合) を入力します。

  5. [今すぐ認証] ボタンをクリックします。

 

OATH トークンによる認証方法のユーザー セットアップを完了するには:

  1. 携帯電話にモバイル アプリをインストールします。

  2. [アクティブ化コードの生成] ボタンをクリックします。

  3. モバイル アプリにアクティブ化コードと URL を入力して、アクティブ化を完了します。

  4. [今すぐ認証] ボタンをクリックします。

または

  1. OATH トークンの裏に印刷されているシリアル番号を入力します。

  2. OATH トークンに現在表示されているコードを入力します。

  3. [OATH トークンの確認] ボタンをクリックします。

 

 

機能

説明

認証方法

電話、テキスト メッセージ、モバイル アプリ、または OATH トークンの認証方法を選択します。 電話による認証方法の場合、Multi-Factor Authentication の電話を受け、# を押して PIN を入力するか、ボイスプリントを登録するように求められます。 テキスト メッセージによる認証方法の場合、テキスト メッセージでワンタイム パスコード (OTP) が送信され、その OTP または OTP と自分の PIN を入力して返信します。 モバイル アプリによる認証方法の場合、通知がモバイル アプリに送信され、認証するように求められます。このとき、場合によっては PIN を入力する必要があります。 OATH トークンによる認証方法の場合、ユーザー ポータルから認証コードの入力が求められます。

 

注:   選択した認証方法は、Multi-Factor Authentication で保護されたアプリケーションにアクセスするときに使用される認証方法になります。  認証方法は、[認証方法の変更] ページでいつでも変更できます。

 

国コード

ドロップダウン リストから適切な国コードを選択します。アカウントの電話番号を構成済みの場合、既定はその電話番号の国コードです。 構成していない場合、既定は [米国およびカナダ +1] に設定されます。

 

注:国際電話には料金がかかる可能性があります。 管理者がこのオプションを実装していない場合、一部の国コードに対する電話は発信されません。

 

電話

Multi-Factor Authentication が認証中に電話をかけるかテキストを送信する電話番号を入力します。電話モードの場合、携帯電話番号、固定電話番号、またはその他の番号を指定できます。 テキスト メッセージ モードの場合、テキスト メッセージ機能を持つ電話が必要です。 電話番号の形式は、[国コード] フィールドで選択した形式と一致する必要があります。

 

例:米国とカナダの電話番号の場合、10 桁の電話番号を入力する必要があります (555-123-4567 など)。

 

内線

電話がつながった後にダイヤルする内線番号を入力します。 数字、コンマ、*、# だけを使用してください。 電話システムのメニューに移動するために 1 秒間途切れさせるにはコンマを使用します。 このフィールドはオプションです。

 

予備の国コード

予備の電話番号に適切な国コードを選択します。

 

予備の電話番号

メインの電話が応答しない場合に Multi-Factor Authentication が発信する電話番号を入力します。

 

予備の内線

予備の電話番号の内線を入力します。

 

PIN

PIN 番号を入力します。 この PIN を使用するのは、認証の電話を受けて入力するとき、テキスト メッセージの返信でワンタイム パスコード (OTP) と共に入力するとき、または認証を完了するためにモバイル アプリで入力するときです。 管理者が、最短の文字数など、PIN の規則を構成していることがあります。

 

シリアル番号

OATH トークンの裏に印刷されているシリアル番号を入力します。

 

コード

OATH トークンに現在表示されているコードを入力します。

 

 

[ログイン] ページに戻るには、[キャンセル] ボタンをクリックします。
