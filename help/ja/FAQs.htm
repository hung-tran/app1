よく寄せられる質問

Multi-Factor Authentication™ はどのように機能しますか。

Multi-Factor Authentication は、確認の電話をかける、テキスト メッセージを送信する、ログイン時に Multi-Factor Authentication モバイル アプリで通知する、および Multi-Factor Authentication モバイル アプリまたはトークンの認証コードの入力を求めるという方法で認証を行っています。

 

手順 1:

通常のユーザー名とパスワードを入力します。

手順 2:

すぐに電話の着信、テキスト メッセージの着信、またはモバイル アプリの通知があるか、認証コードの入力を求められます。電話による認証方法の場合、# を押して PIN を入力するか、ボイスプリントのフレーズを言います。 テキスト メッセージによる認証方法の場合、メッセージに記載されたワンタイム パスコード (OTP)、または OTP と自分の PIN を入力して返信します。 モバイル アプリによる認証方法の場合、デバイスにインストールされている Multi-Factor Authentication モバイル アプリで [認証] を押すか、PIN を入力して [認証] を押します。 OATH トークンによる認証方法の場合、Multi-Factor Authentication モバイル アプリまたはトークンに表示されている認証コードを入力します。

これで完了です。

 

この簡単なプロセスは、2 つの異なるチャンネル (コンピューターと電話サービス) を介して、2 要素の認証を提供しています。Multi-Factor Authentication は、電話モードは任意の固定電話または携帯電話、テキスト メッセージ モードはテキスト メッセージ機能を持つ任意の電話、または Multi-Factor Authentication モバイル アプリをサポートするデバイスで使用できます。

 

電話機をなくした場合は、どうすればよいですか。

[電話番号の変更] オプションを選択して、新しい電話番号を入力してください。また、サポート ヘルプ デスクに電話をかけ、お客様の身元が確認された後に、代替の番号をセットアップすることもできます。

 

携帯電話の電波が届かない場所ではどうなるのですか。

[電話番号の変更] オプションを選択すると、アカウントで使用する電話番号を変更できます。会社によっては、電話またはテキスト メッセージを使用せずに 1 度だけログインできる [ワンタイム バイパス] オプションを使用できることがあります。使用できる場合、[ワンタイム バイパス] オプションを選択してこの機能にアクセスします。または、任意の電話からサポート ヘルプ デスクに連絡して相談します。 OATH トークンによる認証方法の場合、携帯電話の通信可能地域にいる必要はありません。

 

ログインしようとしていないときに、Multi-Factor Authentication から電話、テキスト メッセージ、またはモバイル アプリの通知を受けた場合はどうすればよいですか。

他の誰かがお客様のアカウントにログインを試み、パスワードを既に知っている場合、このような状況になります。電話、テキスト メッセージの送信、またはモバイル アプリの通知は、ユーザー名とパスワードが検証された後にのみ行われます。そのため、この問題が発生した場合、Multi-Factor Authentication が不正なアクセスによってアカウントを保存したことを示します。アカウントの認証試行を一時的にブロックするには、携帯電話による認証方法の場合は、電話メニューのプロンプトに従って操作します。テキスト メッセージによる認証方法の場合は、会社から割り当てられた不正コードを記載して返信します。モバイル アプリの場合は、[認証を拒否して不正を通報] を押します。 会社が [不正アクセスのアラート] を有効にしている場合、アカウントをブロックするだけでなく、直ちにアラートが会社の IT セキュリティ チームに送信されます。アカウントのブロックを解除するには、会社のサポート ヘルプ デスクに連絡してください。
