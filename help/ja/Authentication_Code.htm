認証コード

OATH トークン モードの場合、ユーザー ポータルは、Multi-Factor Authentication モバイル アプリまたはトークンに表示されている認証コードの入力を求めます。

 

ログインを完了するには:

  1. Multi-Factor Authentication モバイル アプリまたはトークンに表示されている認証コードを入力します。

  2. [認証] ボタンをクリックします。

 

[ログイン] ページに戻るには、[キャンセル] ボタンをクリックします。
