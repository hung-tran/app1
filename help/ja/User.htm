ユーザー

[ユーザー] ページには、ユーザー アカウントを変更するユーザーとオプションに関する情報が表示されます。 このページは、[ユーザーの検索] ページで単一のユーザーが見つかったとき、または複数のユーザーが見つかったときに [ユーザーの検索] ページのユーザー リンクをクリックしたときに表示されます。

 

変更を行う前にユーザーの身元を検証できるように、ユーザーの名前と電話番号がページ ヘッダーに表示されます。ユーザーのセキュリティの質問が表示されることもあります。 ユーザーの答を入力して、[検証] ボタンをクリックします。 ユーザーがセキュリティの質問を定義していない場合、それに対応するメッセージが表示されます。

 

ユーザーのアカウントを変更するためのリンクが表示されます。 管理者アカウントに付与されたアクセス許可と全体的な Multi-Factor Authentication の構成によっては、表示されないリンクもあります。

 

機能

説明

ワンタイム バイパス

[ワンタイム バイパス] リンクをクリックして、 選択したユーザーのワンタイム バイパスを開始します。 ユーザーは、Multi-Factor Authentication の電話を受けることなく、1 回でサインオンできます。

 

認証方法の変更

[認証方法の変更] リンクをクリックして、選択したユーザーの方法を変更します。

 

モードの変更

[モードの変更] リンクをクリックして、選択したユーザーのモードを変更します。

 

電話番号の変更

[電話番号の変更] リンクをクリックして、選択したユーザーの電話番号を変更します。

 

PIN の変更

[PIN の変更] リンクをクリックして、選択したユーザーの PIN 番号を変更します。

 

注: このリンクを使用できるのは、ユーザーが電話 PIN モード、テキスト メッセージ OTP と PIN モード、またはモバイル アプリ PIN モードの場合のみです。

 

言語の変更

[言語の変更] リンクをクリックして、選択したユーザーの言語を変更します。

 

ボイスプリントのリセット

[ボイスプリントのリセット] リンクをクリックして、選択したユーザーのボイスプリントをリセットします。

 

ユーザー情報

[ユーザー情報] リンクをクリックして、ユーザーの名前と電子メール アドレスを変更します。

 

OATH トークン

[OATH トークン] リンクをクリックして、ユーザーのサードパーティ OATH トークンを変更します。

 

有効/無効

[有効/無効] リンクをクリックして、選択したユーザーの Multi-Factor Authentication アカウントの有効または無効を切り替えます。

 

注: 無効なユーザーは、Multi-Factor Authentication なしでサインインできます。

 

ユーザーのブロック設定

[ユーザーのブロック設定] リンクをクリックして、選択したユーザーのブロックまたはブロック解除を切り替えます。

 

ユーザーの削除

[ユーザーの削除] リンクをクリックして、選択したユーザーを完全に削除します。

 

注: 削除されたユーザーは、Multi-Factor Authentication なしでサインインできます。

 

認証のテスト

[認証のテスト] リンクをクリックして、選択したユーザーに関する Multi-Factor Authentication の電話または Multi-Factor Authentication のテキスト メッセージの認証をテストします。

 

モバイル アプリのアクティブ化

[モバイル アプリのアクティブ化] リンクをクリックして、選択したユーザーのモバイル アプリ アクティブ化コードを生成します。

 

モバイル アプリの非アクティブ化

[モバイル アプリの非アクティブ化] リンクをクリックして、選択したユーザーの既存のデバイスを非アクティブ化します。
