電話番号の変更

[電話番号の変更] ページでは、選択したユーザー アカウントの電話番号を変更できます。 これは一時的な変更ではありません。以降、新しく指定した電話番号が Multi-Factor Authentication の電話とテキスト送信に使用されます。

 

選択したユーザーの電話番号を変更するには:

  1. 適切な国コードを選択します。

  2. 新しい電話番号を入力します。

  3. [保存] ボタンをクリックします。

 

機能

説明

国コード

ドロップダウン リストから適切な国コードを選択します。既定はユーザーが現在使用している国コードです。

 

注:国際電話には料金がかかる可能性があります。 会社がこのオプションを実装していない場合、一部の国コードに対する電話は発信されません。

 

電話

Multi-Factor Authentication が認証中に電話をかけるかテキストを送信する電話番号を入力します。電話による認証方法として、携帯電話番号、固定電話番号、またはその他の番号を指定できます。 テキスト メッセージによる認証方法を使用するには、テキスト メッセージ機能を持つ電話が必要です。 電話番号の形式は、[国コード] フィールドで選択した形式と一致する必要があります。

 

例:米国とカナダの電話番号の場合、10 桁の電話番号を入力する必要があります (555-123-4567 など)。

 

内線

電話がつながった後にダイヤルする内線番号を入力します。 数字、コンマ、*、# だけを使用してください。 電話システムのメニューに移動するために 1 秒間途切れさせるにはコンマを使用します。 このフィールドはオプションです。

 

予備の国コード

予備の電話番号に適切な国コードを選択します。

 

予備の電話番号

メインの電話が応答しない場合に Multi-Factor Authentication が発信する電話番号を入力します。

 

予備の内線

予備の電話番号の内線を入力します。

 

電子メールの送信

ユーザーに更新について通知する電子メールを送信するかどうかを指定します。

 

注: [電子メールを送信する] チェックボックスが表示されるのは、会社の電子メールが構成されている場合のみです。
