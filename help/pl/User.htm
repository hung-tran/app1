Użytkownik

Strona Użytkownik wyświetla informacje o użytkowniku i opcje służące do modyfikowania konta użytkownika.  Ta strona jest wyświetlana po znalezieniu jednego użytkownika na stronie Znajdowanie użytkownika lub kliknięciu łącza użytkownika na stronie Znajdowanie użytkownika, jeśli znaleziono wielu użytkowników.

 

Nazwa użytkownika i numer telefonu są wyświetlane w nagłówku strony. Mogą także zostać wyświetlone pytania zabezpieczające, aby umożliwić zweryfikowanie tożsamości użytkownika przed dokonaniem zmian.  Wprowadź odpowiedzi użytkownika i kliknij przycisk Weryfikuj.  Jeśli użytkownik nie zdefiniował pytań zabezpieczających, pojawi się odpowiedni komunikat.

 

Wyświetlane łącza umożliwiają wprowadzanie zmian na koncie użytkownika.  Mogą nie być wyświetlane wszystkie łącza, co jest zależne od uprawnień przyznanych dla konta administratora i ogólnej konfiguracji usługi Multi-Factor Authentication.

 

Funkcja

Opis

Jednorazowe obejście

Wybierz łącze Jednorazowe obejście, aby zainicjować  jednorazowe obejście dla wybranego użytkownika.  Użytkownik będzie mógł zalogować się jeden raz bez odbierania połączenia usługi Multi-Factor Authentication.

 

Zmień metodę

Wybierz łącze Zmień metodę, aby zmodyfikować metodę wybranego użytkownika.

 

Zmień tryb

Wybierz łącze Zmień tryb, aby zmodyfikować tryby wybranego użytkownika.

 

Zmień numer telefonu

Wybierz łącze Zmień numer telefonu, aby zmodyfikować numer telefonu wybranego użytkownika.

 

Zmień kod PIN

Wybierz łącze Zmień kod PIN, aby zmodyfikować kod PIN wybranego użytkownika.

 

Uwaga: to łącze jest dostępne tylko w przypadku, gdy użytkownik ma ustawiony tryb kodu PIN połączenia telefonicznego, tryb kodu OTP + kodu PIN wiadomości tekstowej lub tryb kodu PIN aplikacji mobilnej.

 

Zmień język

Wybierz łącze Zmień język, aby zmodyfikować język wybranego użytkownika.

 

Zresetuj próbkę głosu

Wybierz łącze Zresetuj próbkę głosu, aby zresetować próbkę głosu wybranego użytkownika.

 

Informacje o użytkowniku

Wybierz łącze Informacje o użytkowniku, aby zmodyfikować nazwę lub adres e-mail użytkownika.

 

Token OATH

Wybierz łącze Token OATH, aby zmodyfikować zewnętrzny token OATH użytkownika.

 

Włącz/wyłącz

Wybierz łącze Włącz lub Wyłącz, aby włączyć lub wyłączyć konto usługi Multi-Factor Authentication wybranego użytkownika.

 

Uwaga: wyłączony użytkownik może mieć możliwość zalogowania się bez użycia usługi Multi-Factor Authentication.

 

Zablokuj/odblokuj użytkownika

Wybierz łącze Zablokuj/odblokuj użytkownika, aby zablokować lub odblokować wybranego użytkownika.

 

Usuń użytkownika

Wybierz łącze Usuń użytkownika, aby trwale usunąć wybranego użytkownika.

 

Uwaga: usunięty użytkownik może mieć możliwość zalogowania się bez użycia usługi Multi-Factor Authentication.

 

Testuj uwierzytelnianie

Wybierz łącze Testuj uwierzytelnianie, aby przetestować uwierzytelnianie przy użyciu połączenia usługi Multi-Factor Authentication lub wiadomości testowej usługi Multi-Factor Authentication dla wybranego użytkownika.

 

Aktywuj aplikację mobilną

Wybierz łącze Aktywuj aplikację mobilną, aby wygenerować kod aktywacji aplikacji mobilnej dla wybranego użytkownika.

 

Dezaktywuj aplikację mobilną

Wybierz łącze Dezaktywuj aplikację mobilną, aby dezaktywować istniejące urządzenie wybranego użytkownika.
