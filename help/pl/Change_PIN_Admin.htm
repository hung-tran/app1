Zmień kod PIN

Strona Zmiana kodu PIN umożliwia zmianę kodu PIN dla konta wybranego użytkownika.  Ta zmiana jest trwała. Od momentu jej dokonania w usłudze Multi-Factor Authentication będzie używany nowy kod PIN.

 

Aby zmienić kod PIN wybranego użytkownika:

  1. Wprowadź nowy kod PIN.

  2. Wprowadź ponownie nowy kod PIN, aby potwierdzić.

  3. Kliknij przycisk Zapisz.

 

Funkcja

Opis

Nowy kod PIN

Wpisz nowy kod PIN.  Kod PIN można także wygenerować, klikając przycisk Generuj.  Kod PIN musi być zgodny z regułami dotyczącymi minimalnej długości i słabych kodów PIN.

 

Przycisk Generuj

Przycisk Generuj służy do generowania losowego kodu PIN.  Wygenerowany kod PIN będzie mieć długość większą niż 4 znaki lub określoną przez ustawienie minimalnej długości kodu PIN użytkownika.

 

Potwierdź kod PIN

Wpisz ponownie nowy kod PIN, aby potwierdzić zmianę. Kod PIN potwierdzenia musi być zgodny.

 

Wymagana zmiana kodu PIN

Określa, że użytkownik musi zmienić kod PIN podczas następnego uwierzytelniania wieloskładnikowego.

 

Uwaga: pole wyboru Wymagana zmiana kodu PIN będzie wyświetlane tylko w przypadku, gdy użytkownik może zmieniać swój kod PIN.

 

Wyślij wiadomość e-mail

Określa, czy do użytkownika powinna zostać wysłana wiadomość e-mail z informacją o aktualizacji.

 

Uwaga: pole wyboru Wyślij wiadomość e-mail będzie wyświetlane tylko w przypadku, gdy dla Twojej firmy została skonfigurowana poczta e-mail.
