Configuração de Usuário

Se a configuração da conta estiver incompleta, você deve especificar o Número do Telefone (se aplicável) e/ou PIN (se aplicável) para usar com a conta ou pode ser solicitado que você gere um código de ativação para ativar o aplicativo móvel do Multi-Factor Authentication.  Se a empresa permitir, será possível escolher entre os métodos de chamada telefônica, mensagem de texto, aplicativo móvel e token OATH.  Uma chamada do Multi-Factor Authentication pode ser realizada ou uma mensagem de texto enviada ao número de telefone que você especificou para confirmar se ele é válido, uma notificação será enviada ao aplicativo móvel ou será solicitado um código de Autenticação.  Para o método de chamada de telefone, será solicitado que você pressione #, digite o PIN ou registre-se na impressão de voz, dependendo do modo de chamada telefônica.  Para o método de mensagem de texto, você receberá uma senha única (OTP) e replicará com o OTP ou o OTP + PIN.  Para o método do aplicativo móvel, uma notificação será enviada ao aplicativo móvel e alertará para a autenticação, o que pode exigir que você insira o PIN, dependendo do modo de aplicativo móvel.  Para o método de token OATH, um código de Autenticação será solicitado pelo Portal do Usuário.  Se tiver êxito, você pode ser solicitado a criar suas perguntas de segurança.

 

Para concluir a configuração do usuário para o método de chamada telefônica:

  1. Digite o número de telefone e o código do país.  Você também pode inserir um número de telefone de backup (se aplicável).

  2. Digite o PIN (se aplicável).

  3. Clique no botão Telefonar para mim agora para Autenticar.

 

Para concluir a configuração do usuário para o método de mensagem de texto:

  1. Digite o número de telefone e o código do país.

  2. Digite o PIN (se aplicável).

  3. Clique no botão Enviar texto para mim agora para Autenticar.

 

Para concluir a configuração do usuário para o método de aplicativo móvel:

  1. Instale o aplicativo móvel no telefone.

  2. Clique no botão Gerar Código de Ativação.

  3. Digite o código de ativação e a URL no aplicativo móvel para concluir a ativação.

  4. Digite o PIN (se aplicável).

  5. Clique no botão Autentique-me agora.

 

Para concluir a configuração do usuário para o método de token OATH:

  1. Instale o aplicativo móvel no telefone.

  2. Clique no botão Gerar Código de Ativação.

  3. Digite o código de ativação e a URL no aplicativo móvel para concluir a ativação.

  4. Clique no botão Autentique-me agora.

OU

  1. Digite o número de série impresso na parte de trás do token OATH.

  2. Digite o código exibido atualmente no token OATH.

  3. Clique no botão Verificar Token OATH.

 

 

Recurso

Descrição

Método

Selecione o método de Chamada Telefônica, Mensagem de Texto, Aplicativo Móvel ou Token OATH.  Para o método de chamada telefônica, você receberá uma chamada do Multi-Factor Authentication solicitando que pressione #, digite o PIN ou registre a impressão de voz.  Para o método de mensagem de texto, você receberá uma senha única (OTP) em uma mensagem de texto e replicará com o OTP ou o OTP + PIN.  Para o método do aplicativo móvel, uma notificação será enviada ao aplicativo móvel e alertará para a autenticação, que pode exigir que você insira o PIN.  Para o método de token OATH, um código de Autenticação será solicitado pelo Portal do Usuário.

 

Observação:    o método selecionado será o método que você usará ao acessar o aplicativo protegido do Multi-Factor Authentication.  O método sempre pode ser alterado posteriormente na página Alterar Método.

 

Código do País

Selecione o código do país apropriado na lista suspensa. Se um número de telefone já tiver sido configurado para a conta, o padrão é o código do país para esse número de telefone.  Caso contrário, o padrão é definido para Estados Unidos & Canadá +1.

 

Observação: algumas chamadas internacionais resultam em encargos.  Se o administrador não implementou esta opção, uma chamada não será feita para os códigos de alguns países.

 

Telefone

Especifique o número de telefone que o Multi-Factor Authentication chamará ou texto durante a autenticação. Este pode ser um número de telefone celular, número de telefone de mesa ou algum outro número no modo de chamada telefônica.  Ele deve ser um telefone com recursos de mensagem de texto no modo de mensagem de texto.  O formato de número de telefone deve corresponder ao formato selecionado no campo Código do País.

 

Exemplo: para números de telefone dos EUA e Canadá, você deve inserir um número de telefone de 10 dígitos (por exemplo, 555-123-4567).

 

Ramal

Insira o ramal a ser discado depois que a chamada for atendida.  São permitidos dígitos, vírgulas, * e #.  Use vírgulas para pausas de um segundo para navegar nos menus do sistema do telefone.  Esse campo é opcional.

 

Código do País de Backup

Selecione o código do país apropriado para o número do telefone de backup.

 

Telefone de Backup

Insira o número do telefone que o Multi-Factor Authentication chamará, se a chamada para o telefone principal não for atendida.

 

Extensão de Backup

Insira o ramal para o número do telefone de backup.

 

PIN

Digite um número PIN.  Este PIN será inserido durante cada chamada de autenticação, inserido com a senha única (OTP) em resposta a cada mensagem de texto, ou inserida no aplicativo móvel para concluir a autenticação.  O administrador pode ter configurado regras de PIN, como um comprimento mínimo.

 

Número de Série

Digite o número de série impresso na parte de trás do token OATH.

 

Código

Digite o código exibido atualmente no token OATH.

 

 

Para retornar à página Fazer Logon, clique no botão Cancelar.
