Ativar Aplicativo Móvel

A página Ativar Aplicativo Móvel permite a você gerar um código de ativação de aplicativo móvel para ativar o aplicativo móvel do Multi-Factor Authentication no seu dispositivo.  Se você já ativou os dispositivos, eles estarão listados e podem ser desativados, conforme necessário.

 

Para ativar o aplicativo móvel do Multi-Factor Authentication no seu dispositivo:

  1. Clique no botão Gerar Código de Ativação.

  2. Insira o código de ativação e a URL no aplicativo móvel do Multi-Factor Authentication no dispositivo para concluir a ativação
    . Observação:  

    • Se um código de barras 2D for exibido, você pode usar um aplicativo de scanner de código de barras 2D para iniciar automaticamente o aplicativo móvel do Multi-Factor Authentication com o código de ativação e URL preenchidos.

    • Se você estiver acessando Portal do Usuário Multi-Factor Authentication em um navegador em seu dispositivo, um link Ativar será exibido ao invés do código de ativação e URL.  Clicar no link Ativar iniciará automaticamente o aplicativo móvel do Multi-Factor Authentication com o código de ativação e a URL preenchidos.

 

Para desativar um dispositivo:

  1. Clique no link Desativar ao lado do dispositivo.

  2. Confirme a desativação clicando em OK.
