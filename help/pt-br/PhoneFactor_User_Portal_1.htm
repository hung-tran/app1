Portal do Usuário Multi-Factor Authentication

O Portal do Usuário do Multi-Factor Authentication permite a você ignorar o Multi-Factor Authentication por um acesso, gerenciar as configurações de conta do Multi-Factor Authentication, incluindo o método, número de telefone, PIN, idioma e perguntas de segurança, e para ativar o aplicativo móvel do Multi-Factor Authentication. Alguns desses recursos podem ser desabilitados pelo administrador.  Consulte Perguntas Frequentes para obter informações sobre como usar o Multi-Factor Authentication.

 

Um sumário, índice e pesquisa estão disponíveis para ajudá-lo a localizar informações no sistema Ajuda.
