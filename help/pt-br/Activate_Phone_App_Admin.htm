Ativar Aplicativo Móvel

A página Ativar Aplicativo Móvel permite a você gerar um código de ativação de aplicativo móvel para o usuário selecionado.

 

Para gerar um código de ativação de aplicativo móvel para o usuário selecionado:

  1. Clique no botão Gerar Código de Ativação.

  2. Forneça o código de ativação e a URL para o usuário para que ele possa digitá-los no aplicativo móvel do Multi-Factor Authentication no dispositivo para concluir a ativação.
