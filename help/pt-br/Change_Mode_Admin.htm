Alterar Método

A página Alterar Método permite que você altere o Método para a conta do usuário selecionado.  Para o método de chamada telefônica, o usuário receberá uma chamada telefônica quando ele autenticar.  Para o método de mensagem de texto, o usuário receberá uma mensagem de texto quando ele autenticar.  Para o método de aplicativo móvel, o usuário receberá uma notificação no aplicativo móvel do Multi-Factor Authentication instalado no dispositivo.  Para o método de token OATH, um código de Autenticação será solicitado ao usuário.

 

Para alterar o Método do usuário selecionado:

  1. Selecione o método apropriado.

  2. Clique no botão Salvar.

Método

Selecione o método apropriado na lista suspensa. O padrão é o método atual do usuário.

 

Enviar Email

Especifica se um email deve ser enviado para informar ao usuário sobre a atualização.

 

Observação: a caixa de seleção Enviar Email só será exibida se o email tiver sido configurado para a sua empresa.
