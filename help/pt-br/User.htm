Usuário

A Página de Usuário exibe informações sobre o usuário e as opções para modificar a conta de usuário.  A página é exibida quando um único usuário é localizado na página Localizar Usuário ou ocorre um clique no link de usuário na página Localizar Usuário, quando diversos usuários são localizados.

 

O nome de usuário e número de telefone do usuário são exibidos no cabeçalho da página e as perguntas de segurança podem ser exibidas para permitir a verificação da identidade do usuário antes de se fazer as alterações.  Basta inserir as respostas do usuário e clicar no botão Verificar.  Se o usuário não tiver definido as perguntas de segurança, uma mensagem para esse efeito será exibida.

 

Os links são exibidos para modificações que podem ser feitas para a conta do usuário.  Nem todos os links podem ser exibidos, dependendo das permissões concedidas à conta de administrador e à configuração completa do Multi-Factor Authentication.

 

Recurso

Descrição

Bypass Descartável

Selecione o link Bypass Avulso para iniciar um  bypass avulso para o usuário selecionado.  O usuário poderá acessar em uma única vez sem receber uma chamada do Multi-Factor Authentication.

 

Alterar Método

Selecione o link Alterar Método para modificar o método do usuário selecionado.

 

Alterar Modo

Selecione o link Alterar Modo para modificar os modos do usuário selecionado.

 

Alterar Telefone

Selecione o link Alterar Telefone para modificar o número de telefone do usuário selecionado.

 

Alterar PIN

Selecione o link Alterar PIN para modificar o número PIN do usuário selecionado.

 

Observação: este link somente está disponível quando o usuário está no modo PIN de chamada telefônica, OTP + modo PIN de mensagem de texto, ou modo PIN de aplicativo móvel.

 

Alterar Idioma

Selecione o link Alterar Idioma para modificar o idioma do usuário selecionado.

 

Redefinir Impressão de Voz

Selecione o link Redefinir Impressão de Voz para modificar a impressão de voz do usuário selecionado.

 

Informações do Usuário

Selecione o link Informações do Usuário para modificar o nome e endereço de email do usuário.

 

Token OATH

Selecione o link Token OATH para modificar o token OATH de terceiros do usuário.

 

Habilitar/Desabilitar

Selecione o link Habilitar/Desabilitar para habilitar ou desabilitar a conta Multi-Factor Authentication do usuário selecionado.

 

Observação: um usuário desabilitado pode ser capaz de entrar sem o Multi-Factor Authentication.

 

Bloquear/Desbloquear Usuário

Selecione o link Bloquear/Desbloquear Usuário para bloquear ou desbloquear o usuário selecionado.

 

Remover Usuário

Selecione o link Remover usuário para remover permanentemente o usuário selecionado.

 

Observação: um usuário removido pode ser capaz de entrar sem o Multi-Factor Authentication.

 

Testar Autenticação

Selecione o link Testar Autenticação para testar uma chamada do Multi-Factor Authentication ou autenticação de mensagem de texto do Multi-Factor Authentication para o usuário selecionado.

 

Ativar Aplicativo Móvel

Selecione o link Ativar Aplicativo Móvel para gerar um código de ativação de aplicativo móvel para o usuário selecionado.

 

Desativar Aplicativo Móvel

Selecione o link Desativar Aplicativo Móvel para desativar um dispositivo existente para o usuário selecionado.
