Bypass Descartável

A página Bypass Avulso permite ignorar o Multi-Factor Authentication para o próximo acesso do usuário selecionado. Quando você iniciar um bypass, o usuário será capaz de autenticar sem o Multi-Factor Authentication uma única vez.  O bypass expirará se não for usado.

 

Para iniciar um bypass avulso para o usuário selecionado:

  1. Especifique os segundos do bypass.

  2. Clique no botão Confirmar para ignorar o Multi-Factor Authentication para o próximo acesso do usuário selecionado.
