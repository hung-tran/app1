Token OATH

A página Token OATH permite que você associe o usuário selecionado com um token OATH de terceiros.  O token deve ser definido no Servidor Multi-Factor Authentication antes de ser associado a um usuário.  O token deve ter suporte para códigos OATH com base no tempo.  Essa é uma alternativa para o uso de aplicativos móveis do Multi-Factor Authentication para códigos OATH.

 

Para especificar um token OATH:

  1. Digite os valores apropriados.

  2. Clique no botão Salvar.

 

Recurso

Descrição

Número de Série

Digite o número de série do token OATH de terceiros.

 

Código

Digite o código exibido atualmente no token OATH de terceiros.
