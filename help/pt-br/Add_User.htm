Adicionar Usuário

A página Adicionar Usuário permite que você crie uma nova conta do Multi-Factor Authentication.

 

Observação:  várias configurações do usuário, como o método (chamada telefônica, mensagem de texto ou aplicativo móvel), são padrão com base nos padrões do usuário especificados na seção Configurações da Empresa do Servidor Multi-Factor Authentication.

 

Para adicionar um novo usuário:

  1. Insira os detalhes do usuário.

  2. Clique no botão Adicionar.

 

Recurso

Descrição

Nome de Usuário

O Nome de Usuário geralmente é o email do usuário no formato de email apropriado.

 

Exemplo: chuck@ABC.com

 

Nome

Insira o nome do usuário.

 

Sobrenome

Insira o sobrenome do usuário.

 

Endereço de Email

Insira o endereço de email do usuário.  Os emails serão enviados para esse endereço de email.  Se não especificado, os emails serão enviados ao nome de usuário, se estiverem em formato de email apropriado.

 

Grupo de Usuários

Especifique o grupo de usuários do usuário.

 

Código do País

Selecione o código do país apropriado na lista suspensa. O padrão é definido para Estados Unidos & Canadá +1.

 

Observação: algumas chamadas internacionais resultam em encargos.  Se sua empresa não implementou esta opção, uma chamada não será feita para os códigos de alguns países.

 

Telefone

Especifique o número de telefone que o Multi-Factor Authentication chamará ou texto durante a autenticação. Este pode ser um número de telefone celular, número de telefone de mesa ou algum outro número para o método de chamada telefônica.  Ele deve ser um telefone com recursos de mensagem de texto para o método de mensagem de texto.  O formato de número de telefone deve corresponder ao formato selecionado no campo Código do País.

 

Exemplo: para números de telefone dos EUA e Canadá, você deve inserir um número de telefone de 10 dígitos (por exemplo, 555-123-4567).

 

Ramal

Insira o ramal a ser discado depois que a chamada for atendida.  São permitidos dígitos, vírgulas, * e #.  Use vírgulas para pausas de um segundo para navegar nos menus do sistema do telefone.  Esse campo é opcional.

 

Código do País de Backup

Selecione o código do país apropriado para o número do telefone de backup.

 

Telefone de Backup

Insira o número do telefone que o Multi-Factor Authentication chamará, se a chamada para o telefone principal não for atendida.

 

Extensão de Backup

Insira o ramal para o número do telefone de backup.

 

PIN

Digite o PIN do usuário.  Em alternativa, você pode gerar o PIN clicando no botão Gerar.  O PIN deve seguir as regras para o Comprimento Mínimo de PIN e Pins fracos.

 

Botão Gerar

O botão Gerar fornece um PIN gerado aleatoriamente.  O comprimento do PIN gerado será o maior de 4 dígitos ou o comprimento mínimo de PIN do usuário.

 

Confirmar PIN

Digite novamente o PIN inserido no campo PIN. O PIN da confirmação deve ser idêntico ao digitado anteriormente.

 

Habilitado

Especifica se o usuário está habilitado.  Por padrão, essa opção está marcada.

 

Observação: se um registro do usuário não estiver habilitado, esse usuário não receberá chamadas do Multi-Factor Authentication ao fazer acesso e pode ter o acesso concedido ou negado para o aplicativo, dependendo do padrão da empresa para o comportamento desabilitado.

 

Alteração de PIN Obrigatória

Especifica se um usuário deve alterar o PIN durante a próxima chamada do Multi-Factor Authentication.

 

Observação: a caixa de seleção Alteração de PIN Obrigatória somente será exibida se a configuração da empresa permitir que os usuários alterem o PIN.

 

Enviar Email

Especifica se um email deve ser enviado para o usuário adicionado.

 

Observação: a caixa de seleção Enviar Email só será exibida se o email tiver sido configurado para a sua empresa.
