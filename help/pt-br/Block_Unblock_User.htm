Bloquear/Desbloquear Usuário

A página de Bloquear/Desbloquear Usuário permite a você bloquear ou desbloquear um usuário.  A página será flexível dependendo do status de bloqueio atual do usuário.  Status de bloqueio do usuário que é mantido no POP do Multi-Factor Authentication.  As tentativas de autenticação de um usuário bloqueado automaticamente serão negadas.  

 

Para bloquear o usuário selecionado:

  1. Especifique um motivo para o bloqueio.

  2. Clique no botão Confirmar para bloquear o usuário.

Descrição

Motivo do Bloqueio

Insira o motivo pelo qual que o usuário está sendo bloqueado.

 

 

Para desbloquear o usuário selecionado:

  1. Especifique um motivo para o desbloqueio.

  2. Clique no botão Confirmar para desbloquear o usuário.

Descrição

Motivo do Desbloqueio

Insira o motivo pelo qual que o usuário está sendo desbloqueado.
