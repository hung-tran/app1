Remover Usuário

A página Remover Usuário permite que você remova permanentemente o usuário selecionado. O usuário removido pode ter o acesso concedido ou negado, dependendo da configuração do aplicativo no Servidor Multi-Factor Authentication.  Se o aplicativo for definido para Exigir Correspondência do Usuário, o usuário removido terá o acesso negado já que ele não será mais encontrado na lista de usuários do Servidor Multi-Factor Authentication.  Se o aplicativo não for definido para Exigir Correspondência do Usuário, o usuário terá uma autenticação com êxito sem o Multi-Factor Authentication.

 

Clique no botão Confirmar para remover permanentemente o usuário selecionado.
