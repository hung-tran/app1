Localizar Usuário

A página Localizar Usuário permite que um administrador procure uma conta de usuário existente.

 

Para localizar um usuário:

  1. Especifique os critérios de pesquisa.

  2. Clique no botão Localizar.

 

Se um único usuário corresponder aos critérios especificados, a página Informações do Usuário será exibida para esse usuário.  Se mais de um usuário corresponder, uma lista de usuários será exibida como links.  Clique no link do usuário correto para ser levado até a página Informação do Usuário.

 

Recurso

Descrição

Nome de Usuário

Insira a cadeia de pesquisa para correspondência de nome de usuário.

 

Observação: curingas são permitidos na cadeia de pesquisa ao se especificar um asterisco (*) para corresponder a diversos caracteres.

 

Nome

Insira a cadeia de pesquisa para correspondência de nome.

 

Sobrenome

Insira a cadeia de pesquisa para correspondência de sobrenome.

 

Endereço de Email

Insira a cadeia de pesquisa para correspondência de endereço de email.

 

Grupo de Usuários

Insira a cadeia de pesquisa para correspondência de grupo de usuários.

 

Telefone

Digite a cadeia de pesquisa para corresponder ao telefone.

 

Marca

Selecione Todas  para corresponder a todos os usuários ou selecione uma marca específica para corresponder apenas ao usuários com essa marca.
