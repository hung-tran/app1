Fazer Logon

Para fazer logon no Portal de usuário, digite seu nome de usuário e senha.  Uma chamada do Multi-Factor Authentication será realizada, uma mensagem de texto será enviada ao número de telefone, uma notificação será enviada ao aplicativo móvel do Multi-Factor Authentication, ou será solicitado que você insira o código de autenticação.  Se você concluir a autenticação com êxito, será levado à página principal.

 

Se a configuração da conta estiver incompleta, você será redirecionado à página Configuração de Usuário.

 

Se este for seu primeiro logon no Portal do Usuário, pode ser que seja solicitado que você crie suas perguntas de segurança.

 

Se o Multi-Factor Authentication for negado, pode ser que seja solicitado que você responda às suas perguntas de segurança para concluir o logon.

 

Para fazer logon:

  1. Digite seu nome de usuário e senha.

  2. Clique no botão Fazer Logon.

 

Recurso

Descrição

Nome de Usuário

O nome de usuário que você normalmente usa para fazer logon em seu aplicativo protegido do Multi-Factor Authentication.  Geralmente, este é o seu nome de usuário do Windows.

 

Senha

A senha que você normalmente usa para fazer logon em seu aplicativo protegido do Multi-Factor Authentication.  Geralmente, esta é a senha do Windows.
