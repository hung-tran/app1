Alterar PIN

A página Alterar PIN permite que você altere o PIN para a conta do usuário selecionado.  A alteração é permanente e o PIN recém especificado será usado no Multi-Factor Authentication no futuro.

 

Para alterar o PIN do usuário selecionado:

  1. Digite um novo PIN.

  2. Digite o novo PIN novamente para confirmar.

  3. Clique no botão Salvar.

 

Recurso

Descrição

Novo PIN

Digite um novo PIN.  Em alternativa, você pode gerar o PIN clicando no botão Gerar.  O PIN deve seguir as regras para o Comprimento Mínimo de PIN e Pins fracos.

 

Botão Gerar

O botão Gerar fornece um PIN gerado aleatoriamente.  O comprimento do PIN gerado será o maior de 4 dígitos ou o comprimento mínimo de PIN do usuário.

 

Confirmar PIN

Insira novamente o novo PIN para confirmar a alteração. O PIN da confirmação deve ser idêntico ao digitado anteriormente.

 

Alteração de PIN Obrigatória

Especifica se um usuário deve alterar o PIN durante o próximo Multi-Factor Authentication.

 

Observação: a caixa de seleção Alteração de PIN necessária somente será exibida se o usuário tiver permissão para alterar o PIN.

 

Enviar Email

Especifica se um email deve ser enviado para informar ao usuário sobre a atualização.

 

Observação: a caixa de seleção Enviar Email só será exibida se o email tiver sido configurado para a sua empresa.
