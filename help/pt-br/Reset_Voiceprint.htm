Redefinir Impressão de Voz

A página Redefinir Impressão de Voz permite que você redefina a senha de impressão para a conta de usuário selecionada.  Será solicitado que o usuário faça o registro de uma nova senha de impressão de voz durante a próxima chamada do Multi-Factor Authentication.

 

Clique no botão Confirmar para redefinir a senha de impressão de voz para a conta de usuário selecionada.
