OATH-token

Op de pagina OATH-token kunt u een OATH-token aan uw account koppelen.  Dit vormt een alternatief voor het gebruik van de mobiele Multi-Factor Authentication-app voor OATH-codes.

 

Ga als volgt te werk om een OATH-token op te geven:

  1. Geef de relevante waarden op.

  2. Klik op de knop Opslaan.

 

Functie

Beschrijving

Serienummer

Geef het serienummer op dat achter op het OATH-token is gedrukt.

 

Code

Geef de code op die op dat moment op het OATH-token wordt weergegeven.
