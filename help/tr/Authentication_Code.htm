Kimlik Doğrulaması Kodu

OATH Belirteci modunda Kullanıcı Portalı, Multi-Factor Authentication mobil uygulaması ya da belirtecinde görüntülenen kimlik doğrulaması kodunu isteyecektir.

 

Oturum açma işleminizi tamamlamak için:

  1. Multi-Factor Authentication mobil uygulamada ya da belirtecinde güncel olarak görüntülenen kimlik doğrulaması kodunu girin.

  2. Kimliği Doğrula düğmesine tıklayın.

 

Oturum Aç sayfasına geri dönmek için İptal düğmesine tıklayın.
