Güvenlik Sorusu Oluştur

Eğer ilk kez Kullanıcı Portalında oturum açıyorsanız sizden dört güvenlik sorusu oluşturmanız istenebilir.

 

Güvenlik sorularınızı oluşturmak için:

  1. Açılan listeden soruları seçin ve dört güvenlik sorunuzun yanıtlarını yazın.

  2. Devam düğmesine tıklayın.

 

Özellik

Açıklama

Güvenlik Sorusu 1

Açılan listeden birinci soruyu seçin.

 

Yanıt 1:

Birinci güvenlik sorusunu yanıtlayın. Girdi büyük/küçük harfe duyarlı değil.

 

Güvenlik Sorusu 2

Açılan listeden ikinci soruyu seçin.  Soru bir önceki sorunun tekrarı olmayabilir.

 

Yanıt 2:

İkinci güvenlik sorusunu yanıtlayın. Girdi büyük/küçük harfe duyarlı değil.

 

Güvenlik Sorusu 3

Açılan listeden üçüncü soruyu seçin.  Soru bir önceki sorunun tekrarı olmayabilir.

 

Yanıt 3:

Üçüncü güvenlik sorusunu yanıtlayın. Girdi büyük/küçük harfe duyarlı değil.

 

Güvenlik Sorusu 4

Açılan listeden dördüncü soruyu seçin.  Soru bir önceki sorunun tekrarı olmayabilir.

 

Yanıt 4:

Dördüncü güvenlik sorusunu yanıtlayın. Girdi büyük/küçük harfe duyarlı değil.

 

 

Oturum Aç sayfasına geri dönmek için İptal düğmesine tıklayın.
