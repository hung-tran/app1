用户设置

如果没有完成帐户设置，则必须指定电话号码（如果适用）和/或 PIN（如果适用）以便与帐户结合使用，或者系统可能会提示您生成一个激活代码以激活 Multi-Factor Authentication 移动应用程序。 如果您的公司允许，则您可以在电话呼叫、短信、移动应用程序和 OATH 令牌方法之间进行选择。 系统将会向您指定的电话号码发起 Multi-Factor Authentication 呼叫或发送短信以确认号码有效，向移动应用程序发送通知，或者提示您输入身份验证代码。 对于电话呼叫方法，系统会提示您按 #，然后输入您的 PIN 或注册声纹，具体取决于您的电话呼叫模式。 对于短信方法，系统会向您发送一次性密码 (OTP)，您可以使用 OTP 或 OTP 加上您的 PIN 进行回复。 对于移动应用程序方法，系统会向移动应用程序发送通知，提示您进行身份验证（可能要求您输入 PIN），具体取决于您的移动应用程序模式。 对于 OATH 令牌方法，系统会提示您输入用户门户使用的身份验证代码。 如果成功，则系统会提示您创建安全问题。

 

若要完成电话呼叫方法的用户设置：

  1. 输入国家/地区代码和电话号码。 您还可以输入备份电话号码（如果适用）。

  2. 输入您的 PIN（如果适用）。

  3. 单击“立即呼叫我以进行身份验证”按钮。

 

若要完成短信方法的用户设置：

  1. 输入国家/地区代码和电话号码。

  2. 输入您的 PIN（如果适用）。

  3. 单击“立即向我发短信以进行身份验证”按钮。

 

若要完成移动应用程序方法的用户设置：

  1. 在您的电话上安装移动应用程序。

  2. 单击“生成激活代码”按钮。

  3. 在移动应用程序中输入激活代码和 URL 以完成激活。

  4. 输入您的 PIN（如果适用）。

  5. 单击“立即对我进行身份验证”按钮。

 

若要完成 OATH 标记方法的用户设置：

  1. 在您的电话上安装移动应用程序。

  2. 单击“生成激活代码”按钮。

  3. 在移动应用程序中输入激活代码和 URL 以完成激活。

  4. 单击“立即对我进行身份验证”按钮。

或者

  1. 输入在 OATH 标记背面打印的序列号。

  2. 输入在 OATH 标记上显示的当前代码。

  3. 单击“验证 OATH 标记”按钮。

 

 

功能

描述

方法

选择电话呼叫、短信、移动应用程序或 OATH 标记方法。 对于电话呼叫方法，您会接到 Multi-Factor Authentication 呼叫，提示您按 # 并输入您的 PIN 或注册声纹。 对于短信方法，您会收到包含一次性密码 (OTP) 的短信，可以使用 OTP 或 OTP 加上您的 PIN 进行回复。 对于移动应用程序方法，系统会向移动应用程序发送通知，提示您进行身份验证（可能要求您输入 PIN）。 对于 OATH 令牌方法，系统会提示您输入用户门户使用的身份验证代码。

 

注意：  您选择的方法将会在访问受 Multi-Factor Authentication 保护的应用程序时使用。 您稍后随时可以在“更改方法”页面上更改方法。

 

国家/地区代码

从下拉列表中选择相应的国家/地区代码。如果已经为帐户配置了电话号码，则默认值为该电话号码的国家/地区代码。 否则，默认设置为“美国和加拿大 +1”。

 

注意：某些国际电话呼叫需要收取费用。 如果您的管理员尚未实施此选项，则系统不会向某些国家/地区代码发起呼叫。

 

电话号码

输入身份验证期间 Multi-Factor Authentication 呼叫或发送短信时将使用的电话号码。这有可能是移动电话号码、座机号码，或者是电话呼叫模式中的其他一些号码。 对于短信模式，电话必须具备短信功能。 电话号码的格式必须与“国家/地区代码”字段中选定的格式一致。

 

示例：对于美国和加拿大的电话号码，必须输入 10 位数字的电话号码（例如 555-123-4567）。

 

分机号

输入在呼叫应答后将要拨打的分机号。 允许使用数字、逗号、* 和 #。 逗号可用于暂停一秒钟以导航电话系统菜单。 此为可选字段。

 

备份国家/地区代码

请选择用于备份电话号码的相应国家/地区代码。

 

备份电话号码

输入在呼叫主要电话没有应答时，Multi-Factor Authentication 将要呼叫的电话号码。

 

备份分机号

输入备份电话号码的分机号。

 

PIN

输入 PIN 号码。 您会在每次身份验证呼叫期间输入 PIN，在回复每条短信时输入 PIN 和一次性密码 (OTP)，或者在移动应用程序中输入 PIN 以完成身份验证。 您的管理员可能已配置 PIN 规则，如最小长度。

 

序列号

输入在 OATH 标记背面打印的序列号。

 

代码

输入在 OATH 标记上显示的当前代码。

 

 

若要返回到登录页面，请单击“取消”按钮。
