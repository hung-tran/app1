登录

若要登录到用户门户，请输入您的用户名和密码。 这将会向您的电话号码发起 Multi-Factor Authentication 呼叫和发送短信、向 Multi-Factor Authentication 移动应用程序发送通知，或者提示您输入身份验证代码。 如果您成功完成身份验证，则会进入主页面。

 

如果您的帐户设置得不完整，则系统会将您导航到用户设置页面。

 

如果您是首次登录到用户门户，则系统会提示您创建您的安全问题。

 

如果 Multi-Factor Authentication 遭到拒绝，则系统会提示您回答您的安全问题以完成登录过程。

 

若要登录：

  1. 请输入您的用户名和密码。

  2. 单击“登录”按钮。

 

功能

描述

用户名

您为了登录到 Multi-Factor Authentication 受保护应用程序而通常使用的用户名。 通常此为您的 Windows 用户名。

 

密码

您为了登录到 Multi-Factor Authentication 受保护应用程序而通常使用的密码。 通常此为您的 Windows 密码。
