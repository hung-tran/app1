常见问题解答

Multi-Factor Authentication™ 是如何工作的？

Multi-Factor Authentication 的工作内容涉及：安排确认呼叫、发送短信、登录期间通知 Multi-Factor Authentication 移动应用程序，或者提示从 Multi-Factor Authentication 移动应用程序或标记中输入身份验证代码。

 

步骤 1：

输入您的用户名和密码。

步骤 2：

您不久就会收到一个电话呼叫、短信或移动应用程序通知，或者系统会提示您输入身份验证代码。对于电话呼叫方法，请在应答时按 #，然后输入您的 PIN 或者说出您的音纹短语。 对于短信方法，请在回复短信时使用一次性密码 (OTP) 或者结合使用 OTP 与您的 PIN。 对于移动应用程序方法，请在设备上安装的 Multi-Factor Authentication 移动应用程序中按“身份验证”，或者输入您的 PIN 并按“身份验证”。 对于 OATH 令牌方法，请输入 Multi-Factor Authentication 移动应用程序或令牌中显示的身份验证代码。

就这么简单！

 

这个简单的流程通过两个不同的渠道（计算机和电话服务）提供了两种身份验证方法。在电话呼叫模式中，它可以使用任何常规电话或移动电话；在短信模式中，它可以使用任何具备短信功能的电话；或者使用支持 Multi-Factor Authentication 移动应用程序的设备。

 

如果我的电话丢了，该如何处理？

选择“更改电话号码”选项，以便输入新的电话号码。一旦建立了强大的身份标识，则还可以通过呼叫支持问讯台来设置备用号码。

 

如果移动电话在某一地区没有信号，该如何处理？

可以通过选择“更改电话号码”选项，更改您的帐户以指向备用的电话号码。某些公司还提供了“免验证一次”选项，允许您在不使用电话呼叫或短信的情况下登录一次。如果可用，请选择“免验证一次”选项来访问此功能。或者从任意一部电话致电支持问讯台以获取帮助。 OATH 令牌方法不要求有移动电话信号覆盖。

 

在我没有试图登录时，我却从 Multi-Factor Authentication 收到一个电话呼叫、短信或移动应用程序通知，这该怎么办？

这种情况只会发生在其他人试图登录您的帐户时，他们已经知道了您的密码。请记住，只有在用户名和密码经过验证后，才会安排电话呼叫、发送短信或发送移动应用程序通知。因此，如果发生此类情况，则说明 Multi-Factor Authentication 刚才保护了您的帐户免遭非法访问！若要临时阻止对您的帐户进行身份验证，对于电话呼叫方法，请遵循电话菜单中的提示；对于短信方法，请使用您公司提供的期诈代码回复短信；或者，在移动应用程序中按“拒绝并举报欺诈”。 如果您的公司启用了欺诈警报，则系统除了会阻止您的帐户之外，还会立即向您公司的 IT 安全团队发送警报。若要解除锁定您的帐户，请联系您的支持问讯台。
