用户

“用户”页面显示的信息涉及用户和修改用户帐户的选项。 如果您在查找用户页面上找到一位用户，或者在查找用户页面上找到多位用户并且单击了某用户链接时，则系统会显示该页面。

 

用户的用户名和电话号码会显示在页面的页眉位置，另外还可能会显示出用户的安全问题，进而允许您在执行更改之前验证用户的身份。 只需输入用户的答案，然后单击“验证”按钮即可。 如果用户尚未定义安全问题，则系统会针对此效果显示一条消息。

 

系统会显示一些您可以对用户帐户进行修改的链接。 系统并非会显示所有的链接，具体取决于向管理员帐户授予的权限以及 Multi-Factor Authentication 的整体配置。

 

功能

描述

免验证一次

选择“免验证一次”链接可以为选定的用户发起免验证一次。  用户将能够登录一次，并且不会接到 Multi-Factor Authentication 呼叫。

 

更改方法

选择更改方法链接可以修改选定用户的方法。

 

更改模式

选择更改模式链接可以修改选定用户的模式。

 

更改电话

选择更改电话号码链接可以修改选定用户的电话号码。

 

更改 PIN

选择更改 PIN 链接可以修改选定用户的 PIN 号码。

 

注意：这个链接只有在用户处于电话呼叫 PIN 模式、短信 OTP + PIN 模式，或移动应用程序 PIN 模式时才有效。

 

更改语言

选择更改语言链接可以修改选定用户的语言。

 

重置声纹

选择重置声纹链接可以重置选定用户的音纹。

 

用户信息

选择用户信息链接可以修改用户的名称或电子邮件地址。

 

OATH 令牌

选择 OATH 标记链接可以修改用户的第三方 OATH 标记。

 

启用/禁用

选择启用或禁用链接可以启用或禁用选定用户的 Multi-Factor Authentication 帐户。

 

注意：被禁用的用户也许可以登录，而无需 Multi-Factor Authentication。

 

阻止/解除阻止用户

选择阻止/解除阻止用户链接可以阻止或解除阻止选定的用户。

 

删除用户

选择删除用户链接可以永久地删除选定的用户。

 

注意：被删除的用户也许可以登录，而无需 Multi-Factor Authentication。

 

测试身份验证

选择测试身份验证链接可以测试选定用户的 Multi-Factor Authentication 呼叫或 Multi-Factor Authentication 短信身份验证。

 

激活移动应用程序

选择激活移动应用程序链接可以为选定的用户生成移动应用程序激活代码。

 

取消激活移动应用程序

选择停用移动应用程序链接可以为选定的用户停用现有的设备。
